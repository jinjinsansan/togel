## レベル1：基礎編（世界観の理解）

### 目的
引き寄せの法則を理解するための世界観・前提知識を身につける

### セクション構成

| No | タイトル | 学習目標 | 前提 | 所要時間目安 |
|----|----------|----------|------|--------------|
| S1 | 私とソースの関係 | 「私＝ソース」という根本概念を理解する | なし | 15-20分 |
| S2 | ソースの本質 | ソースがジャッジメントしない存在であることを理解する | S1 | 15-20分 |
| S3 | 思考ちゃんとマインド様 | 思考とマインドの違いを区別できるようになる | S1, S2 | 20-25分 |
| S4 | 「無」と時間の概念 | 「無」から生まれた世界、時間の幻想を理解する | S1-S3 | 20-25分 |

### 依存関係図

```
S1（私とソース）
    ↓
S2（ソースの本質）
    ↓
S3（思考とマインド）
    ↓
S4（無と時間）
    ↓
【レベル2へ】
```

### このレベルで身につけること
- [ ] 「私はソースの一部である」と腹落ちする
- [ ] ソースは良い悪いを判断しないことを理解する
- [ ] 思考（エゴ）とマインド（本質）の違いがわかる
- [ ] 時間は人間が作った概念であることを理解する

## 復習・戻りのルール

### 戻りが推奨される状況

1. **理解度チェックに2回連続で不合格**
   → 1つ前のセクションに戻る

2. **ユーザーが「よくわからない」と言った**
   → 該当セクションを再説明、必要なら前のセクションへ

3. **ネガティブ感情をコントロールできないと申告**
   → S13に戻る（必須）

4. **レベル4で行き詰まった**
   → レベル3の該当セクションに戻る

### スキップが許可される状況

1. **診断で高い理解度が確認された**
   → 該当レベルをスキップ可能

2. **特定テーマに興味がない**
   → レベル3の一部セクション（S8-S9等）はスキップ可能
   → ただしS13（ネガティブ対処）はスキップ不可

3. **時間がない**
   → 最短学習パスを提案
