# 機能不全家庭

## 定義

機能不全家庭とは、本来の家庭の機能（役割）が正常に働いていない家庭環境のこと。反対語は「円滑に機能している家庭環境」。子供にとって必要な役割が果たされず、青い旗（ガムテープ）が大量に立てられる環境。

## 背景・目的

### この世界の絶対法則

1. **必ず母親から生まれる** - 逆らえない仕組み
2. **必ず0歳からスタートする** - いきなり25歳から始まることはない
3. **必ず幼少期を経て大人になる** - 誰も例外はない

### もし25歳から生まれられたら

- 機能不全家庭だと気づいてすぐ出ていける
- 「ハローワーク行ってくるわ」ができる
- ガムテープを貼ることもなかったかもしれない

**しかし、そういう仕組みになっていない**

### 幼少期の重要性

テープ式心理学の考え方：

**幼少期の家庭環境によって、その後の人生が決定されると言っても過言ではない**

それぐらい幼少期は大事。

## 親になるプログラム

### 全ての生き物に備わっている

- 人間だけでなく動物、鳥、魚、昆虫も同じ
- **誰にも教わっていないのに**我が子を愛する
- 我が子を惜しみなく愛することを自然にしていく

### だからほとんどの家庭は

- 我が子を愛するような日々を送っている
- 世界中で動物たちも子供を愛している
- 温かいシチューが作られるようなイメージ
- **これが大多数派**

## 円滑に機能している家庭の特徴

### サザエさんの家のイメージ

| 特徴 | 説明 |
|------|------|
| 感情がそのまま表現されている | 言いたいことが言える |
| 全員に感情を口に出す機会がある | 抑圧されていない |
| 感情が受け入れられている | 批判されない |
| 理解と支えがある | サポートし合う |
| 自由に質問できる雰囲気がある | 何でも聞ける |
| 建設的な友好関係がある | 敵対していない |
| 肯定的な意味での境界線とルールがある | 互いが守られる |

### 役割分担

- **母は母の役割**をしっかり担当
- **父は父の役割**をしっかり担当

伝統的な例：
- 男は狩りに出て食べ物を取ってくる（経済的な大黒柱）
- 女性は洞窟の中で子供と家族を守る

※縄文時代から続くプログラム

### 青い旗について

円滑に機能している家庭では：
- 青い旗がほとんど立たない
- 立ったとしても家族で下ろし合う
- だから青い旗がほとんどないまま大人になれる

### 円滑家庭で育った子の特徴

- **自分がいる**（私がいる）
- 人との境界線が分かる
- 多少は無価値観もあるが**根っこにベースの安全がある**
- 「私は安全である」
- 「私には帰る家があって父母もいる」
- 小さい頃から「私」というものがいる

**トリシアの表現：**
- **価値を認められている**
- **無防備になれる**（ガードする必要性がない）
- **不完全でいられる**（完璧でなくていい）

安全な環境なのでのびのびできる。

## 機能不全家庭の特徴

### 真逆の環境

| 特徴 | 説明 |
|------|------|
| 感情が抑圧される | 言いたいことが言えない |
| 秘密になる | 感情を正直に口にできない |
| 感情を出すと批判・非難される | 怒られる |
| 友好関係がない | あっても支配・操り・戦うため |
| ルールは恐れを基盤としている | 罪悪感、恐れ、「他人に話すな」など |

### 家庭内のルールの基盤

機能不全家庭のルールは3つの要素を基盤としている：

**1. 恥（世間体）**
- 「こんなの世間ではどう言われるか分かってるの？」
- 「恥ずかしい」
- 「世間はね、世間はね」
- 子供に対して世間体を押し付ける

**2. 罪悪感**
- 「あなたが悪いのよ」
- 「あなたが悪い、あなたが悪い」
- 子供に責任を押し付ける
- 全て子供のせいにする

**3. 恐れ**
- 「もうこの家は無理よ」
- 「もうみんな離婚するし、家族バラバラ」
- 「お金ないわよ」
- 子供を不安にさせ、脅かす

**さらに：**
- 「家庭内の暴力を他人に話すな」
- 秘密にすることを強要される

### ネガティブが基盤

機能不全家庭は**ネガティブが基盤**になっている。
円滑な家庭のように愛情や安全が基盤ではなく、恥・罪悪感・恐れが土台。

### 機能不全家庭で育った子

- **自分がとても小さくなる**
- 自分がない
- その状態で社会に出る
- **安全が分からない**
- 居場所がない

### 青い旗について

機能不全家庭では：
- 青い旗だらけ
- 立てられて下ろされない
- 青い旗が原因でまた青い旗が立つ
- 4歳、5歳、6歳、7歳...全部に青い旗

## 毒親という言葉について

### テープ式心理学では使わない

理由：
- 「毒」という言葉がネガティブなイメージ
- 怒りを出している感じがする
- 悪者探しになってしまう

### テープ式の考え方

- 本来、親は子供を愛している
- 子供も親を愛している
- しかし親側が**自分自身のネガティブをコントロールできない**
- ガムテープがたくさんあってコントロールできない
- 結果として我が子を傷つけてしまう

**毒親という言葉は使わないが、仕組みとして存在する**

## 子供の無力さ

### 子供にできないこと

- 自分で稼ぐことができない
- 知識も知恵もない
- 味方もいない
- 出て行くこともできない
- この世界の仕組みすら知らない

### だから

**親の言うことを聞かなければ生きていけない**

- ご飯が食べられない
- 家から放り出されたら死んでしまう
- 命を守るために虐待を受け入れていく

### 機能不全家庭での生存

残酷な虐待であっても：
- 泣きながら
- 苦しみながら
- ガムテープを何枚も何枚も貼り付けて
- バランスを保ちながら
- 対処機構を使いながら

**私たちは生きてきた**

## トリシアの言葉

### 子供の頃のあなたについて

**「子供の頃のあなたには100%責任はありません」**
**「あなたは何も悪くなかったのです」**

### なぜ悪くないか

- 子供は無力
- 稼ぐこともできない
- 出て行くことすらできない
- 従うしかなかった
- その家で過ごすしかなかった

### しかし

**「今のあなたを修正するには100%あなたしかできません」**

- ガムテープを剥がすのは本人だけ
- トリシアでも仁さんでも他人ではできない

## 円滑家庭と機能不全家庭の比較

### 健全な家庭では

**両親が子供の傷を緩和する**

青い旗を下ろしてくれる

### 機能不全家庭では

**両親が子供に傷を追わせる**

全く逆の仕組み

## 機能不全家庭で起きること

### 子供の怒りの否認

- 子供は「やめてほしい」「嫌だ」と感じる
- しかしそれを否定させられる
- 怒りを封じ込められる

### 慢性的な危機状態

- 現実を否定することで機能不全が作られる
- 周囲で悪いことが起きると全て自分のせいだと責める
- 恥と罪の意識を感じる
- **無価値観のガムテープがたくさん**

### 防御の壁

苦しすぎて：
- 怒り続ける
- 話を聞かない
- 妄想
- 入っていけないような壁を作る

## 子供時代の喪失

### 凍りつく

トラウマの瞬間に：
- 本当の自己を否認したまま凍りつく
- 「自分はダメだ」「無価値」のまま
- ガムテープを貼っている

### 失うもの

- 自分の本当の姿を知る可能性
- 向き合うこと
- 自我を守るため防御の壁を維持し続ける

## 機能不全家庭育ちのパターン

### 大人になってから

- 機能不全な大人になる
- 何かに中毒（依存）する
- 依存している人間と結婚する
- **感情的・心理的に機能不全家庭のパターンを再現してしまう**

ガムテープを再度再現してしまう

## この勉強の目的

### 悪者探しではない

- 母が悪い、父が悪い、と言いたいわけではない
- 母も父も同じくガムテープを貼っていた
- 悪者探しをしたいのではない

### 目的

1. 自分の家庭環境が機能不全家庭だったか円滑な家庭だったかを認識する
2. 機能不全家庭だった場合、青い旗がほとんど降りなかったことを理解する
3. ガムテープがついたままで生きづらいことを認識する
4. 子供の頃は悪くなかったが、今の自分を修正するのは自分しかいないことを受け入れる

## 「そんな家あるんですか？」

### 少数派からの質問

円滑な家庭の特徴を話すと：
- 「そんなお家ってあるんですか？」
- 「映画の話でしょ？」

と言われることが多い

### 事実

**円滑に機能している家庭が大多数派**

機能不全家庭は少数派だが、その少数派の方が仁さんのところにアクセスしてくる

## よくある誤解・注意点

- ❌ 機能不全家庭で育った＝人生終わり
- ⭕ ガムテープは剥がせる。今から修正できる

- ❌ 親を恨むべき
- ⭕ 親も同じくガムテープを持っていた。悪者探しではなく仕組みを理解する

- ❌ 円滑な家庭は問題がない
- ⭕ 問題が起きても下ろし合えるのが円滑な家庭

- ❌ 0歳から始まるのは当たり前すぎて意味がない
- ⭕ この「当たり前」が全ての始まり。逆らえない仕組みを理解することが大事

## まとめ

機能不全家庭とは、家庭の本来の機能が働いていない環境。子供は無力で、どうしても0歳から母のもとで育つしかないプログラムになっている。円滑な家庭では青い旗が下ろし合われるが、機能不全家庭では青い旗が立てられ続け、下ろされない。子供の頃は100%悪くなかったが、今の自分を修正できるのは100%自分だけ。悪者探しではなく、仕組みを理解することが重要。
