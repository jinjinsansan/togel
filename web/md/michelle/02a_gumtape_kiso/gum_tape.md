# ガムテープ

## 定義

ガムテープとは、幼少期に貼り付けられた「**思い込み**」のこと。これが無意識レベルで意味づけを決定し、ネガティブな感情を発生させる原因となる。

## 背景・目的

### なぜ「ガムテープ」と呼ぶか

- 心に貼り付いているイメージ
- 剥がすことができるもの
- 粘着性があり、簡単には取れない
- でも剥がせば取れる

### テープ式心理学の名前の由来

「テープ式心理学」の「テープ」は、この「ガムテープ」から来ている。

## 中核ポイント

### ガムテープの特徴

| 特徴 | 説明 |
|------|------|
| 幼少期に貼られる | 主に0歳〜7歳頃に形成される |
| 無意識に作用する | 本人は気づいていないことが多い |
| 自動的に意味づけを決定 | 見えないスピードで意味づけが行われる |
| 剥がすことができる | 気づきによって解放される |

### ガムテープの形成過程

1. 幼少期に何らかの出来事が起きる
2. その出来事に対して意味づけをする
3. その意味づけが「思い込み」として定着
4. 以降、似た出来事に対して自動的に同じ意味づけをする

### 具体例

**出来事：** 子供の頃、親に「あなたは何をやってもダメね」と言われた

**形成されたガムテープ：** 「私は何をやってもダメだ」

**その後の影響：**
- 新しいことに挑戦しようとすると「どうせダメだ」と思う
- 失敗すると「やっぱり私はダメだ」と確認してしまう
- 成功しても「たまたまだ」と思う

## ガムテープの種類

### よくあるガムテープの例

- 「私は愛されない」
- 「私は価値がない」
- 「私は幸せになってはいけない」
- 「私は決して成功しない」
- 「私はピエロである」
- 「私は奴隷である」
- 「私は邪魔者だ」
- 「私は存在してはいけない」

### ガムテープの見つけ方

1. **なりたい私の逆接** - なりたい私の真逆を見る
2. **心理テスト** - 6つの文章で一発で見つける
3. **ピールダウン** - 感情を深掘りして見つける

## ガムテープを剥がすとは

### 剥がすプロセス

1. ガムテープの存在に**気づく**
2. それが**思い込み**であると認識する
3. **決断**する（変わろう、向き合おう）
4. 解放テクニックを使う

### 気づきの瞬間

- 「あ、これ思い込みだったんだ」
- 「そうか、これがあったから苦しかったんだ」
- この気づきの瞬間に、ガムテープが剥がれ始める

### 剥がれた後

- 同じ出来事に対して、違う意味づけができるようになる
- ネガティブな感情が発生しなくなる
- 「なりたい私」に自然となれる

## 1枚剥がれると連鎖する

### ガムテープの連鎖性

- ガムテープは1枚だけ存在するわけではない
- 複数のガムテープが絡み合っている
- しかし、1枚剥がれると関連するものが**ごっそり剥がれる**

### 例

「私は愛されない」というガムテープが剥がれると：
- 「私は価値がない」も剥がれやすくなる
- 「私は幸せになれない」も剥がれやすくなる
- 根っこが同じだから

## よくある誤解・注意点

- ❌ ガムテープは性格
- ⭕ ガムテープは後天的に貼られた思い込み。性格ではない

- ❌ ガムテープは一生剥がれない
- ⭕ 気づきと決断があれば剥がすことができる

- ❌ ガムテープを貼った人（親など）が悪い
- ⭕ 親も親のガムテープで苦しんでいた。責めるのではなく理解する

## まとめ

ガムテープとは幼少期に貼られた思い込みであり、無意識レベルで意味づけを決定している。ガムテープの存在に気づき、それが思い込みであると認識し、向き合う決断をすることで剥がすことができる。1枚剥がれると関連するガムテープも連鎖的に剥がれていく。
