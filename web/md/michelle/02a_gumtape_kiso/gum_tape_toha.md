# ガムテープとは

## 定義

ガムテープとは、幼少期に貼られた「思い込み」のこと。過去の出来事に対する意味づけが固定化され、現在の行動や感情に無意識に影響を与え続けているもの。

## 背景・目的

### なぜ「ガムテープ」と呼ぶか

仁さんの師匠であるトリシア・カエタノ氏は、生徒たちにこのように教えた：

> 「私達は、常に思い込みの中で生きている。この世界の全てに意味は存在せず、この世界に意味付けをしているのは本人である。」

この「意味づけ」「思い込み」を、トリシア師匠は「テープ」という呼称で教えた。そのため、「テープ」という言葉を使用して「テープ式心理学」を構築することになった。

### ガムテープの視覚イメージ

心の中は目には見えない。そのために、目に見えるようにするために、故意的に「ガムテープ」という視覚イメージを作った。

**イメージ：**
```
段ボールに貼り付けるガムテープに
マジックで「思い込みの言葉」が書いてある
    ↓
それが体にベタベタと貼り付けてある
```

**例：**
- 「犯罪行為をする人間はダメな人間、価値の無い人間」
- この文章そのものがマジックで書いてあり、体に貼り付けているイメージ

### ガムテープの特徴

- 心に貼り付いているイメージ
- 剥がすことができる（＝解放できる）
- 一般的な心理学用語では「ビリーフ」「信念」などと呼ばれる

### 別名：かたる信念（怖臭）

ガムテープは**かたる信念**（または**怖臭**）とも呼ばれる。

- 「かたる」＝固まる、固定される
- 心の奥深いところで「そうなんだ」と決めてしまった信念
- 決断しないと、その時耐えられなかったから決めた

### テープ式心理学の立場

- 精神病の原因は「ガムテープ」
- 薬では剥がせない
- 向き合って剥がすしかない

## 対処機構としてのガムテープ

ガムテープは**対処機構**によって貼られる。

### 対処機構とは

心のバランスを保つために、ガムテープを貼る仕組み。

```
【元の状態】バランスの取れた心
    ↓
【出来事】強いストレス・トラウマ
    ↓
【心の反応】バランスが崩れそう（こけそう）
    ↓
【対処機構発動】ガムテープを貼る（足を出す）
    ↓
【結果】歪んだ形でもバランスを保つ
```

### ガムテープは悪くない

- 自分を守るために必要だった
- 貼らなければ心がこけていた
- 悪いことではなく、対処として正しかった

## 中核ポイント

### ガムテープの特徴

| 特徴 | 説明 |
|------|------|
| 幼少期に形成 | 主に0〜7歳頃に貼られることが多い |
| 無意識に作用 | 本人は気づいていないことが多い |
| 自動的に発動 | 特定の状況で自動的に意味づけを行う |
| 剥がせる | 向き合うことで解放できる |

### ガムテープの形成プロセス

```
① 幼少期の出来事
    ↓
② 子供なりの意味づけ（思い込み）
    ↓
③ ガムテープとして固定
    ↓
④ 大人になっても無意識に影響
```

### 思い込み（ガムテープ）が作られるきっかけ

今のあなたの思い込みは、様々な要素によって作られている：

| 要素 | 説明 |
|-----|------|
| 生まれた時代 | その時代の常識・価値観 |
| 生まれた家 | 家庭環境・家族の価値観 |
| 生まれた親 | 親からの言葉・態度 |
| 性別 | 性別による社会的期待 |
| テレビ | メディアからの情報 |
| 学校 | 教育・先生からの影響 |
| 友達 | 友人関係での経験 |
| 書籍・雑誌 | 読んだ本からの影響 |
| 歌 | 音楽の歌詞からの影響 |

**ありとあらゆる要素が、あなたの思い込みを作るきっかけ**＝**ガムテープを貼り付けるきっかけ**となっている。

### 具体例

**出来事：** 幼少期に親から「あなたは何をやってもダメね」と言われた

**子供の意味づけ：** 「私は何をやってもダメなんだ」

**ガムテープ：** 「私は無価値である」

**大人になってからの影響：**
- 新しいことに挑戦できない
- 失敗を極度に恐れる
- 自己肯定感が低い

## ガムテープと3ステップの関係

### 通常の3ステップ

```
① 出来事
    ↓
② 意味づけ（自由に選べる）
    ↓
③ 感情発生
```

### ガムテープがある場合

```
① 出来事
    ↓
② 意味づけ（ガムテープに支配される）
    ↓
③ ネガティブ感情発生
```

ポイント: ガムテープがあると、意味づけを「自由に選べない」状態になる。

## ガムテープの種類

### よくあるガムテープの例

- 「私は愛されない」
- 「私は価値がない」
- 「私は幸せになってはいけない」
- 「私はダメな人間だ」
- 「私は人に迷惑をかける存在だ」
- 「私は一人ぼっちだ」
- 「世界は危険な場所だ」
- 「人は信用できない」

### 心理テストで発見できる

6つの文章の心理テストを使うと、ガムテープが一発で出てくる。

## ガムテープを剥がすとどうなるか

### 剥がす前

- 同じパターンで苦しみ続ける
- 何十年経っても消えない
- 薬を飲んでも根本解決しない

### 剥がした後

- 同じ出来事でもネガティブにならない
- 意味づけを自由に選べるようになる
- 「なりたい私」になれる

### 例

**ガムテープ：** 「人前で話すのが怖い」

**剥がす前：** 人前で話す場面になると毎回緊張、恐怖、逃げたくなる

**剥がした後：** 人前で話すことが普通にできる。「なりたい私」が実現

## パニック障害の例

### 一般的な説明

「満員電車でパニック発作が起きる」

### テープ式の説明

```
① 出来事：満員電車に乗る
    ↓
② ガムテープ発動：「危険だ」「逃げられない」「死ぬかもしれない」
    ↓
③ パニック（恐怖の極限）
```

### ガムテープの起源

- 過去に満員電車で怖い思いをした
- または、狭い場所で怖い思いをした経験がある
- その時の意味づけがガムテープになっている

### 解放の方向性

- 満員電車そのものは危険ではない
- ガムテープ（「危険だ」という思い込み）を剥がす
- 剥がせたら、満員電車に乗っても平気になる

## トラウマ・ガムテープ・インナーチャイルドは同じ

### 青い旗理論との関係

トリシア・カエタノの教えでは、人生のタイムライン上で強いストレスがかかった時に「青い旗」が立つ。この青い旗が下ろされないまま現在に至っている状態を、様々な呼び方で表現する。

### 全て同じものを指す

| 用語 | 説明 |
|------|------|
| 青い旗 | トリシアの表現（タイムライン上に立つ旗） |
| トラウマ | 一般的な心理学用語 |
| ガムテープ | テープ式心理学での呼び方 |
| インナーチャイルド | その時の子供が残ったままという表現 |

### つまり

- 「私にはトラウマがある」と言うなら
- 「その日にガムテープがある」ということ
- 「インナーチャイルドも残っている」ということ
- **全部同じ、呼び方が違うだけ**

## よくある誤解・注意点

### 人生が望まない方向に進んでいる場合

どのような意味づけをするのも自由自在であり、良い悪いも存在しない。全ては本人が好きなように意味づけをしても良い。

しかし、もしあなたが**自分の人生が望まない方向で進んでいる**ならば：

- 何かしらのネガティブな意味づけをたくさん貼り付けている
- そのためにがんじがらめに動けない状態になっている可能性がある

**まずは、自分自身の中にベタベタと貼り付けているガムテープを見つけることが重要。**

### その他の誤解

- ❌ ガムテープは性格だから変えられない
- ⭕ ガムテープは思い込みであり、剥がすことができる

- ❌ 大人になってからも新しいガムテープが貼られる
- ⭕ 基本的に幼少期に形成される。大人になってからのトラウマも、幼少期のガムテープと紐づいていることが多い

- ❌ ガムテープを剥がすには何年もかかる
- ⭕ 向き合う決断ができれば、1時間で剥がせることもある

## まとめ

ガムテープとは幼少期に形成された思い込みであり、現在の意味づけを無意識に支配している。ガムテープがある限り、同じパターンで苦しみ続ける。しかし、ガムテープは剥がすことができる。向き合う決断をし、適切なテクニック（ピールダウンなど）を使えば、解放が可能。
