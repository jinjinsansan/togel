# ネガティブは選んでいる

## 定義

ネガティブな感情は、外部の出来事によって自動的に発生するのではなく、私たち自身が「選んでいる」ものであるという考え方。出来事と感情の間には「意味づけ」があり、その意味づけによってネガティブを選ぶかポジティブを選ぶかが決まる。

## 背景・目的

### よくある誤解

多くの人は次のように思い込んでいる：

```
【思い込み】
嫌な出来事が起きた → だからネガティブが発生する
```

この考え方では、出来事がネガティブを「生み出している」と解釈している。

### 実際の仕組み

```
【実際】
出来事が起きる → 意味づけをする → ネガティブを選ぶ
```

ネガティブは出来事から自動的に発生するのではなく、意味づけを通じて「選んでいる」。

## 夢の例による証明

### 夢を見るという現象

一番分かりやすい証明は「夢」。

**体験：**
- 夢の中で何か幸せな出来事が起きる
- 朝起きた時にとても幸せな感覚で起きる

**ポイント：**
- 何も出来事は起きていない（現実には何もなかった）
- 何も出来事が起きていないのに、心の感覚はポジティブになっている

### この現象が示すこと

- **出来事がなくてもポジティブになれる**
- **出来事がなくてもネガティブになれる**
- つまり、**出来事と感情は直接結びついていない**

## 心の仕組み

### 選べるようになっている

私たちの心の仕組みは：
- 出来事がどうであっても
- 出来事が起きていても起きていなくても
- **ポジティブもネガティブも選べる**ようになっている

### 意味づけのスピード

問題は、意味づけをしているスピードが**光のスピードぐらい速い**こと：
- あまりにも速いので意識できない
- 心を見る訓練をしていなければ追いつかない
- だから「出来事→ネガティブ」と思い込んでしまう

## 「選んでいる」という認識

### 今日からできること

ネガティブを感じている時：

**❌ 従来の認識**
「不幸なことばかり起きている→だからネガティブ→だから無価値」

**⭕ テープ式の認識**
「私はネガティブを選んでいる」
「わざと選んで無価値になっている」
「わざわざ寂しくなっている」
「わざわざ馬鹿にされたと感じている」

### なぜこの認識が大事か

「選んでいる」と認識することで：
- 被害者意識から脱却できる
- 自分の心の主導権を取り戻せる
- 変化の可能性が開ける

## 選ぶ力を覚えておく

### 仁さんからのメッセージ

**「ポジティブを選ぶのもネガティブを選ぶのも、選べるということだけを覚えておいてください」**

これは：
- 最初から最後まで伝え続ける
- 何度も何度も言い続ける
- 最も大事なことの一つ

### 今日ネガティブを感じている人へ

ネガティブを感じていても構わない。
ただし、次のことだけ覚えておく：

**「選んでいるんだ」ということ**
**「わざと選んでいる」ということ**

ここまでがとても大事。

## 出来事は全く関係ない

### 根本的な理解

**出来事は全く関係ない**

- 出来事が起きたからネガティブではない
- 出来事が起きても起きなくてもネガティブを選べる
- 出来事が起きても起きなくてもポジティブを選べる

### 例

| 状況 | 従来の解釈 | テープ式の解釈 |
|------|-----------|--------------|
| 嫌なことが起きた | 「だからネガティブ」 | 「私がネガティブを選んだ」 |
| 何も起きていない（夢） | 「ポジティブになれる」 | 「心は選べるようになっている」 |
| 同じ出来事を体験 | 人によって反応が違う | 意味づけが違うから |

## 実践への応用

### ネガティブを感じた時

1. **気づく**：「今、ネガティブを感じている」
2. **認識する**：「これは私が選んでいる」
3. **覚えておく**：「選べるということ」

### 注意

これは「ネガティブを感じるな」という意味ではない：
- ネガティブを感じてもいい
- ネガティブは悪いことではない
- ただし「選んでいる」という認識を持つ

## リフレーミングへの応用

### 「置いていかれている」という思い込み

例えば、グループで学んでいる時：
- 「置いていかれている」と感じる

**リフレーミング：**
- 置いていかれているも何も、誰も進んでいない
- 置いていかないという事実がある
- 「置いていかれている」という意味づけを選んでいる

### ネガティブな意味づけを見つける

自分がどんな意味づけをしているか：
- どんな言葉で自分に語りかけているか
- その言葉は事実なのか、思い込みなのか
- 別の意味づけは可能か

## よくある誤解・注意点

- ❌ 「選んでいる」＝ネガティブを感じる自分が悪い
- ⭕ 選んでいるという認識は、変化の可能性を開くためのもの。自分を責めるためではない

- ❌ ポジティブだけ選べばいい
- ⭕ ネガティブもポジティブも両方大事。「選べる」ということを知ることが大事

- ❌ 嫌な出来事が起きても平気でいるべき
- ⭕ 感情を感じることは自然。ただし出来事と感情は直接結びついていないと知る

- ❌ すぐに選び変えられるはずだ
- ⭕ 長年の意味づけパターンがある。時間をかけて気づいていく

## まとめ

ネガティブは出来事によって自動的に発生するのではなく、私たち自身が「選んでいる」もの。夢を見て幸せな感覚で起きることがあるように、出来事がなくても感情は発生する。これは心が「選べる」ようになっている証拠。ネガティブを感じている時は「私はこれを選んでいる」と認識することが重要。出来事は全く関係なく、意味づけによってネガティブもポジティブも選べる。この「選べる」という事実を覚えておくことが大切。
