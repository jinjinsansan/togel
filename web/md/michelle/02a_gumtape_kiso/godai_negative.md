# 五大ネガティブ

## 定義

テープ式心理学では、人間が感じる根本的なネガティブ感情を5つに分類し、「**五大ネガティブ**」と呼ぶ。

- **恐怖**
- **悲しみ**
- **怒り**
- **寂しさ**
- **無価値観**

ポイント: ネガティブ＝悪い・ダメな感情ではない。ポジティブもネガティブもどちらも素敵な感情であり、単に「気分が良いか悪いか」の違いである。

## 各感情の特徴

### 恐怖

| 項目 | 内容 |
|------|------|
| 身体的感覚 | 体がぐーっとすくむ |
| 涙の流し方 | 声が小さくなる、すくんだ泣き方 |
| 解放方法 | **勇気**を出す |

### 悲しみ

| 項目 | 内容 |
|------|------|
| 身体的感覚 | 胸に穴がぽっかり開いたような感覚 |
| 涙の流し方 | - |
| 解放方法 | **時間**が解決する。吐き出しを受け止め続ける |

### 怒り

| 項目 | 内容 |
|------|------|
| 身体的感覚 | 頭に来る、肩に来る、お腹でぐっと我慢 |
| 涙の流し方 | 歯を食いしばって泣く |
| 解放方法 | **許し**。ガードの怒りの下にある本当の感情を掘る |

ポイント: 怒りは「ガードの怒り」であることが多い。怒りの下には必ず他の感情（無価値観、寂しさ、恐怖など）が隠れている。

### 寂しさ

| 項目 | 内容 |
|------|------|
| 身体的感覚 | シューンとなる、孤独感 |
| 涙の流し方 | しくしく泣く |
| 解放方法 | 「みんな繋がっている」という全体性に気づく + 自らの手足を使って行動する |

### 無価値観

| 項目 | 内容 |
|------|------|
| 身体的感覚 | 足元がなくなる、自分が小さくなる |
| 涙の流し方 | ギャーと泣く |
| 解放方法 | 「ありのままの私で十分価値がある」と気づく |

## 罪悪感の特殊な位置づけ

五大ネガティブに加えて、**罪悪感**は特に重要な感情として扱われる。

### 五大ネガティブ＋1

- 罪悪感は五大ネガティブに近いほど強い感情
- カウンセリングの申し込み理由として挙がることがある
- 「六大ネガティブ」とは呼ばず、「五大ネガティブ＋罪悪感」という位置づけ

### なぜ五大ネガティブなのか

心理カウンセラー（テープ式）が対応するネガティブは、ほとんどがこの五大ネガティブだから。

**カウンセリングの申し込み理由として稀なもの：**
- 「恥ずかしいのでカウンセリングお願いします」→ ほぼない
- 「悔しいのでカウンセリングお願いします」→ ほぼない
- 「情けないのでカウンセリングお願いします」→ ほぼない

**カウンセリングの申し込み理由として多いもの：**
- 五大ネガティブ（恐怖・悲しみ・怒り・寂しさ・無価値観）に関連する苦しみ

### 五大ネガティブの影響力

五大ネガティブは**人生を変えてしまうぐらいの強烈なネガティブ**。他の感情（恥ずかしい、悔しい、情けない等）も苦しいが、五大ネガティブが最強。

## その他のネガティブ感情

五大ネガティブ以外にも以下のような感情がある。これらは五大ネガティブから派生していることが多い。

- 悔しい
- 恥ずかしい
- 苦しい
- 情けない
- 劣等感
- 嫌だ・嫌い（ネガティブな感情の一種）

### 「絶望」について

「絶望」という言葉を使う人がいるが、ピールダウンしていくと無価値観に繋がることが多い。

## 中核ポイント

- 全てのネガティブな感情は種類が異なり、感じる身体的感覚も違う
- 心の中はスパゲティのようにぐちゃぐちゃで、何を感じているか見えにくい
- テープ式心理学では「感情の種類を分けていく」ステップが非常に重要
- 感情の種類ごとに解放の仕方が異なる

## 日本人が混同しやすい感情

### 「悲しい」の誤用

日本人は何でも「悲しい」と表現する傾向がある（国語の教科書の影響）。

**テープ式心理学での「悲しみ」の定義：**
- 失う感覚、喪失感
- 大好きだったおばあちゃんが亡くなった時
- ペットのワンちゃんが亡くなった時
- 「失った」「なくした」という感覚

**よくある間違い：**

| 間違った表現 | 正しい感情 |
|-------------|-----------|
| 彼氏からLINEの返事が来なくて**悲しい** | **寂しい** |
| 失敗して**悲しい** | **無価値観** |
| 理不尽なことがあって**悲しい** | **怒り** |

### 寂しさと悲しみの違い

| 寂しさ | 悲しみ |
|-------|-------|
| 繋がりを求める | 喪失感 |
| 「そばにいて」 | 「もういない」 |
| 孤独を感じる | 穴がぽっかり開く |

### 寂しさと無価値観の違い

| 寂しさ | 無価値観 |
|-------|---------|
| 「一人ぼっち」 | 「私はダメだ」 |
| 繋がりを求める | 存在価値を疑う |
| シューンとなる | 足元がなくなる |

### 怒りと恐怖の見分け

怒りと恐怖は比較的分かりやすいが、寂しさ・悲しみ・無価値観は混同しやすい。感情日記などで訓練が必要。

## カウンセリングでの使い方

1. クライアントが「苦しい」と言っている場合、それが五大ネガティブのどれに該当するかを特定する
2. ピールダウンを使って核となる感情を見つける
3. 感情の種類が特定できたら、その感情に応じた解放テクニックを使う

## よくある誤解

- ❌ ネガティブ感情は悪いもの、排除すべきもの
- ⭕ ネガティブ感情は気分が悪いだけで、悪いものではない。次のステージに進むための必要な体験

## まとめ

五大ネガティブ（恐怖・悲しみ・怒り・寂しさ・無価値観）を理解することで、クライアントの苦しみの根本を特定できる。感情の種類によって解放方法が異なるため、まず「何を感じているか」を明確にすることがカウンセリングの第一歩となる。
