# 認知の歪みという概念への批判

## 定義

テープ式心理学では「認知の歪み」という表現を使用しない。認知（解釈）が異なるのは、ガムテープ（トラウマ）があることで生じる自然な反応であり、「歪んでいる」と否定されるべきものではない。多数派と異なる解釈を持つことは、単に少数派であるだけ。

## 背景・目的

### 一般的な「認知の歪み」の定義

精神医学・心理学で広く使われる用語：

> 「認知とは、その人自身の物事の捉え方や考え方を意味する。いつも自分を精神的に追い詰めてしまったり、些細なことで落ち込んだり、すぐに悪い方向に考えてしんどくなってしまう考え方のパターンを『認知の歪み』という」

### 認知療法の説明

> 「認知の歪みに焦点を当て修正をしていくことで、症状などを軽減していく療法。患者の偏った物事の捉え方（認知）を修正させ、より柔軟的で現実的な考え方や行動ができるように手助けする」

### つまり精神医学では

- ネガティブ側で解釈する人＝認知が歪んでいる
- 「あなたは歪んでいる」と表現している
- 多数派が正しく、少数派が歪んでいるというレッテル

## 中核ポイント

### テープ式心理学の見解

この概念は**精神病を作り上げるための嘘**である。

### 電車の例え話

| Aさん（多数派） | Bさん（少数派） |
|----------------|----------------|
| 毎日電車に乗る | 電車に乗ると気分が悪くなる、パニックになる |
| ストレスなく通勤 | 呼吸困難になることがある |

**問い：Bさんの認知は「歪んでいる」のか？**

「歪んでいる」という言葉は、何かを否定しているように聞こえる。Bさんは「治療すべきである」「治すべきである」とされてしまう。

### 偉人たちの例

- アインシュタイン
- コロンブス
- 多くの偉人たち

彼らは大多数派の言っていることの真逆の論理を展開していた。これらの偉人たちは「認知が歪んでいた」のか？

### 基準を作っているのは誰か

「認知の歪み」という基準そのものを作っているのは精神医学。精神病名を付与するための概念に過ぎない。

## ガムテープとの関係

### 心に深い傷がある場合の解釈

幼少期から心に深い傷（ガムテープ）がある状態で社会に出ると、大多数派と意味づけが異なることがある：

- 大人を全く信頼できない
- 自分には全く価値がない
- 常に自分は孤独である
- 常に怖い

### 具体例：新人研修の場面

| 大多数派 | 心に傷がある状態 |
|---------|-----------------|
| 気軽に、気楽に、ストレスなく挨拶 | みんなが私を見ている... |
| 仲良くなれる | みんな私を馬鹿にしている... |
| | みんな私の話をしている... |
| | みんな私を嫌っている... |

### これはガムテープが原因

- 思い込みがあるから
- トラウマがあるから
- 幼少期からそのガムテープを貼り付ける原因があった
- **決してあなたが悪いのではない**

## 正しい理解

### 認知が歪んでいるのではない

```
❌ あなたの認知は歪んでいる
⭕ ガムテープがあることでそのように感じてしまうだけ
```

### それは今のタイミングだけ

- 何にも悪いことではない
- ましてや「歪んでいる」と否定される必要もない
- ガムテープを解放すれば、解釈は変わる

### この世界に正しい・正しくないは

- 法律では存在する
- しかし**人の考え方そのものには正しい・正しくないは存在しない**

## 精神科医に言われた場合の対応

「あなたは認知が歪んでいる」と言われたら：

**「そうですね〜だって私は少数派だも〜ん」**

と回答する。

### 自分を否定しない

- 私は歪んでいるからダメな人間などと考えない
- 認知は歪んでいるのではない
- 今の時代で、たまたまあなたの考えが少数派なだけ

## よくある誤解・注意点

- ❌ 認知の歪みは治療すべき病気
- ⭕ ガムテープがあることで解釈が異なるだけ。ガムテープを解放すれば変わる

- ❌ 多数派の考え方が正しい
- ⭕ 考え方に正しい・正しくないは存在しない。角度が違うだけ

- ❌ 少数派の考えは間違っている
- ⭕ 偉人たちも少数派だった。少数派であることは悪いことではない

- ❌ 認知が歪んでいると言われたら自分はダメな人間
- ⭕ 少数派であるだけ。自分を否定する必要はない

## まとめ

テープ式心理学では「認知の歪み」という表現を使用しない。この概念は多数派を正しいとし、少数派を「歪んでいる」と否定するレッテルであり、精神病名を付与するための嘘である。心に深い傷（ガムテープ）がある場合、解釈が異なるのは自然なこと。それは「歪み」ではなく、ガムテープの影響。ガムテープを解放すれば解釈は変わる。今の時代でたまたま少数派なだけであり、自分を否定する必要は全くない。
