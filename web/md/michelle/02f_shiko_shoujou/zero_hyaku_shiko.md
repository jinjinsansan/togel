# 0-100思考（白黒思考）

## 定義

物事を「0か100か」「白か黒か」で捉える思考パターン。曖昧な状態を許容できず、全てに明確な意味づけ・答えを求める。虐待を受けた子供時代に形成されることが多い。

## 精神医学界での扱いへの注意

### 現代の「発達障害ブーム」

- 0-100思考、白黒思考をすること自体が発達障害と勝手に結びつけられている
- 精神医学界は「やりたい放題やっている」状態
- **テープ式心理学では、これは形成されるメカニズムを持つ心理的パターンとして捉える**

## なぜ0-100思考が形成されるのか

### アーキタイプ（元型）

全ての人間に備わっているプログラム：
- 親は子を愛する
- 子は親を愛する
- 兄弟も愛する
- **何もしなくてもこれがデフォルト**

動物も同じ：
- ライオンに食べられそうな時、自分が犠牲になってでも我が子を守る
- 理由で説明できないぐらい我が子を愛する

### デフォルトの真逆が起きている

- アーキタイプ通りなら苦しまない
- **真逆のことが起きているから苦しむ**
- 愛されているはずなのに愛されていないと感じる
- この矛盾の中で子供は生きていかなければならない

## 曖昧な状態に耐えられない

### 愛されているのか、愛されていないのか

虐待を受けている子供の心の中：
- 「私は愛されているのか、愛されていないのか」がわからなくなる
- 自分の心のバランスが崩れていく

### 曖昧では立っていられない

- どんどん崩れそうになる
- 曖昧な状態では壊れそう
- 立っていられない
- **だから答えが必要**

## 意味づけを欲する

### 合っていなくても答えが欲しい

- 答えが合っていても合っていなくても
- 必ず意味づけをしたい
- 答えを見つけたい
- **こういうことにしたい**

### 子供の意味づけの例

```
実際：
母親がただネガティブをコントロールできなくてバンと叩いた

子供の解釈：
「私が成績が悪かったから今日は怒られたんだ」
「こうだったからお母さんは今日怒っているんだ」
```

### 虐待される根拠がわからない

- なぜ虐待されているのかわからなすぎる
- 答えを見つけないと収まらない
- 解放処理ができない
- **だから0か100か、白か黒かで全部に理由をつけて生きていくしかなかった**

## 大人になってからの影響

### 思考パターンの固定化

- 小さい頃からずっとそうして生きてきた
- 長く生きてきたのでこれがデフォルトになった

### 全ての事象に対して

- これはこうだ
- これはこうだ
- 全部に理由をつける

### 価値についても

```
0-100思考の例：
「私は価値がない」⇔「私は価値がある」
「これを頑張ったら価値がある」
「1個でも崩れたら全部価値がない」

意味がある⇔意味がない
という意味づけで世界を見ている
```

## 動画教材の例（機能不全家庭の女の子）

### 愛されているか愛されていないかわからない毎日

- 母親の機嫌で環境が激変する
- 普通の日と暴力的な日の落差
- 自分がこの家にいていいのかわからない

### 常に答えを出して生き延びる

```
子供の心の中：
「こうだったからこうだったんだ」
「私が悪かったんだ」
「お母さんはこうだったんだ」
「私が悪かったから」
→ 全部に理由が欲しい
→ そうでないと保っていられない
```

### ガムテープを貼っていく

- 曖昧な状態では壊れそうになる
- だから0-100、白黒になっていく
- これがガムテープとして貼られていく

## 0-100思考からの解放

### 気づくことが第一歩

- 自分が0-100思考をしていることに気づく
- それが幼少期に形成されたパターンだと理解する

### 曖昧を許容する

- 世の中には曖昧なこともある
- どちらでもいいこともある
- 答えがないこともある

### 意味づけを変える

- 意味づけは自分で選んでいる
- 別の意味づけも選べる

## よくある誤解・注意点

- ❌ 0-100思考は発達障害の症状
- ⭕ 幼少期の環境で形成された思考パターン。解放可能

- ❌ 生まれつきの性格
- ⭕ 曖昧な状態に耐えられない環境で生き延びるために身につけた

- ❌ 治せない
- ⭕ ガムテープを剥がすことで解放できる

## まとめ

0-100思考（白黒思考）は、虐待などで「愛されているか愛されていないか」がわからない環境で育った子供が、曖昧な状態に耐えられず、全てに意味づけ・答えを求めることで形成される。これは生き延びるための適応だった。大人になってもこのパターンが続くと、全ての事象を「こうだ」「こうじゃない」と二極化して捉え続ける。しかしこれは心理的パターンであり、ガムテープを剥がすことで解放できる。
