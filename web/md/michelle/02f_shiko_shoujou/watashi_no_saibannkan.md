# 私の中の裁判官

## 定義

トリシアから教わった概念。頭の中に「正しい/正しくない」を常に判断する裁判官を抱えていて、これによって苦しんでいる場合、その裁判官をクビにする（解雇する）ことでネガティブを解放できる。

## 世の中に「正しい」はあるのか

### 広い視点で見た場合

この世界の「正しいこと」「正しくないこと」というのは：
- すごく広い視点で見た場合、**ない**
- 人の意見、いろいろな考え方がある
- **どれが正しいのか、正しくないのかは断定できない**

### 法律的な正しさはある

- 法律的に「これをしてはダメ」はある
- 人のものを取ってはダメ
- 法的に正しい/正しくないは存在する

### しかし倫理・考え方については

- 人の理念、倫理、考え方について
- 様々な角度がある
- **どれが正しいとは一概に言い切れない**

## 様々な角度の例

### 国際情勢の例（ロシア・ウクライナ）

```
一般的な日本の報道：
「プーチンが悪い」「ロシアが悪い」「ウクライナがかわいそう」
→ 私たちはその通り思い込む（特に日本人はニュースを信じる）

しかしプーチンの演説：
「最も悪いのはアメリカでしょう」
「アメリカは日本に核を2発撃った」
「イラクなども全部侵略して資源を取っている」
「やりたい放題しているのはアメリカですよ」

→ 角度によって正しい/正しくないは変わる
```

### テープ式心理学自体も

- 仁さんはむちゃくちゃ自信がある（たくさんゴールに連れて行った人がいるから）
- しかし「精神医学が正しいのか正しくないのか」は様々な議論がある
- **どれが正しいとは広い視点で見たら言えない**

### あなたの意見はあなたのもの

結論：
- 「あなたの意見はこうなのね」
- これがフラットな立場

## 裁判官を抱えて苦しむ

### 常に正しい/正しくないを判断

- 頭の中に裁判官がいる
- 正しい、正しくない
- 正しい、正しくない
- 常に考えすぎている

### 虐待経験者に多い

よく虐待されてきた方の体験：
- 親が「世間はこう言っている」「世間は」「世間は」と話し続ける
- 「普通はこう」「普通はこう」
- この裁判官で苦しんでいる

## 裁判官をクビにする

### トリシアの教え

もし裁判官を抱えていて苦しんでいるなら：
- その裁判官を**クビにしてください**
- 退職させてください
- 雇いっぱなしにしないで
- **お疲れ様**と解雇する

### クビにするとどうなるか

- 「どれが正しい」「何が正しい」がなくなる
- どのことだって、こっち側の意見の人もいればあっち側の意見の人もいる
- **何が正しいなんてわからない**
- という場所に立てる

### 苦しくなくなる

```
裁判官がいる状態：
「世間はこうかもしれない」
「普通はこう」
「だから私は間違っている」
→ 苦しい

裁判官をクビにした状態：
「世間はこうかもしれないけど、普通じゃないかもしれない」
「大多数の方はこう言っているけど少数派だっている」
「ロシアはこうって言っているけど実際はアメリカだって…」
→ 苦しくない
```

## 具体例：リストラ

### 一つの事象に対する様々な見方

```
状況：45歳でリストラされた

Aさんの見方：
「この厳しい時代でリストラされたらまずいよ」
「人生終わりだ」

Bさんの見方（余命1ヶ月のがん患者）：
「リストラされたところでまだまだやれることあるでしょ」
「思う存分、命があるんだからやってみなさい」
```

→ 一つの事象に対してもいろんな考え方がある

## ふわっとした柔らかさ

### 裁判官がいない状態

- 「これも正しい」「私は正しい人間だ」と必死にならなくていい
- **ゆるく**
- **それもいいけど、でもこれもいいんじゃないの**
- という感じ

### 選べる自由

- 私は常にどちらでも選べる
- 選びたいところを選べる
- 「これもいいな」「あれもいいな」
- **苦しくなくなる**
- **ガムテープが剥がれる**

## ガムテープ解放の一手法

### 私の中の裁判官というテーマ

これは：
- ネガティブを解放する一つの手法
- ガムテープを剥がす時のきっかけの考え方

### 実践方法

1. 「私の中に裁判官がいないか」を確認する
2. 「正しい/正しくない」で苦しんでいないかを見る
3. 裁判官がいたら「お疲れ様」とクビにする
4. 様々な角度・立場があることを受け入れる
5. ふわっと柔らかく、どちらでも選べる状態になる

## よくある誤解・注意点

- ❌ 正しい/正しくないをはっきりさせることが大事
- ⭕ 様々な角度があり、一概に正しいとは言えないことが多い

- ❌ 世間や普通に従うべき
- ⭕ 世間はこうかもしれないが、そうじゃないかもしれない

- ❌ 裁判官がいることで正しく生きられる
- ⭕ 裁判官がいることで苦しんでいるなら、クビにしていい

## まとめ

頭の中に「正しい/正しくない」を判断する裁判官がいて、それで苦しんでいるなら、その裁判官をクビにする。法律的な正しさはあるが、倫理や考え方については様々な角度があり、何が正しいかは断定できない。裁判官をクビにすると、「これもいい、あれもいい」とふわっと柔らかく選べる状態になり、苦しみから解放される。これはガムテープを剥がす一つの手法として使える。
