# 無気力状態と心身のリンク

## 定義

テープ式心理学では、無気力状態になることを**3ステップ**、または**うつ状態**と呼ぶ。

無気力とは文字通り、気力が無い、何にもやる気が出ない、という状態。

## テープ式心理学の見解

### 精神医学との違い

| 精神医学 | テープ式心理学 |
|---------|---------------|
| 精神病とする | 精神的な病気など存在しない |
| うつ病などの病名をあてはめる | 3ステップ、うつ状態と呼ぶ |

### 無気力状態の定義

**ネガティブな感情が長期間に渡り心の中にあり続けることで現れる身体的特徴**

## 心と身体のリンク

### 私を構成する3つの要素

```
1. Body（ボディ）- 身体
2. Mind（マインド）- 心
3. You（ユー）- 言葉では表せないもの
```

今回は1と2、つまり**身体と心**について説明する。

### 身体と心は別々に存在しながらリンクしている

身体と心は別々に存在しながら、**常に繋がった状態、リンクしている状態**であるとテープ式心理学では考える。

### パターン1：心→身体

```
何かストレスがかかる出来事が起きた（例：失恋）
    ↓
心が苦しい
    ↓
身体も不調になる
```

**①心がネガティブになる → ②身体が不調になる**

### パターン2：身体→心

```
冬の寒い時に風邪をひいた
    ↓
身体がしんどい
    ↓
心も落ち込む
```

**①身体が不調になる → ②心がネガティブになる**

### 相互作用

このように、どちらも別々で存在しながらも**相互作用をしあっており**、常にリンクしている。

### パターンの見分け方

心の中を観る訓練ができている方であれば、①が心なのか、身体の不調なのか、を見分けることができる。

| パターン | 対処法 |
|---------|-------|
| 心→身体 | 心の中と向き合ってネガティブを解放 → 身体の不調も消える |
| 身体→心 | 身体を休めたり、病院で治療 → 心もリラックスする |

## ネガティブ感情の身体的特徴

心の中にネガティブな感情を長期間に渡り感じており、その相互作用として現れる身体的特徴：

### 感覚系
- ソワソワする
- モヤモヤする
- ゾワゾワする
- イライラする
- フワフワする
- ドキドキする
- 緊張している
- 落ち着かない

### 日常生活への影響
- 眠れない
- ご飯が美味しくない
- 楽しくない
- 太陽が嫌い
- カーテンをあけたくない

### 身体症状
- 胸のあたりが気持ち悪い
- パニックが起きる
- 肩こり
- 頭痛
- 腹痛
- 胃が痛い
- 腸が痛い

### 心理症状
- 解離現象
- 人格分裂
- 視野が狭い
- **無気力状態**

## 無気力状態の仕組み

無気力状態というのも、ネガティブな感情が長期間あることで、相互作用としてリンクしている身体に「無気力」が現れている。

**逆に、相互作用としてネガティブな感情が解放されてしまえば、無気力状態も終了する。**

## 無気力状態への対処法

### レベルによって対処法が変わる

無気力状態のレベルによっても、どのようにすればいいのかが変わってくる。

---

## 例1：重度の無気力状態

### 状態

- 無気力状態が**1ヶ月を超えて**続いている
- 仕事や学校も全て休んでいる
- お風呂などの日常生活もあまりできない

### 危険度

**非常に緊迫した状態。** このままネガティブな感情を感じ続けると、命に関わることを想定したりなどの段階に至る。

### 対処法：まずは休む

「休む」というのは身体的にではなく、**心の中を休める**。

心の中を休めるとは：

```
ネガティブな感情を感じることを認める
ネガティブな感情を感じることを嫌がらない
ネガティブな感情を感じることを抵抗しない
    ↓
早く治さなきゃなど慌てない
周囲の人々の否定的言葉を受け流して自分を守る
ゆっくりしている自分、休んでいる自分を許す
    ↓
なにもしない。なにも考えない。
とにかく休む。休み。休め。
```

### 回復のプロセス

```
上記を繰り返す
    ↓
ある日、非常に楽になる日がやってくる
    ↓
楽になる日を見つけて、少し心の中を勉強する訓練をする
    ↓
また無気力状態になったら「休む。休み。休め。」を繰り返す
```

---

## 例2：なんとか動ける無気力状態

### 状態

- なんとか仕事や学校には頑張って行っている
- けれど心の中は完全な無気力状態
- いつ寝込んでしまうか？というレベル

### 対処法：共通部分

まずは例1と同じ：

```
ネガティブな感情を感じることを認める
ネガティブな感情を感じることを嫌がらない
ネガティブな感情を感じることを抵抗しない
```

### 対処法：自分に問いかける

例1と違う部分は、まだなんとか動くことができている。

そのため、**自分に問いかける**：

```
「このまま、ネガティブな感情を感じ続けると
 いよいよ仕事も学校も行けなくなるけれど
 どうしたい？」

「ネガティブな感情を感じ続けるのは
 何にも悪くないからね。どうする？
 もう寝込んでしまうのも良いし、
 ネガティブと向き合うのも良い」
```

### 答えによる分岐

| 答え | 対処 |
|-----|------|
| 「まだまだ休みたい」 | 例1のレベルに進む（何も悪くない） |
| 「ネガティブと向き合いたい！これ以上無気力は嫌だ！」 | 心の仕組み、テープ式心理学を少しだけ学ぶ |

## よくある誤解・注意点

### 無気力は病気？

- ❌ 無気力状態＝うつ病という精神病
- ⭕ ネガティブな感情が長期間あることで現れる身体的特徴

### 無気力は身体の問題？

- ❌ 無気力は身体だけの問題
- ⭕ 心と身体はリンクしており、心のネガティブが身体に現れている

### 無気力は薬で治る？

- ❌ 薬を飲めば無気力が治る
- ⭕ ネガティブな感情が解放されれば無気力状態も終了する

### 休むのは悪いこと？

- ❌ 休んでいる自分はダメ
- ⭕ 休んでいる自分を許す。何も悪くない

## まとめ

テープ式心理学では無気力状態を3ステップ、またはうつ状態と呼び、精神病とは考えない。無気力状態は、ネガティブな感情が長期間心の中にあり続けることで現れる身体的特徴。身体と心は別々に存在しながら常にリンクしており、相互作用している。無気力状態のレベルによって対処法が異なり、重度の場合はまず「休む」こと、なんとか動ける場合は自分に問いかけて向き合うか休むかを選ぶ。ネガティブな感情が解放されれば、無気力状態も終了する。
