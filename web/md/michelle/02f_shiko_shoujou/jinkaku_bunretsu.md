# 人格分裂の仕組み

## 定義

### 人格分裂とは

主人格から副次人格に切り替わる際に**記憶が伴わない**状態。厳密に二重人格者・多重人格者というならば、**全く記憶がない方**を指す。

### 実際にいるのか

本当にいる。ただし**稀**。10年間で対応したのは3名レベル。ほとんどの人が自分が人格分裂していることを**言わない・隠す**。

## 背景・目的

### 心はバランスでできている

私たちの心はバランスで作られている。これは心だけでなく、この世界の全て（体、細胞、心臓、脳、指、木、森、空気、海、重力）が**バランスよくできている**。なぜかは分からないが、常にバランスが保たれるようになっている。

### 赤ちゃんの状態

おぎゃあと生まれた赤ちゃんの時は、ハート（心）がバランスよく保たれた状態で「私」が存在している。

### ストレスによるバランスの崩れ

機能不全家庭における虐待によって強いストレスを受けると、バランスが保たれているハートが右や左に傾き、**バランスを崩す＝耐えられない**状態になる。

## 中核ポイント

### 「生」というプログラム

私たちには**生きていくというプログラム**が元々備わっている。脳にも心にも「死んではいけない」というプログラムがある。

そのため、虐待やストレスで心がぐっと崩れそうになった時、**なんとかバランスを保とうとする**。**形を崩してでも**。

### バランスを保つ方法の段階

| 段階 | 方法 | 説明 |
|-----|------|------|
| 1 | **ガムテープ** | 「私は価値がない」等の思い込みでバランスを保つ |
| 2 | **感情封印** | 感情そのものを殺してしまう。喜びも幸せも何もかも感じない |
| 3 | **依存** | 何かに依存することでバランスを保つ |
| 4 | **人格分裂** | 上記で間に合わない時の最終手段 |

### 人格分裂が起きるレベルの虐待

人格分裂が起きるまでに及ぶ虐待の例：
- 肉親によるレイプなどの性的虐待
- 戦争で目の前で肉親が次々に残酷に殺されるのを見る
- 両親共に守ってくれない環境での残酷な体験

**注意**: 苦しみは人によって違う。数値化できるものではない。その人にとって「地獄だった」と感じたものは全てそう。

### 人格分裂の仕組み

```
あまりにひどいストレス・虐待を受ける
    ↓
ガムテープで間に合わない
感情封印で間に合わない
依存で間に合わない
    ↓
主人格がこれ以上耐えられない
    ↓
主人格が消える（寝る）ことを選ぶ
    ↓
副次人格が交代して私の代わりをする
```

### 主人格と副次人格の関係

**通常の状態**:
- 私たちは副次人格（様々なパート）を持っている
- 男性でも母性、女性でも父性を持っている
- 優しさ、子供っぽさなど様々な副次人格がある
- 通常は**コントロールできる**（今日は父親らしくするぞ、など）

**人格分裂の状態**:
- 主人格が**機能しない**（拒否・拒絶）
- 生きる気力がない、全部放棄している状態
- 副次人格に任せている
- 主人格は**「自分がない」状態の最強版**

### 副次人格の特徴

| 項目 | 説明 |
|-----|------|
| 名前 | それぞれ決まっている |
| 声 | 全く違う、別人 |
| 性格 | それぞれ違う |
| 性別 | 違うこともある |
| 年齢 | 違う |
| 国籍 | 外国人もいることがある |

### 全ての副次人格の目的

**全員が主人格を守るために現れた人格**。

例えば怒りのパートは、どのようなやつが近づいてこようと全部ぶった切る。主人格を守っている。

### 記憶の断絶

**具体例**:
```
主人格がお昼にカフェでパンを食べ、コーヒーを飲んだ
    ↓
自宅に帰ってきてすっと副次人格に変わった
    ↓
副次人格が誰かとLINEをした
    ↓
また主人格に戻った
    ↓
主人格は副次人格がLINEしていたことを知らない
```

記憶が**繋がっていない**。

### 主人格と副次人格の視点

| 主人格 | 副次人格 |
|-------|---------|
| 副次人格の動きが見えない | 奥から主人格の動きを見ている |
| 見ようともしない | 自分の動きも主人格の動きも分かっている |
| 生きる気力がない | 主人格を守っている |
| 全部放棄している | 交代して代わりをする |

### 熟練すると

- 主人格も人格分裂していることに気づいてくる
- 各人格がコミュニケーションを取り合う
- 副次人格が主人格に手紙を残すこともある
- コントロールして人格を変えられるようになる（面倒な人が来たら副次人格に対応させる等）

## 統合について

### 病気ではない

精神医学では「解離性同一性障害」として病気にしてしまうが、テープ式心理学では**何の病気でもない**。

精神的に強いストレス・虐待を受けた時に、ガムテープ→感情封印→依存でも間に合わず、**人格分裂しか生きる術がなかった**時に分裂する。

### 必ず統合できる

何人に人格分裂しようとも、**必ず1人しかいない**。全て私から派生しているだけで**全部私**。

名前が何人出てこようとも、人格が何分裂しようとも、声が何人変わろうとも、結局は私から派生しており、**必ず1に戻る**。

## よくある誤解・注意点

### 演技ではない

- ❌ 人格分裂は演技や嘘
- ⭕ 本当に起きている。声も性格も全く違う別人になる

### 珍しいが確実に存在する

- ❌ 人格分裂はフィクションの世界だけ
- ⭕ 稀だが必ずいる。ほとんどの人は隠している

### 主人格へのメッセージ

主人格はずっと「私はおかしいんだ」「死ぬしかないんだ」という状態。しかし必ず統合できる。

## まとめ

人格分裂は、あまりにひどい虐待・ストレスを受けた時に、ガムテープ・感情封印・依存でも間に合わず、生きるために主人格が消え（寝て）、副次人格が交代して代わりをする仕組み。全ての副次人格は主人格を守るために現れた。記憶は断絶するが、何人に分裂しようとも全て私から派生しており、必ず1に戻ることができる。病気ではなく、統合可能。
