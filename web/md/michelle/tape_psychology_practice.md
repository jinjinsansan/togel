## トピック88：臨床心理士との違い（テープ式の優位性）

### 臨床心理士とは
- 国家資格で心理カウンセラー資格の最高峰
- 大学4年間で心理学を学び、大学院に進み、試験に合格して取得
- 表向きの心理カウンセラー資格としては最高峰

### 臨床心理士のカウンセリング方法（直接聞いた内容）
1. 症状をヒアリングする
2. 現状何が起きているかを聞く
3. 起きたことを分析する（動悸がする、緊張する、パニックが起きているなど）
4. 必要であれば認知行動療法、箱庭療法などの改善方法を提案
5. 経過観察する

### 臨床心理士の限界
**仁さんが意地悪な質問をした：**
- 「人格分裂している人が来た場合はどうするんですか？」
- 「感情封印している人が来た場合はどうするんですか？」
- 「言ってきている内容が嘘だったらどうするんですか？（ペルソナ）」

**臨床心理士の回答：「見抜けないです」**

### ペルソナの問題
- 相談者が「朝からパニックが起きて、電車で緊張して、人と喋るのが恐怖で...」と相談
- しかし本当はそれが全部嘘（ペルソナ）だった場合
- 臨床心理士は見抜けない

### 臨床心理士の現実
- クライアントが話している内容をただただ聞く
- それを分析する
- 「こうですね」とジャッジしているだけ
- **それしかできない**

### 他の資格も同様
- 公認心理士
- 産業カウンセラー
- 民間の心理カウンセリング資格
- **ほとんどが似たようなもの、傾聴がほぼ全て**

### テープ式心理学は全く違う
- ペルソナだろうが何だろうが通用しない
- 「私はもう楽なんです、向き合う必要もないんです」と言っても通用しない
- **心の仕組みは全員共通しており、反応するポイントがそれぞれあるから**
- 嘘が丸見え、ガラス張り

## トピック89：なりたい私（アラジンの魔法のランプ）

### 根底にある考え方
**全ての人間は例外なく幸せになりたいと思っている**

これがテープ式心理学の根底にある考え方。

- 幸せになりたいと願うから、高いお金をかけて2期生になる
- 幸せになりたいと願うから、インターネットで心理カウンセラーを探してくる
- **例外なく全員、世界中**

### カウンセラー側の視点
- 「この人は幸せになりたいからここに来てるんだな」と最初から見ている
- だから迷子にならない
- 「この人は絶対幸せになりたいんだ」と確信している

### ペルソナへの対応
- 「私別に幸せなんで今」「幸せとかあんまり興味ないんで」と言う人がいる
- **そんなわけない**
- 「何か隠しているな」「カウンセラーに見つかりたくないんだな」とすぐ分かる
- もう丸見え

### なりたい私とは
- 「幸せになりたい」は曖昧すぎる
- それを**言語化**する
- 「どう幸せになりたいの？」を明確にするツール

### アラジンの魔法のランプの聞き方

```
もしあなたの前にアラジンの魔法のランプがあったとして、
魔法使いのジーニーが現れました。

ジーニーがあなたにこう言いました：
「あなたは本当によく頑張って生きてきました。
よく頑張って生きてきたので、何でも願いを叶えてあげよう」

ただし、願いを叶えてあげるのは：
- 物理的なもの（物が欲しい）ではない
- お金（1億円欲しい）ではない
- 場所（ヨーロッパに行きたい）ではない
- 若返りたいとかでもない

「なりたい私」ならば何でも叶えてあげよう。

条件は一切なし：
- 結婚しているとか
- 男性だから、女性だから
- 仕事がどうだとか
- 住んでる場所がどうだとか
- お給料がどうだとか
一切抜きで「なりたい私」を書き留めてください。
数はいくつでもいい。
```

### なりたい私の本質
- 書いた「なりたい私」は**本当の私の願い**
- マインド様からの願い
- どれだけネガティブであっても出てくる
- **幸せになりたいから**

### 逆接的に捉える（重要）
**なりたい私 = 慣れていない私**

例：「人前で緊張せずに喋れるような私になりたい」
- 一見：「偉いね、頑張ってるね、素晴らしい目標だね」
- テープ式の見方：**「人前で話すことが怖いと感じている」**

**なりたい私の真逆を見ていく：**
- 「人前で話すことが怖い」というガムテープがある
- そのガムテープさえ剥がせたら、なりたい私になれる

### ガムテープを剥がせば叶う
- 「人前で話すのが怖い」というガムテープを1時間で剥がせたら
- その瞬間「人前で話せる私」になる
- **魔法は叶う**

### なりたい私の使い方
1. 心理カウンセリングの場面で**最初に聞く**（ガムテープの話をする前に）
2. カウンセリング中に迷子になったら**何度も見せる**
3. 「やるぞ」というマインド様を引っ張り出す

### なりたい私はナビゲーション
- カーナビと一緒
- 導いてくれる
- 迷子にならないために使う

### カウンセリング中の使い方
- 「もうどうでもいい」「もう無理」となった時
- 「なりたい私を見てください」と言う
- 「そうでした、私幸せになりたかったんでした」と思い出させる
- マインド様を引っ張り出す

### 破壊力のあるツール
- これだけで心理カウンセラーいらないくらい
- 毎日「なりたい私」をメモに書いて、逆を見つける
- 必ずガムテープがある（思い込みだから）
- 慣れないというのは思い込み
- 1秒先の未来は無限の可能性がある

## トピック90：心理テスト（6つの文章）

### 心理テストの目的
- 有料カウンセリングは時間が限られている
- 会話しながらガムテープを探すと時間がかかる
- **心理テストを使うと最初から一発でガムテープが見つかる**

### 相談者目線の考え方
- 精神的に苦しんでいる方は経済的にも苦しんでいることが多い
- 日本では心理カウンセリングに国民健康保険が使えない
- カウンセリングは高くなる
- **なるべく時間も短く、経済的負担も少なく**

### テープ式の自信
- 1発で気づきが起きたら一瞬
- わざと引き延ばして儲けるという考え方は1mmもない
- あくまでも相談者目線

### 心理テストのやり方

**注意点：**
- 集中するというよりリラックス
- 思いついたことをサラサラっと書く
- 深呼吸して書く

**6つの文章（穴埋め）：**

```
1. この人生で私は【　　　】しなければならない

2. この人生で私は【　　　】できない

3. もしこの人生の目的を遂げようとする時、達成しようとする時、
   起こりうる最悪の状況は【　　　】である

4. 私はいつでも【　　　】すべき

5. 私は決して【　　　】してはいけない

6. 私は【　　　】である（I am 〜）
```

**書き方：**
- 日本語としておかしくなっても構わない
- なるべく1個以上書く
- 複数出てきたら何個でもいい
- 思いついたまま書く
- いちいち思考ちゃんを使って考えない

### 心理テストの結果
**出てきたものはほぼ全部ガムテープ**

実際に出てきた例：
- 「私はピエロである」
- 「私は奴隷である」
- 「私は決して幸せになってはいけない」
- 「この人生で私は幸せになれない」

### 心理テストの威力
- 初めて会った人でも一発でガムテープが出てくる
- 会ったことない、初めてお話しする方であっても
- **破壊力のあるツール**

### なりたい私との組み合わせ
- なりたい私と心理テストの答えを並べて見比べる
- リンクしていることが出てくる
- 例：「幸せになりたい」と書いているのに「決して幸せになってはいけない」と書いている
- **もう丸見え、透明、ガラス張り**

### カウンセリングでの使い方
1. 全ての答えを俯瞰する
2. 「1番気になる答えはどれですか？」と聞く
3. 丸をつけてもらう
4. 2番目に気になる答えにも丸をつけてもらう
5. そこから取り組んでいく

### 優先順位
- 1番気になるものから取り組む
- 命に関わる言葉が出たらそこから
- 臨機応変に対応
- 最終的には全部取り組みたい

### たくさん出ても心配ない
- 40枚、60枚ガムテープが出ても問題ない
- 例えば無価値のガムテープが1個剥がれたら全部ごっそり剥がれる
- むしろ「よく書いてくださったな」
- カウンセリングは進みやすくなる

- 1時間1回ではやらない
- 継続的なサポート

## トピック92：ピールダウンとは

### 復習：ネガティブは自分で選んでいる
- 出来事が起きて感情が発生していると世界中の人が思っている
- しかし、誰が見ても「ひどい」「悲しい」「残酷」と思う出来事も、その人が**意味づけている**
- 出来事には何の意味もなく、その時その時に見た人が意味付けをしている
- **常に自分の心は選んでいる**

### 例：幼少期の虐待と大成功
- 小さい頃に毎日殴られ、相手にされず、ご飯も食べさせてもらえず、罵声を浴びせられた子
- その姿を見ると「なんてひどい、残酷だ」と心が痛む
- しかしその人が大人になって大成功した時（孫正義、松下幸之助、前澤友作のように）
- 両方のポイントを見た時に「辛かった経験が良かったんですね」となる
- **出来事は全てに何の意味もない、意味付けしているのは自分**

### 重要な気づき
**仁さんが対応しなくてもガムテープは剥がれる**

- ブログなどで発信している内容で十分
- 「仁さんが対応してくれないからガムテープが剥がれない」は思い込み
- 私の思い込みは私がコントロールできる

### ピールダウンの定義
- 英語で「Peel Down」= めくる、1枚ずつめくる
- 玉ねぎカウンセリング：1枚1枚外側の皮を剥いて、中の核となる部分に当たる
- **核となるネガティブな感情の種類にたどり着くまで聞いていく手法**

### なぜピールダウンが必要か
- ガムテープを剥がす時に必要なこと：**文章 + ネガティブな感情の種類**
- 「苦しい苦しい」だけでは解放できない
- テープ式心理学は言語化に特化した論理化されたもの
- 感情の種類ごとに解放の仕方が違う（無価値→こう、恐怖→こう、寂しさ→こう）

### ピールダウンの威力
- ネガティブな感情の種類を特定できる
- ペルソナを見破れる（嘘も現れる）
- **仁さんには嘘が通用しないレベルの技術**
- 心理カウンセラーというよりメンタリスト
- パートナー、仕事関係者、誰にでも使えるテクニック

## トピック93：ピールダウンのやり方

### 基本的な聞き方
1. 相談者が口にした言葉を**そのまま**使う
2. 同じことを聞き返す
3. 逆を聞く
4. 「それはどうなのですか？」と聞いていく

### 具体例

```
相談者：「私は何もうまくできないんです」

カウンセラー：「なるほど、では何もうまくいかなかったら、
              あなたはどう感じるんですか？」

相談者：「私は愛されないと感じます」

カウンセラー：「なるほど、じゃあ愛されないとどう感じますか？」
         または「愛されたらどう感じますか？」（逆を聞く）

相談者：「価値がないと感じます」

カウンセラー：「なるほど、あなたは無価値を感じていたんですね」
```

**結果：「私は何もうまくできない」→ 無価値観から発生していた**

### ピールダウンは一直線に繋がっている
```
私は何もうまくできない
    ↓
私は愛されないと感じる
    ↓
価値がないと感じる
    ↓
【無価値観】← 核となる感情
```

**1番下の無価値感が消えたら、上も全部なくなる**

### ピールダウンを止めるポイント
1. **同じ答えが返ってきた時**
   - 「愛されないとどう感じますか？」→「愛されないと感じます」
   - これ以上掘り下げられない

2. **核となるネガティブな感情の種類が出てきた時**
   - 5大ネガティブ（無価値観、寂しさ、恐怖、怒り、罪悪感）など
   - 特定できたら終了

### 質問を変えてみる
同じ答えが返ってきたら、逆を聞いてみる：
- 「愛されないとどう感じますか？」→「愛されないと感じます」
- 「じゃあ愛されたらどう感じますか？」→「幸せです」
- 「じゃあ愛されないならば不幸せってことですか？」

## トピック94：Why（なぜ）を使わない

### 重要なポイント
**「なぜ」「どうして」「なんで」は聞かない**

### Whyを使うと何が起きるか
- 理論が出てくる
- 言い訳が出てくる
- 本当に聞きたいネガティブの感情の種類が現れてこない

### 例
```
❌ 悪い聞き方：
「どうしてうまくできないと思うんですか？」
→「私はこうでこうで、昔からうつ病と診断されていて、
  経済的にもこうで...」（理論・言い訳が出てくる）

✅ 良い聞き方：
「うまくできなかったらどう感じますか？」
→「愛されないと感じます」（感情が出てくる）
```

### 使うべき質問
- **How（どう）を使う**
- 「どう感じますか？」
- 「どうなりますか？」
- 「どういうことですか？」

### Why vs How
| Why（なぜ） | How（どう） |
|------------|-----------|
| 理論が出る | 感情が出る |
| 言い訳が出る | 本音が出る |
| 夫婦喧嘩のような議論 | ピールダウンが進む |
| 一般社会でよく使われる | わざと使う |

## トピック95：ピールダウン実例①「私は人形である」

### 心理テストの回答
「私は【人形】である」

### ピールダウンの流れ

```
「私は人形である」とはどういうことですか？

→ 自分の意見・意思を伝えても受け入れてもらえないので、
  人形であると思いました

では、自分の意思や意見を伝えても受け入れてもらえなかった時、
どう感じますか？

→ 存在していても意味がないと感じます

なるほど、存在しても意味がなかった場合、どう感じるんですか？

→ 消えてしまいたい

消えてしまいたいとは、どういうネガティブの感情の種類ですか？

→ 無価値観です
```

### 結果
```
私は人形である
    ↓
自分の意見を伝えても受け入れてもらえない
    ↓
存在しても意味がない
    ↓
消えてしまいたい
    ↓
【無価値観】
```

### ガムテープ化
**「私は人形である」+ 無価値観**

→ 次の質問：「これは何歳から感じていますか？」

### 重要ポイント
**1番下の無価値感が消えたら、全部なくなる：**
- 消えてしまいたいもなくなる
- 存在しても意味がないもなくなる
- 自分の意見を伝えても受け入れてもらえないもなくなる
- 私は人形であるもなくなる

## トピック96：ピールダウン実例②「両親を悲しませてはいけない」

### 心理テストの回答
「私は決して【両親を悲しませては】いけない」

### ピールダウンの流れ

```
両親を悲しませるとどうなるんですか？
（※「なんでそう思うんですか」とは聞かない）

→ 罪悪感を感じます
```

### 結果
**この時点で終了！**
- 罪悪感という核となる感情が出てきた
- ピールダウン完了

### ガムテープ化
**「私は両親を悲しませてはいけない」+ 罪悪感**

### 重要ポイント：カウンセラーが守るべきこと

1. **勝手にジャッジしてはいけない**
   - 「これ無価値観ですよね」「恐怖ですよね」と先入観で決めつけない

2. **相談者が使っていない言葉を使わない**
   - 相談者：「両親を悲しませてはいけない」
   - ❌ カウンセラー：「それって親不孝ってことですもんね、親不孝をしてしまうとどうなりますか？」
   - ✅ カウンセラー：「両親を悲しませるとどうなりますか？」

3. **相手が使っている言葉を最優先**

## トピック97：ピールダウン実例③「調子に乗ってはいけない」

### 心理テストの回答
「私は決して【調子に乗っては】いけない」

### ピールダウンの流れ

```
調子に乗ってしまったらどうなりますか？

→ 目をつけられる、非難を浴びる

良くない目で見られたり非難を浴びたらどう感じるんですか？

→ 苦しい

良くない目で見られたり非難を浴びることはどう苦しいですか？

→ 存在したくない

存在したくないというのは、ネガティブの感情の種類は何ですか？

→ 無価値です
```

### 追加の発見：シナリオ分岐
- 「良くない目で見られる」= **恥ずかしい**
- 「非難を浴びる」= **苦しい** → 存在したくない → 無価値観

### ガムテープの構造
```
私は決して調子に乗ってはいけない
    ↓
調子に乗ったら良くない目で見られる → 恥ずかしい
調子に乗ったら非難を浴びる → 苦しい
    ↓
存在したくない
    ↓
【無価値観】
```

### 2次的なガムテープ
- 「調子に乗ってはいけない」は2次的なガムテープだった
- 大元のガムテープ（無価値観）を剥がしたら、これも剥がれた
- 「良くない目で見られようが、非難を浴びようが、調子に乗りまくりです！」

## トピック98：ピールダウン実例④「アドバイスをしてはいけない」

### 心理テストの回答
「私は決して【アドバイスをしては】いけない」

### ピールダウンの流れ

```
アドバイスをしたらどうなりますか？

→ 人が嫌がる、話が続かない、相手が黙ってしまう

話が続かなかったり相手が黙ってしまった時、何を感じますか？

→ 嫌われたと感じる
→ 相手をコントロールしてるのかな、マウント取ってるのかなと思う

コントロールすること、マウントを取ることはどう思うんですか？

→ 良くないと思うけど、どこかで有価値を感じる

嫌われたと感じたら、どう感じますか？

→ 無価値感と孤独感と恐怖を感じます
```

### ガムテープの構造
```
私は決してアドバイスをしてはいけない
    ↓
アドバイスすると話が続かない、相手が黙る
    ↓
嫌われたと感じる
    ↓
【無価値感】【孤独感】【恐怖】
```

### 複雑な構造：やめられない問題
- アドバイスしてはいけないと思い込んでいる
- しかしアドバイスをやめられない
- なぜ？→ 優越感・有価値感を感じたいから（依存のループ）

### 仁さんの解説
```
アドバイスしてはいけない×いけない×いけないと思い込みながら
アドバイスし続けると、ネガティブが乗ったアドバイスになる

さらに、優越感を感じるため、有価値感を得るための
依存を満たすためのアドバイスにもなっている

これがループの原因
```

### 解決策
1. このガムテープをそっくり剥がしてしまう
2. 根本的に「私は価値がある」「孤独じゃない」「怖くない」のリラックス状態になる
3. そうすれば即席有価値アドバイスをしなくなる
4. 本当に必要な適切なアドバイスだけになる

### 重要な考え方
- 「〜してはいけない」という自分の中の思い込みを外していく
- **全部外す**
- 良いか悪いかは言わない、選択

## トピック100：心理カウンセラーとは何か

- 実際に苦しんでいる人たちは本当に存在している

「世の中を変えなければならないとか、人を救わなければならないというのは必要ないです。思い込みです。全てはありのままでいい」

1. 全てはありのままでいいと受け入れる
2. その上で、心の仕組みを学んだことで、苦しんでいる方に対してカウンセリングをしてあげる
3. これも1つの必要（角度の違い）

### 本来私たちは
- 何者になってもいいし、何者にもならなくていい

## トピック101：ネガティブの本当の意味

## トピック103：鏡になるということ

- 最悪の場合、傷つけてしまう恐れもある

## トピック104：基本的なカウンセリングの流れ

### ステップ1：初めましての挨拶
- 「どうされましたか」と聞く
- 「何かお悩みがありますか」とは聞かない（軽く感じるから）
- **「今何に苦しんでおられますか」と聞く**

**なぜ「お悩み」と言わないか：**
- ヘビー級の方々からすると「生きるかどうなるか」という状態
- 「お悩みありますか」は軽い感じがする
- 命に関わる方に対して「何に悩んでいますか」は軽いように感じる

### ステップ2：何が起きているかを知る
- 説明するのが苦手な方もいる
- 「大丈夫です、そのまま話してください」と伝える
- 話を聞きながら**推測していく**

**推測する内容：**
- 恋愛相談なのかな
- 仕事かな
- 書き出しかな
- ヘビー級なのかな
- ペルソナかな
- ガムテープを剥がす必要があるのかな

### ステップ3：判断と分岐

**A. ガムテープを剥がす話ではなかった場合：**
- 一通り話を聞く
- 思い込みを見つけて「この意味づけを変えられますか」ぐらいで終わる
- 1時間ぐらいで終了することもある
- ただただ吐き出しを聞いてあげるだけで楽になる方もいる

**B. ガムテープに取り組む必要がある場合：**
→ ステップ4へ

### ステップ4：テープ式心理学の説明
- 初めての方には概念を説明しないといけない
- 「テープ式心理学とは精神病は存在しないと考えている、新しい心理学であり...」
- **ここを受け入れてもらわないと次に進まない**

**精神病だと思い込んでいる場合：**
- 「私は発達障害のせいだと思います」「〇〇クリニックでは〇〇病だと言われました」
- 精神病とは何か、思い込みとは何かという話をする必要がある
- これだけで1時間かかることもある

### ステップ5：アラジンの魔法のランプ
- なりたい私を聞く
- 「これは必ず叶います」と伝える
- 逆を話す（なれない私でもある）
- その思い込みが何なのか、心理テストで見つけていきましょう

### ステップ6：心理テスト
- 6つの文章でガムテープを見つける

### ステップ7：ガムテープの特定と取り組み
- 「1番気になるガムテープはどれですか」と聞く
- 「2番目に気になるのはどれですか」と聞く
- なりたい私と繋がっているか確認
- 「このガムテープがあるため、このなりたい私に今なれないんですね」
- 1番目のガムテープから取り組んでいく

### 時間の目安
- 初回の1時間目が終わるか、2回目の1時間ぐらいが終わっている
- 基本的にカウンセリングは60分〜長くても90分を月4回1ヶ月

## トピック105：話の聞き方の基本

### テープ式心理カウンセラーの仕事
**話を聞くことが仕事ではない**

- もちろん話も聞いて吐き出しも受け止める
- しかし本当の仕事は：**ガムテープを見つけて、ガムテープを剥がせるということを伝えること**
- **気づいてもらうのが仕事**

### 違和感を見つける
話を聞きながら違和感を見つける：
- 「私ってやっぱり価値がないから」（さらっと出てくる）
- 「僕は生きている意味がないと昔から思ってて」
- 「私なんて周りよりあと順番、後回しでいいんですよ」
- 「恋愛でも1番大切にしてもらわなくていいんです」

**相談者側はさらっと言ってくる（思い込んでいるから）**

### メモする
- 違和感として覚えておく
- 「今のは何かガムテープだな」とメモ書きしていく
- 違和感の部分を掘り下げると、心理テストのどこかのガムテープと繋がっていることに気づく

### 俯瞰する
- メモした内容や心理テストの答えをテーブルの上に置く
- 「今はこっちのこと言ってるな」
- 「さっき心理テストで出てきたこのガムテープはここと紐づいてるな」
- カウンセラー側が迷子にならないように

### 常に現在地を伝える
- 相談者にも「今はこのガムテープに取り組んでいます」と分かるようにする
- 相談者はいろんな話をするし、話が飛んでいくこともある
- メモをしながらガムテープとの関係性をつなげていく

### 落書きカウンセリング
- テープ式ではメモを「落書き」のように書いていく
- スラスラとたくさん書いていく
- 相談者は苦しいから吐き出したくてしょうがない
- スピードが早いので綺麗に清書はできない
- 落書きのようにカウンセリングしていく

### 吐き出しの受け止め方
- 相談者はまず吐き出しをしたくて次々話が出てくる
- 「ちょっと待ってください」と待ってくれるものではない
- 苦しいから吐き出したいし聞いてほしい
- 質問されたことにも回答していきたい

## トピック106：ノージャッジの実践

### 意見を求められた時
相談者から「〇〇さんはどう思いますか？」と聞かれることがある

**意見 = ジャッジ**

### 必ず付け加える言葉
「**個人的には**〇〇だと思いますけどね」

**例：**
- ❌「その方が悪い人ですね、良い人ですね」
- ✅「個人的には、いやもうありえないなって思いますよ」

### 「普通は〜」を使わない
- 「普通はこうやん」「普通こうじゃないですか」と言ってしまうと
- 相談者は「正しい/正しくない」と受け取ってしまう
- アドバイスだらけになってカウンセリングにならない

## トピック107：アクティブリスニング

### 定義
**引き出す技術**

カウンセラーに限らず、どんな職業でも使える技術。

### やり方
1. **全てを肯定する**
2. **ジャッジしない**
3. **「その時はどうでしたか？」とフォーカスして掘り下げる**
4. **相手が言った言葉を繰り返してあげる**

```
             そんなこと考えられる環境じゃなかったですもんね」（肯定）
             
             「その時って、実際に学校行った時は、

             （フォーカスして掘り下げる）
```

### ポイント

- 本人の中でも気づいていなかった言葉がどんどん出てくる

### 効果

- 本人の中でも気づいていなかった言葉がどんどん出てくる
- 感情が出てくる
- 相手からすると「めっちゃ話を聞いてもらえた」という感覚になる

### カウンセラー側の利点
- 相談者からたくさん話が引き出せる
- ガムテープの奥深くを見つけていくことができる

## トピック108：自分の話は不要

### 基本原則
**カウンセラー側は自分の話を基本的に一切する必要がない**

- 他の心理カウンセラーで自分の話ばっかりして1時間終わっちゃう人もいるらしい
- テープ式の心理カウンセラーは自分の話をする必要がない
- カウンセラーは鏡なので、ただただ相手の話を聞いて鏡になって支える

### 例外
- コミュニケーション的に必要な時
- カウンセラーとしてヒントになる場合
- → その時も**エネルギーを入れすぎない**

### エネルギーを入れすぎるとは
- 自分の話をする際に有価値を感じたくて強いエネルギーを出してしまう
- 「圧を強める言い方」
- → 相談者側が縮こまってしまう、「強く言われた」と感じてしまう

- 強いエネルギーを出した瞬間、相談者にバレる
- グッと来られた感じになる
- **境界線外になってしまう**

### 正しい姿勢
カウンセリングを受ける状態の方は繊細なので、常にカウンセラー側は**リラックス**をしてお話をするのが重要

## トピック136：境界線を持つ

### 基本中の基本
カウンセラーとして、特にお仕事の時はしっかり境界線を持つこと

### 1度でもダメ
1度でも近寄ったかのようなカウンセリングをしてしまった場合、2度目は相談者は**冷たいと感じる**

→ 相談者を本当に惑わしてしまう悪意のあるカウンセリングになってしまう
→ 1回でもダメ、1回やったならカウンセリング中止

### 近寄るとは
立場を超えて：
- 「私がいるからね」
- 「大丈夫だよ」
- 「深夜1時でも2時でも連絡があったら取るよ」
- 「〇〇さんの電話だったらいつでも取るよ」

### 恋愛感情を持たれた場合
トリシアの教え：すぐにお断りを伝える（カウンセリングを断る）

相談者に伝える言葉：
**「これ以上カウンセリングは続けられませんが、いかがしますか」**

## トピック137：相談者が依存する場合の対応

### 相談者が言ってくること
「カウンセラーや人に依存してしまいがちです」
「依存してしまいそうで、すいません」

### 対応
**「大丈夫ですよ」「依存してて大丈夫ですよ」**と伝える

### 依存が成立する条件
両方がネガティブを持っていて、両方ともがネガティブを埋め合うから初めて依存が成立する
（親子でも恋愛でも友達でも兄弟でも全て同じ）

### 依存にならない条件
どちらか片方がネガティブで、どちらか片方が**ノージャッジ**だった場合、依存にならない

### テープ式心理カウンセラーの場合
- 相談者が怒りを出してくる
- 涙を流してくる
- 命に関わるアクションを見せてくる

何をしようとも、カウンセラー側の心がセンターで「今日も元気だね、大丈夫かね」とニコニコしている状態だった場合、**依存にならない**

- 17秒でネガティブをコントロールしてしまう
- 相談者のネガティブが映らない
- だから相談者側がどんどん依存しようとしてきても「いいですよ、全然大丈夫ですよ、依存して大丈夫ですよ。こちらは依存になりませんので」

## トピック138：X界隈のカウンセラーの問題点

### 問題のあるパターン
カウンセラーと名乗っているけど、すごい投稿している人：
「私がいるからね」「私がいるからね」

### その人たちの本音
- 私が褒めて欲しい
- 自分が寂しくてその人のそばにいたい
- 「いるよいるよ」「ありがとうありがとう」とその繋がりを感じておきたい

### 本当のテープ式心理カウンセラー
- X、Twitter、YouTubeは営業しないといけないので発信している
- しかし基本的には**沈黙**
- 湖のコメのように「シーン」
- 「何かをしなければ、助けなければ」は全くない
