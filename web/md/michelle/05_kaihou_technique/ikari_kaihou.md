# 怒りの解放

## 定義

怒りは五大ネガティブの一つ。怒りの解放には「**許す**」ことが必要。ただし、怒りは他の感情を守る「ガードの怒り」であることが多い。

## 怒りの特徴

### 身体的感覚

- 体が熱くなる
- 拳を握りしめる
- 歯を食いしばる
- 声が大きくなる
- 攻撃的になる

### 怒りの涙

- 「なにくそ」という涙
- 悔しさを伴う涙
- 拳を握りしめながらの涙
- 悲しみの涙とは違う

### 怒りが現れる場面

- 不当な扱いを受けた時
- 裏切られた時
- 自分や大切な人が傷つけられた時
- 思い通りにならない時
- 境界線を侵された時

## 中核ポイント：ガードの怒り

### ガードの怒りとは

**怒りは他の感情を守っていることが多い**

```
表面：怒り（ガード）
    ↓
隠れている感情：悲しみ、恐怖、寂しさ、無価値観
```

### なぜガードするのか

- 悲しみを感じたくない
- 恐怖を認めたくない
- 弱さを見せたくない
- 傷つきたくない

### 怒りの下を見る

```
カウンセラー：「その怒りの下には何がありますか？」
クライアント：「...本当は悲しいんです」
```

→ 怒りの下にある本当の感情を見つける

## 怒りの解放方法：許し

### 許しとは

- 相手を許す
- 自分を許す
- 状況を許す
- 「許す」は「忘れる」ではない

### 許しのプロセス

```
怒りを感じる
    ↓
怒りの下の感情を見つける（悲しみ、恐怖など）
    ↓
その感情を解放する
    ↓
相手の立場を理解する（相手もガムテープがあった）
    ↓
許す決断をする
    ↓
怒りが解放される
```

### 許すことの意味

- 相手のためではなく自分のため
- 怒りを持ち続けることは自分を傷つける
- 許すことで自分が自由になる

## 実践/カウンセリングでの使い方

### 1. 怒りを受け止める

```
カウンセラー：「怒っているんですね」
クライアント：「当たり前です！あんなことされて！」
カウンセラー：「それは怒って当然ですね」
```

### 2. 怒りの下を探る

```
カウンセラー：「その怒りの奥には、何があると思いますか？」
クライアント：「...本当は悲しいのかもしれません」
```

### 3. 下の感情を解放する

- 悲しみなら涙を流す
- 恐怖なら勇気を持って行動
- 寂しさなら全体性と行動
- 無価値観ならありのままを受け入れる

### 4. 許しへ導く

```
カウンセラー：「相手も何かガムテープがあったのかもしれませんね」
クライアント：「...そうかもしれません」
カウンセラー：「許すことで、あなた自身が自由になれます」
```

## 許しと向き合うためのポイント

### 1. 許しを強制しない

- 準備ができていない人もいる
- 「許すべき」とは言わない
- 本人の決断を待つ

### 2. 相手を理解する視点

- 相手にもガムテープがある
- 相手も傷ついている
- 相手も幸せになりたかった
- 理解することで許しやすくなる

### 3. 自分を許す

- 自分への怒りもある
- 「なぜあの時...」という後悔
- 自分を責め続けない
- 自分も許す

### 4. 許しは一瞬ではない

- 時間がかかることもある
- 何度も怒りが戻ってくることもある
- その都度向き合う

## 怒りを吐き出す方法

### 安全に怒りを表現する

- 枕を叩く
- 紙に書いて破る
- 大声を出す（場所を選んで）
- 運動する

### 怒りを溜め込まない

- 溜め込むと爆発する
- 適切に表現する
- 感じたらその場で処理

## よくある誤解・注意点

- ❌ 怒りは悪い感情
- ⭕ 怒りは境界線を守る大切な感情。問題は怒りに支配されること

- ❌ 許す＝忘れる
- ⭕ 許すことと忘れることは違う。覚えていても許せる

- ❌ 許したら相手の勝ち
- ⭕ 許すことで自分が自由になる。自分のための許し

- ❌ 怒りをそのまま解放すればいい
- ⭕ 怒りの下にある感情を見つけることが重要

## まとめ

怒りは他の感情（悲しみ、恐怖など）を守る「ガードの怒り」であることが多い。怒りの解放には、まず怒りの下にある本当の感情を見つけ、その感情を解放し、最終的に「許す」ことが必要。許しは相手のためではなく、自分が自由になるためのもの。許しを強制せず、本人の決断を待つ姿勢が大切。
