# インナーチャイルドの統合

## 定義

インナーチャイルドを癒すとは、統合すること。バラバラになっている私たちの心を1つにしていくプロセス。

## 1は全てであり、全ては1である

### アーキタイプの教え

私たちは元々1つであった：

- Y（ソース、大いなる源）は全て1つである
- スピリチュアル業界では「ワンネス」という言葉を使う
- 私たちは元々は1であった
- それが数えきれない形で子供たちに分かれていった
- 動物、昆虫、魚、木、何もかもに分かれていった
- 最終的にはまた1つに統合されていく

### ホオポノポノの考え方

ハワイ発祥のホオポノポノ（「ありがとう、ごめんなさい、愛しています、許します」）の教え：

1. 私たちの人生は生まれ変わるたびに1つ1つのドラマを味わう
2. そのドラマが解決（解放・気づき）したら次のドラマが起こる
3. 全てのドラマが終わったら1つになる

**テープ式心理学も全く同じ考え方**

## インナーチャイルドと統合

### トリシアの本当の願い

トリシアは本当は「インナーチャイルドセラピー」ではなく**「統合療法」**という名前をつけたかった。しかし病気を治す療法だと取り扱われそうなのでインナーチャイルドセラピーにした。

**本当の意味：統合するもの**

### バラバラになった私を1つに

- 本当の私は1人
- 強いストレスがかかったことによって分裂してしまっている
- 5歳のチャイルド、10歳のチャイルド、12歳のチャイルド...たくさんいる
- これらを全て1に戻していく
- バラバラになっていたものが収まっていく

### 多重人格の統合事例

仁さんがカウンセリングした多重人格の方：

- 15人の人格に分裂していた
- 「必ず1になる」「必ず全て統合される」ことを伝え続けた
- 1つ1つの人格が解放されていった
- 15→14→10→7→5→3→2→1とだんだん統合されていった

**これが世界の仕組み。必ず統合される。**

## 賢い大人との対話

### インナーチャイルド解放の核心

インナーチャイルドを解放するプロセスで最も重要なこと：**賢い大人の部分と対峙する**

### 賢い大人とは

- 私の中にいる大人の私
- 賢者の私
- マインド様の状態の私
- 必ず誰の中にもいる

### なぜ賢い大人が必要か

**間違った対話：**
```
5歳のチャイルド：「もっとこうしたかった！寂しいよ！」
47歳の私（思考ちゃん状態）：「だってそんなこと言ったって無理でしょう！」
→ 解放されない
```

**正しい対話：**
```
5歳のチャイルド：「もっとこうしたかった！寂しいよ！」
47歳の私（賢い大人）：「そうだったんだね。どうしたの？」
→ 対話ができる → 解放に向かう
```

### 同化してはいけない

- 47歳側もネガティブで思考ちゃん状態だと、チャイルドと「同化」してしまう
- 同化＝チャイルドそのものになっている状態
- 対話も統合もできない

### 対話のプロセス

1. **マインド様の状態になる**（瞑想など）
2. **チャイルドを呼び出す**
3. **賢い大人の私を出す**
4. **対話を始める**
   - 「こんにちは」「どうしたの？」
   - 喋らなそうだったら喋るまで待つ
   - 変顔したり、トランプやったり、子供に合わせる
   - 5歳なら5歳らしく接する
5. **チャイルドの願いを叶える**
   - 謝るところは謝る
   - 愛するところは愛する
   - そばにいる時はそばにいる
   - 遊ぶ時は遊ぶ
   - やりたいようにさせてやる
6. **対話を重ねる**
   - チャイルドは必ず答えを教えてくれる
   - どうしたらいいのかを

**ガムテープ：言語化・意味付けを変える**
**インナーチャイルド：対話・会話**

## チャイルドの確認事項

チャイルドが出てきたら必ず確認すること：

- 年齢（何歳なの？）
- 髪型
- 服装
- 顔つき
- 足元・靴
- いる部屋・場所

**細かく瞑想の中で確認する**

## 解放後のチャイルド

### 消えていくチャイルド

- 解放が起きたら「ありがとう」のような表情をして
- すーっと消えていくこともある

### そばにずっといるチャイルド

- 解放後もずっといるチャイルドもいる
- これはネガティブがあるということではない
- エネルギーとしては残っている
- チャイルドがいる状態でバランスを保つこともある
- ガムテープ貼ったままでもバランス保てる状態と同じ

## 注意事項

### 触ってはいけないチャイルド

- 準備ができていないチャイルドは出てこない
- 無理やり呼び出そうとするのは危険
- 瞑想状態では思考ちゃんが機能していない（ガード空き）
- 触ってはいけないチャイルドを無理やり呼ぶと、瞑想から冷めた時にものすごく苦しくなることがある

### セラピストの責任

- 思考ちゃんはネガティブだけでなく防御もする大事な役割
- 瞑想状態ではその防御ができない
- セラピストの言葉によって誘導されてしまうことがある
- 催眠術は本当に存在する
- だからこそセラピストは慎重に

## 覚えておくべきこと

**常に私たちは統合されていく**

- バラバラになっているスパゲティのようにぐちゃぐちゃになっているネガティブ
- それが1つになっていく
- 3歳、5歳、10歳、12歳...たくさんバラバラにチャイルドがいたとしても
- **最後は1になる**

**全ては1である**

## よくある誤解・注意点

- ❌ チャイルドと同じレベルで対話する（思考ちゃん状態）
- ⭕ 賢い大人・マインド様の状態で対話する

- ❌ チャイルドを無理やり出そうとする
- ⭕ 準備ができているチャイルドしか出てこないと信じる

- ❌ チャイルドがいる＝問題がある
- ⭕ チャイルドがいる状態でバランスを保つこともある

## まとめ

インナーチャイルドを癒すとは統合すること。1は全てであり全ては1である。バラバラになった私たちの心を1つにしていく。そのためには賢い大人の私を出してチャイルドと対話する。対話を重ねることでチャイルドは答えを教えてくれる。最後は必ず1になる。
