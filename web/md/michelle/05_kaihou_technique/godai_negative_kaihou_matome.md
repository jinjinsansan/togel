# 五大ネガティブ解放方法まとめ

## 定義

五大ネガティブ（恐怖、悲しみ、怒り、寂しさ、無価値観）それぞれに対応する解放方法の一覧。

## 中核ポイント：解放方法一覧

| ネガティブ | 解放方法 | アーキタイプ |
|-----------|---------|-------------|
| **恐怖** | 勇気を出す | 勇気 |
| **悲しみ** | 時間が解決する | 永遠 |
| **怒り** | 許し（ガードの下のガムテープを掘る） | 許し |
| **寂しさ** | みんな繋がっている＋手足を使う | 全体性 |
| **無価値観** | ありのままの私で十分価値がある | 有価値 |

## 各感情の詳細解説

### 恐怖 → 勇気

**アーキタイプ：勇気**

- 勇気はアーキタイプから出る
- 全員が持っている（持っていないは思い込み）
- 「勇気なんてない、怖い怖い」はYOU様が作ってくれた**勇気を感じるための真逆のエッセンス**
- 私たちは**勇気そのもの**

**解放のアプローチ：**
- 開き直る
- 「受けるぞ」という決断をする
- 勇気を奮わせる
- **覚悟をする**

### 悲しみ → 時間

**アーキタイプ：永遠**

- 悲しみは喪失感（物を失う、ペットが亡くなる、おばあちゃんが亡くなる）
- 時間が解決してくれる
- 亡くなったとしても**永遠に心の中で生き続ける**という体験ができる
- 必ず解決する

**解放のアプローチ：**
- 吐き出しを受け止め続ける
- 何度でも共にい続けてあげる
- 時間しかない
- 同じ話を何度でも聞き続ける

### 怒り → 許し

**アーキタイプ：許し**

- ガードの怒りがあるので、その下のガムテープを掘っていく必要がある
- 許しそのものがアーキタイプ
- **許せる**（その要素を原型として持っている）

**解放のアプローチ：**
- まずガードの怒りの下にある本当の感情を特定
- 本当の感情に対する解放を行う
- 純粋な怒りの場合は「許し」のワーク

### 寂しさ → 全体性

**アーキタイプ：全体性（みんな繋がっている）**

- 人生がコントロールできないレベルの寂しさは**過去に作られた寂しさが反応しているだけ**
- 現在は「今この瞬間どうなのか」
- **自らの手と足を使って**友達を作ったり恋人を作ったりして寂しくならないようにしていく

**解放のアプローチ：**
- 全体性の認識（みんな繋がっている）
- 実際に行動する（手足を使う）
- 過去の寂しさと今を分ける

### 無価値観 → 有価値

**アーキタイプ：有価値**

- ありのままのあなたのまんまでいい
- 本来の私のままで十分価値があることに気づく
- アーキタイプの有価値に気づく

**解放のアプローチ：**
- 条件なしで価値があることを認識
- 「〜だから価値がある」ではなく「ありのままで価値がある」
- 自己受容

## 感情の見分け方

### 身体的感覚の違い

| 感情 | 身体的感覚 |
|------|-----------|
| 恐怖 | 緊張、心臓がドキドキ、体が固まる |
| 悲しみ | 胸が苦しい、涙が出る |
| 怒り | 熱くなる、拳を握りしめる |
| 寂しさ | 胸にぽっかり穴が空いた感じ |
| 無価値観 | 自分が小さくなる感じ、消えたい |

### ピールダウンで特定

表面の感情から核となる感情に到達するまでピールダウンを行い、五大ネガティブのどれに該当するかを特定する。

## カウンセリングでの流れ

```
① なりたい私を聞く
    ↓
② 心理テストでガムテープを発見
    ↓
③ ピールダウンで核となる感情を特定
    ↓
④ 五大ネガティブに分類
    ↓
⑤ 対応する解放テクニックを適用
    ↓
⑥ 確認・統合
```

## よくある誤解・注意点

- ❌ 全ての感情に同じ解放方法を使う
- ⭕ 感情の種類によって解放方法が異なる

- ❌ 怒りはそのまま怒りの解放をする
- ⭕ 怒りは「ガードの怒り」の可能性があるため、下に何があるか確認する

- ❌ 一回で完全に解放される
- ⭕ 複数回のセッションが必要な場合もある

## まとめ

五大ネガティブにはそれぞれ対応する解放方法とアーキタイプがある。恐怖→勇気、悲しみ→時間（永遠）、怒り→許し、寂しさ→全体性、無価値観→有価値。ピールダウンで核となる感情を特定し、適切な解放テクニックを適用することで、ガムテープを剥がすことができる。
