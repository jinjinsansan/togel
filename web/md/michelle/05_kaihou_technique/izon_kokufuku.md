# 依存の克服法

## 依存症の種類

依存症には複数の種類がある：

| 種類 | 依存対象 |
|-----|---------|
| アルコール依存症 | お酒 |
| 薬物依存症 | 薬物 |
| オーバードーズ依存症 | 過剰摂取 |
| ギャンブル依存症 | ギャンブル |
| 過食依存症 | 食べ物 |
| 拒食依存症 | 食べないこと |
| ネット依存症 | インターネット |
| リストカット依存症 | 自傷行為 |
| 反社会的依存症 | 反社会的行動 |
| 仕事依存症 | 仕事 |
| 恋愛依存症 | 恋愛 |

## 定義

### テープ式心理学での見解

- 依存症は**病気などではなく**、心のバランスを保つ行為
- 「依存症」の「症」を削除し、**依存**とだけ呼称する
- 「依存」という言葉がネガティブなイメージを持たれているため、**「もたれ中」**と呼ぶ

### 依存は適切なアクション

依存は心のバランスを保つ**適切なアクション**。心がバランスを崩している時に、これ以上ネガティブな感情を感じないようにするために依存行為（アクション）を行う。

## 依存発生のメカニズム

### 依存の方程式

```
【1】ネガティブな感情が発生する
    ↓
【2】ネガティブな感情を感じたくない
    ↓
【3】依存対象に依存アクション
```

### 各依存の回避対象

| 依存の種類 | 回避しようとしている感情・状態 |
|-----------|---------------------------|
| アルコール依存 | 何らかの耐えがたい傷を感じたくない |
| ギャンブル依存 | 勝った時の優越感を求める（無価値観からの回避） |
| 恋愛依存 | 愛されていると感じることで寂しさを回避 |
| 過食依存 | 孤独感・寂しさ等を食べることで回避 |

全ての依存アクションは何らかの形で**自分の心の中で耐えがたい感情を感じないようにバランスを保っている**。

## 克服の原理

### なぜ精神医学では克服できないか

- 精神医学では依存症は病気とされ、各種の依存に対して治療法を提案する
- しかし、依存が発生する**メカニズムを理解していなければ**、どのような治療法であっても無意味

### 克服への道

依存を克服したい方は、この原理が理解できなければ**克服には至らない**。

本来の私たちの心は常にバランスを保つように自動的にプログラミングされている。そのバランスを保てないと感じた時に、人は1秒でも長くその苦しみを感じたくないことから**回避行動**を起こしている。それが依存。

## 克服の具体的方法

### 基本ステップ

依存の方程式に則り、**【1】のネガティブな感情が発生した時**に、自分自身で心の内側と向き合ってネガティブな感情を解放すること。

### 訓練1：直前に何を感じていたか思い出す

依存アクションの**直前**に、どのようなネガティブな感情から回避しようとしていたのかを思い出す訓練を重ねる。

**過食依存の例**:
```
空腹ではないのにソワソワしながら何かを食べてしまう時
    ↓
「何から回避しようとしていたか」を思い出す
    ↓
例：別れた元カノのことを思い出して強い孤独感を感じた
    ↓
過食依存の直前には「寂しさ」を感じており、
その回避行動として過食を行っていることが明確に分かる
```

### 訓練2：直後に何が消えて楽になったか思い出す

直前に何を感じていたか分からない場合は、**逆に**依存行動をした後にどのようなネガティブな感情が消えて楽になったのかを思い出す訓練を重ねる。

**過食依存の例**:
```
過食行動を起こした
    ↓
どの感情が消えて楽になったか思い出す
    ↓
例：「寂しさを感じなくなった」
    ↓
過食の前に「寂しさ」を感じていたことが分かる
```

### 訓練3：本当の思い込みを見つけて解放する

依存行動の直前に感じたネガティブな感情に向き合っていく中で、現れた**本当の思い込みの部分**（ガムテープ）を自分自身で解放すれば、依存は克服できる。

```
依存行動の直前に感じたネガティブに気づく
    ↓
そのネガティブと向き合う
    ↓
本当の思い込み（ガムテープ）が見つかる
    ↓
思い込みを解放する
    ↓
【1】がなくなる
    ↓
【2】も発生しない
    ↓
【3】依存アクションも起こらない
    ↓
依存克服
```

## よくある誤解・注意点

### 依存症は病気？

- ❌ 依存症は病気であり治療が必要
- ⭕ 依存は心のバランスを保つ自然的なアクション。病気ではない

### 依存行動そのものをやめようとする

- ❌ 依存行動（飲酒、過食等）をやめることが克服
- ⭕ 依存行動の直前に感じているネガティブを解放することが本当の克服

### メカニズムを理解しないままの治療

- ❌ 依存が発生するメカニズムを理解せずに治療法を試す
- ⭕ メカニズムを理解した上で、根本のネガティブに向き合う

## ポジティブな時は依存しない

心の中のバランスを保つアクションは**ネガティブな感情を感じた時に発生**する。

ポジティブな感情が発生した時にはバランスが取れており、依存アクションを起こす必要がない。そのため依存行動は起こさない。

あくまでも依存アクションを起こす時は：
1. 自分自身の心の中にネガティブな感情を感じている
2. それが耐えがたい状態
3. バランスを保つようにして依存行為を開始する

## まとめ

依存を克服するには、依存が発生するメカニズム（ネガティブ発生→感じたくない→依存アクション）を理解することが必須。克服の方法は、依存行動の直前に感じていたネガティブに気づき、向き合い、その根本にある思い込み（ガムテープ）を解放すること。直前に分からなければ、直後に何が消えて楽になったかを思い出す訓練を重ねる。根本のネガティブが解放されれば、依存は自然と克服できる。
