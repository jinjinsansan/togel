# 罪悪感の解放

## 定義

### 罪悪感とは

ネガティブ感情の種類の一つで、五大ネガティブの1個下だが、**五大ネガティブとも同等くらい非常に苦しいネガティブ感情**。

具体的な感覚：
- 申し訳ない
- すいません
- ごめんなさい
- 許されない

### 罪悪感の特徴

罪悪感には必ず**対象者（相手）が存在する**。その相手に対して「何か申し訳ない」「許されないことをしてしまった」と感じている状態。

## 背景・目的

### 罪悪感を背負って生きる選択

罪悪感を感じることで**バランスを保って生きていく方法**もある。

```
「私は許されないことをした」
    ↓
「背中に十字架を背負って生きていく」
    ↓
「一時も忘れることなく今もずっと生きている」
```

**例：不慮の交通事故で相手を亡くしてしまった場合**

予想だにできない不慮の事故であっても、未来ある若い方との接触事故で相手を亡くしてしまった時、強い罪悪感を感じるガムテープを貼る。その申し訳なさから「忘れたくない」と思い、それが一つのバランス・バネになる。何十年と罪悪感を背負ったまま生きていく方もいる。

### ネガティブは悪いことではない

罪悪感に限らず、ネガティブは何も悪いことでも何でもない。それを感じていることは、それでいい。

## 中核ポイント

### 解放の鍵は「許し」

罪悪感を解放するには**許し**しかない。

```
罪悪感 ＝ 「許されない」と思って貼り付けている
    ↓
解放 ＝ それを「許す」
```

罪悪感のガムテープは**許すことでしか剥がれない**。

### 十分謝ったという前提

「一生消えないのでしょうか」という質問が来るくらいの強い罪悪感を持っている場合、おそらくすでに十分謝っている。何十年、何年と謝り続けている。昨日今日ではない。

謝っても相手が許してくれない、または相手と連絡が取れないというパターンがある。

## 具体的なセッション方法

### 自分で自分を許すのは難しい

罪悪感で「許されない」と思っているため、自分自身を許すのは非常に難しい。

### 瞑想による許しのセッション

**手順：**

1. **目を閉じて瞑想状態に入る**
   - 静かな部屋で行う

2. **信頼できる存在をイメージで呼び出す**
   - 大好きだった亡くなったおばあちゃん
   - おじいちゃん
   - イエス・キリスト様
   - 「この方だったら100%信じてもいい」という存在

3. **その存在から許しの言葉をもらう**
   - 「いいよ」
   - 「大丈夫だよ」
   - 「あなたを許すからね」
   - 「十分謝ったよ」
   - 「もういいよ、許すよ」

4. **許された感覚を味わう**
   - わーっと涙が出る
   - ふっと心が軽くなる

### このセッションの本質

部屋に呼び出すのは**イメージ**。結局は**自分1人で許せた**ということ。

一瞬でも「わっと許された感覚」になれたら、しめたもの。それを物理的に覚えて、何度も何度もセッションする。結果的に許すことができるようになる。

## よくある誤解・注意点

### 対象者との関係

自分の中で罪悪感のガムテープを剥がしても、対象者との関係によっては引き続き「申し訳ありませんでした」「ごめんなさい」という言葉は必要かもしれない。

**区別が必要：**
- **自分の心の中のガムテープ** → 十分苦しんだから剥がしてよい
- **対象者への対応** → 関係性によって継続が必要な場合もある

### 剥がすタイミング

「本当にもう苦しくて苦しくて限界だ」と自分で思い、「これを剥がそう」と思うなら、剥がしていい。

決断さえすれば、必ず許せる。

## まとめ

罪悪感は五大ネガティブと同等くらい苦しい感情で、必ず対象者が存在する。十字架として背負って生きる選択もあるが、限界で解放したい場合は「許し」しかない。自分で自分を許すのは難しいため、瞑想で信頼できる存在をイメージして許しの言葉をもらうセッションが有効。一瞬でも許された感覚を味わえたら、それを何度も繰り返すことで許せるようになる。
