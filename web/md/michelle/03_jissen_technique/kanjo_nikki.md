# 感情日記

## 定義

感情日記とは、日々感じたネガティブ感情の**種類を特定する訓練**のこと。毎日の出来事と、その時感じた感情の種類をメモすることで、心を見る力を養う。

## 背景・目的

### なぜ感情日記が必要か

テープ式心理学では、ネガティブの感情の**種類を特定する**ことが非常に重要。なぜなら、感情の種類によって解放の方法が異なるから。

| 感情の種類 | 解放方法 |
|-----------|---------|
| 恐怖 | 勇気を出す |
| 悲しみ | 時間、吐き出しを受け止める |
| 怒り | 許し（ガードの場合は下の感情を解放） |
| 寂しさ | 繋がりに気づく |
| 無価値観 | ありのままで価値があると気づく |

### 感情の種類が分からない現状

多くの人は、自分が感じている感情の種類を正確に把握できていない。

- 「なんか苦しい」とは分かる
- 「気分が悪い」とは分かる
- でも、それが**何の感情**かは分からない

### 日本人に多い傾向

日本人は何でも「悲しい」と表現する傾向がある（国語の教科書の影響）。

**よくある間違い：**
- 「彼氏からLINEが来なくて悲しい」→ 実は**寂しい**
- 「失敗して悲しい」→ 実は**無価値観**
- 「理不尽で悲しい」→ 実は**怒り**

## 中核ポイント

### 感情日記のやり方

**基本フォーマット：**
```
【日付】○月○日
【出来事】（短く）
【感情の種類】（五大ネガティブのどれか）
```

**例：**
```
【日付】3月11日
【出来事】上司に嫌みを言われた
【感情の種類】無価値観
```

### ポイント

- **短くていい**（2行でも可）
- **毎日続ける**ことが大事
- スマホのメモ帳でOK
- 音声入力でも可

### 心を見る訓練

感情日記は「**観**」の訓練。

- 「観」＝見る（観音様の「観」）
- 心の中を見る力を養う
- ガムテープを剥がすには、心の中を見れるようになる必要がある

## なぜ感情の種類を特定する必要があるのか

### 例：恐怖を感じている人に対して

**間違った対応：**
「ありのままでいいんだよ」（無価値観の解放方法）

**正しい対応：**
「大丈夫、大丈夫」「怖がらなくていいんだよ」（勇気を与える）

### 例：寂しさを感じている人に対して

**間違った対応：**
「勇気を出そうよ」（恐怖の解放方法）

**正しい対応：**
「そばにいるよ」「みんな繋がってるんだよ」（繋がりを感じさせる）

### ポイント

感情の種類が特定できていないと、全く効果のない解放方法を使ってしまう。だから感情の種類を見分ける訓練が必須。

## 感情日記の効果

### 1. 自分の心が見えるようになる

- 最初は何を感じているか分からない
- 日記を続けると「あ、これは無価値観だな」と分かるようになる
- やがて日記を書かなくても瞬時に分かるようになる

### 2. 他人の感情も見えるようになる

- 心の仕組みは全員同じ
- 自分の心が見えるようになると、他人の心も見えてくる
- 人間関係、恋愛、仕事で活かせる

### 3. ガムテープが見つけやすくなる

- 心を見る訓練ができていると、ピールダウンがスムーズ
- ガムテープの発見が早くなる
- 解放も早くなる

## 心の動きの特徴

### 光より速い

心の中はあっという間に揺れ動く。

**例：**
- 朝、上司に嫌みを言われて無価値観を感じた
- 昼、友達と楽しい会話をしてポジティブに
- 夕方には朝の出来事を忘れている

### だから記録が大事

- 忘れてしまう前にメモする
- 後から見直すことで、自分のパターンが見える
- 「この状況で無価値観を感じやすいな」と気づく

## 実践のコツ

### 続けるコツ

- 完璧を求めない
- 短くていい
- 毎日やる（習慣化）
- スマホのメモアプリ活用
- 音声入力を使う

### 音声入力の例

```
「3月12日、今日は満員電車で嫌だった、ネガティブの種類、怒り」
```

これだけでOK。後から文字で見返せる。

### よくある失敗

- 忙しくてやらない
- 面倒でやらない
- 完璧に書こうとして挫折

**対策：** とにかく短く、毎日続けることだけを意識する。

## 見分けが難しい感情

### 寂しさと悲しみ

| 寂しさ | 悲しみ |
|-------|-------|
| 誰かと繋がりたい | 失った喪失感 |
| 孤独を感じる | 穴がぽっかり開く |
| 「そばにいて」 | 「もういない」 |

**例：**
- 彼氏からLINEが来ない → 寂しさ
- 大切な人が亡くなった → 悲しみ

### 寂しさと無価値観

| 寂しさ | 無価値観 |
|-------|---------|
| 繋がりを求める | 自分の存在価値を疑う |
| 「一人ぼっち」 | 「私はダメだ」 |

### 分からない時

LINEグループなどで質問する。「これは何の感情でしょうか？」と聞けば、先輩や仲間が教えてくれる。

## 感情日記と心理カウンセラー

### カウンセラーを目指さない人にも有効

感情日記で養った「心を見る力」は、あらゆる場面で活きる。

- 恋愛：相手の本当の感情が分かる
- 仕事：人間関係がスムーズになる
- 自分：ネガティブをコントロールできる

### テープ式心理学の知識＋心を見る力

この2つが揃うと、人間関係のコミュニケーションが最強になる。

## よくある誤解・注意点

- ❌ 感情日記は心理カウンセラー志望者だけがやるもの
- ⭕ 自分の心を見る訓練として、全員に有効

- ❌ 長文で書かないと意味がない
- ⭕ 2行でも十分。続けることが大事

- ❌ すぐに感情の種類が分かるようになる
- ⭕ 訓練なので時間がかかる。焦らず続ける

## まとめ

感情日記は、日々感じたネガティブ感情の種類を特定する訓練。毎日短くメモすることで、心を見る（観）力が養われる。感情の種類によって解放方法が異なるため、種類を見分けられることがガムテープを剥がす上で必須。続けることで自分の心だけでなく、他人の心も見えるようになる。
