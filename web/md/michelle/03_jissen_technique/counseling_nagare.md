# カウンセリングの流れ

## 定義

テープ式心理学に基づくカウンセリングの標準的な流れ。クライアントのガムテープを発見し、気づきを促し、解放へ導くためのプロセス。

## 全体の流れ

```
1. アイスブレイク・ラポール形成
    ↓
2. なりたい私を聞く
    ↓
3. 心理テスト（6つの文章）
    ↓
4. 優先順位をつける
    ↓
5. ピールダウン
    ↓
6. 五大ネガティブの特定
    ↓
7. 解放テクニック
    ↓
8. クロージング
```

## 各ステップの詳細

### 1. アイスブレイク・ラポール形成

**目的：** 安心できる場を作る

- 挨拶、自己紹介
- 緊張をほぐす会話
- 「今日はどんな気持ちで来ましたか？」
- カウンセリングの流れを説明

**ポイント：**
- 焦らない
- クライアントが話しやすい雰囲気を作る
- アクティブリスニングを心がける

### 2. なりたい私を聞く（アラジンの魔法のランプテスト）

**目的：** ゴールを明確にする。理想の自分から逆算して今の問題を特定する。

#### アラジンの魔法のランプテストの完全な手順

**Step 1: 質問を提示する**

```
ここにアラジンの魔法のランプがあります。
あなたはこの人生を良く頑張って生きてくれたので
願い事を叶えてくれると魔法使いのジーニーが言っています。

ただし、願い事はお金や旅行や物品ではなく
「なりたいわたし」のみです。

なりたいわたしならばどのようなわたしでも構いません。
結婚している、子供がいるなどの今の暮らしの全ての制限を取り払ってください。

そして、なりたいわたしの数はいくつでもOKです。
ではお願いいたします。
```

**Step 2: ユーザーの回答を受け取る**

例：
- 誰にでも優しく出来る人になりたい
- 怒りをコントロールできるわたしになりたい
- 仕事を最後までやり遂げれるわたしになりたい

**Step 3: 感謝と優先順位の確認**

```
ありがとうございます。○つも書いてくださりありがとうございます。
この中で順位をつけるとしたらどれが一番ですか？
```

**Step 4: 逆算で今の問題を特定**

例：「怒りをコントロールできるわたしになりたい」が1番の場合

```
なるほど。怒りをコントロールできるわたしになりたいのですね？
それでは今から取り組みます。

では質問します。
「怒りをコントロールできるわたしになりたい」ということは、
今現在は「怒りをコントロールできないわたし」ということですね？

このことについて具体的に教えてください。
```

**Step 5: 具体的な問題が語られる**

例：「そうです。私は怒りをコントロールできずに恋愛がいつも失敗します」

**Step 6: ミシェル心理学の概念を説明**

```
なるほど。怒りとは、ミシェル心理学では

①ネガティブな感情を感じる
②ネガティブな感情を感じたくない
③自分の心をガードする
④怒り発動

つまり、怒りは自分の心をガードするアクションです。

逆に言えば、①のネガティブな感情をコントロールできれば、
②に進まないので当然のこと、④に進まないですね？
```

**Step 7: 根本の感情を探る**

```
では①は何だと思いますか？
```

→ ここから五大ネガティブの特定に進む

#### アラジンのランプテストを使うべき状況

**使うべきケース：**
- 目標が曖昧、将来のビジョンが不明確
- 「どうしたいか分からない」と言っている
- 相談内容が軽め〜中程度の悩み
- ユーザーが言語化が得意そう
- 対話的に進めたい場合

**心理テスト（6つの文章）を使うべきケース：**
- ヘビー級、深刻な悩み
- トラウマ、深い苦しみ
- 一発でガムテープを見つけたい
- ユーザーが混乱している
- 時間を短縮したい

**ポイント：**
- 最初に聞く（ガムテープの話をする前に）
- 迷子になったら何度も見せる
- 逆接的に見ることでガムテープを推測
- なりたい私は複数出てきてもOK。全て受け止める
- 必ず優先順位をつけてもらう（1番から取り組む）

### 3. 心理テスト（6つの文章）

**目的：** ガムテープを一発で見つける

- 6つの穴埋め文章を書いてもらう
- リラックスして書いてもらう
- 思いついたまま、サラサラっと

**ポイント：**
- 出てきたものはほぼ全部ガムテープ
- なりたい私とリンクしていることが多い

### 4. 優先順位をつける

**目的：** 取り組む順番を決める

- 全ての答えを俯瞰する
- 「1番気になる答えはどれですか？」と聞く
- 丸をつけてもらう

**ポイント：**
- 1番気になるものから取り組む
- 命に関わる言葉が出たらそこから
- 最終的には全部取り組みたい

### 5. ピールダウン

**目的：** 感情を深掘りしてガムテープを発見

- 「それはなぜ？」「その奥には何がある？」
- 感情の層を剥いでいく
- 五大ネガティブにたどり着くまで

**ポイント：**
- 急がない
- クライアントのペースに合わせる
- 答えを誘導しない

### 6. 五大ネガティブの特定

**目的：** 適切な解放テクニックを選ぶ。ネガティブ感情の種類によって解放方法が異なるため、正確に特定する必要がある。

#### 重要な前提

人間が苦しいのは、ネガティブな感情を感じるから苦しい。
その中でも最も苦しいのが五大ネガティブ（六大ネガティブ）である。

#### 五大ネガティブ（六大ネガティブ）

1. **恐怖**
2. **悲しみ**
3. **怒り**
4. **寂しさ**
5. **無価値感**
6. **罪悪感**（5つに追加）

さらに：
7. **悔しさ**
8. **恥ずかしさ**

#### 各ネガティブ感情の特定キーワードと身体症状

**恐怖**
- 身体症状：息苦しい、手足が震える、冷や汗をかく、胃がキュッとなる、足がすくむ、心臓がバクバクする
- キーワード：こわい、恐れ、不安、怖くて動けない

**悲しみ**
- 感覚：喪失感、失った感覚、心にぽっかり穴が空いた
- キーワード：悲しい、辛い、喪失、失った、いなくなった
- 例：大好きなおばあちゃんが亡くなった

**怒り**
- 身体症状：頭に血が上る、顔が熱くなる、拳を握りしめる、歯を食いしばる、体が熱くなる
- 感覚：爆発しそうになる、イライラする
- キーワード：ムカつく、腹が立つ、許せない
- **重要：怒りは「ガードの怒り」の可能性が高い。怒りの下に他の感情（恐怖、悲しみ、寂しさ、無価値感）が隠れていることが多い**

**寂しさ**
- 欲求：誰かに会いたい、声を聞きたい、抱きしめてほしい、温もりが欲しい
- 感覚：孤独感、ひとりぼっち
- キーワード：寂しい、孤独、ひとりぼっち、誰もいない
- **重要：感情がコントロールできないレベルの寂しさは、今起きていることが原因ではなく幼少期から続く寂しさであることが多い**

**無価値感**
- 感覚：消えてしまいたい、存在意義がない、役に立たない、劣等感、みじめな気持ち、自己否定
- キーワード：価値がない、いらない、消えたい、ダメな人間、役立たず

**罪悪感**
- 感覚：申し訳ない気持ち、謝りたい気持ち、後悔、取り返しがつかない感覚
- キーワード：罪、申し訳ない、悪いことをした、謝りたい

**悔しさ**
- 感覚：歯がゆい、もどかしい、負けた感覚
- キーワード：やり返したい、リベンジしたい、悔しい、負けた

**恥ずかしさ**
- 身体症状：顔が赤くなる
- 感覚：穴があったら入りたい、隠れたい、人に見られたくない、小さくなりたい
- キーワード：恥ずかしい、恥、恥らい

#### 特定のための質問例

1. 「今、一番強く感じている感情は何ですか？」
2. 「その時、体にはどんな反応がありますか？」（身体症状を聞く）
3. 「その感情を一言で表すとしたら？」
4. 「心臓がバクバクしますか？」「胃がキュッとなりますか？」（恐怖の確認）
5. 「消えてしまいたいと思いますか？」（無価値感の確認）
6. 「誰かに会いたい、抱きしめてほしいと感じますか？」（寂しさの確認）

#### 特定できない場合の対応

- 複数の感情が混ざっている可能性を伝える
- 「一番近いのはどれですか？」と絞り込む
- ユーザーの言葉や身体症状から推測して「〇〇に近いように感じますが、どうでしょう？」と提案

**ポイント：**
- 怒りは「ガードの怒り」の可能性が非常に高い。必ず怒りの下の感情を探る
- 寂しさが強い場合は幼少期のトラウマを疑う
- 身体症状を聞くことで特定しやすくなる
- ユーザーの言葉に出てくるキーワードに注目する
- 無理に1つに絞らなくてもOK。複数の感情が同時に存在することもある

### 7. 解放テクニック

**目的：** ガムテープを剥がす。ネガティブ感情の種類によって解放方法が異なる。

#### 各ネガティブ感情に対する解放方法

**恐怖 → 勇気**

```
とても怖いのは理解できます。
しかし、必ず大丈夫です。
あなたの中の勇気を出してください。

恐怖を感じることは悪いことではありません。
その恐怖を認めたうえで、小さな一歩でいいので
勇気を持って行動してみませんか？
```

- 励まし、背中を押す
- 「大丈夫」と安心させる
- 小さな行動から始めることを提案

**悲しみ → 時間**

```
今はとても辛い時期ですね。
悲しみに対しては、どうすることもできません。
とにかく時間が過ぎるのを待ちましょう。

焦らなくて大丈夫です。
必ず楽になる日が訪れます。
今は、涙を流したい時に流してください。
```

- 時間が必要であることを伝える
- 焦らせない
- 涙を流すことを許可する
- 必ず楽になることを保証する

**怒り → 許し**

```
怒りを感じているのですね。
怒りは、あなたの心を守るためのガードです。

まず、あなた自身を許してください。
そして、相手を許すプロセスを一緒に考えてみませんか？

許すことは、相手のためではなく、
あなた自身を自由にするためです。
```

- ガードの怒りの場合、手前の感情（恐怖、悲しみ、寂しさ、無価値感）を見つける
- その手前の感情を解放する
- 相手を許す、自分を許す
- 許しは自分のためであることを伝える

**寂しさ → 幼少期の癒し + 行動**

```
感情がコントロールできないレベルの寂しさは、
今起きていることが原因ではなく、
幼少期から続く寂しさであることが多いです。

幼少期の寂しかったトラウマを思い出してみてください。
その時の小さなあなたを、今の大人のあなたが抱きしめて
癒してあげてください。

そして、今の大人のあなたは、自分の手と足を使って、
寂しくならないように人生を謳歌することができます。
誰かとつながる行動を一緒に考えてみましょう。
```

- 幼少期のトラウマを特定する
- インナーチャイルドを癒す
- 大人の自分が行動できることを伝える
- 具体的な行動（人とつながる）を提案

**無価値感 → 受け入れる**

```
あなたは何も間違っていません。
あなたはありのままで価値があります。
あなたは存在するだけで素晴らしいのです。

自分を認めてください。
自分を受け入れてください。
現実を受け入れてください。

あなたの価値は、誰かに認められることで決まるのではありません。
あなたは生まれた瞬間から価値があるのです。
```

- 存在を肯定する
- 「ありのまま」を強調
- 自己受容を促す
- 他者評価に依存しないことを伝える

**罪悪感 → 許し**

```
罪悪感を感じているのですね。
あなたは、自分を責めていませんか？

でも、それはあなたが悪いのではありません。
自分を許してください。
過去の自分も、今の自分も、全て許してください。

罪悪感から解放されることで、
あなたは前に進むことができます。
```

- 自分を責めていることを認識させる
- 「あなたが悪いのではない」と伝える
- 自己許容を促す
- 過去を手放すことを提案

**ポイント：**
- クライアントの決断が必要
- 向き合う準備ができているか確認
- 無理強いしない
- ネガティブの種類を正確に特定してから適切な解放方法を提示
- 複数のネガティブが混在している場合は、1つずつ取り組む

### 8. クロージング

**目的：** セッションを締めくくる

- 今日の気づきを確認
- 次回に向けての課題（必要なら）
- 感謝を伝える

**ポイント：**
- 急に終わらせない
- クライアントの状態を確認
- フォローアップの予定を伝える

|----------|-------------------|
| 1. アイスブレイク | 5分 |
| 2. なりたい私 | 5分 |
| 3. 心理テスト | 10分 |
| 4. 優先順位 | 5分 |
| 5. ピールダウン | 15分 |
| 6. 五大ネガティブ特定 | 5分 |

| 8. クロージング | 5分 |

## よくある誤解・注意点

- ❌ 流れ通りに進めなければならない
- ⭕ クライアントの状態に合わせて柔軟に対応

- ❌ 1回で全て解決しなければならない
- ⭕ 複数回のセッションで徐々に解放していく

- ❌ カウンセラーが答えを出す
- ⭕ 答えは100%クライアントが知っている

## テープ式カウンセリングの時間枠

### 基本的な考え方

テープ式は1時間1回というやり方をしない。**最低でも1ヶ月間**（月4回）。

### 1ヶ月目のタイムライン

| 週 | 内容 |
|----|------|
| 1週目 | なりたい私＋心理テスト→ガムテープ発見 |
| 2週目 | ガムテープに取り組み始める |
| 3週目 | ガムテープに取り組む |
| 4週目 | 剥がしていく |

**最初の1時間でなりたい私と心理テストを行い、ガムテープを見つけてしまう。2週間目からもうガムテープに取り組んでいく。**

### カウンセラー側の経済的視点

カウンセラーも生活がある：
- 前時間：情報分析、予習
- 本番：1時間〜1時間半
- 後時間：サポートのLINEを送る
- 結局1回で3〜4時間使う
- だから1時間1,000円や2,000円ではできない
- カウンセリングはどこでも高くなる

### 相談者目線の考え方

**テープ式は相談者目線**：
- わざと引き延ばして儲けるという考え方は1mmもない
- なるべく時間も短く、経済的負担も少なく
- 1発で気づきが起きたら一瞬

会話しながらガムテープを探すと2〜3ヶ月かかる。心理テストを使えば最初から一発で見つかる。これが相談者のためになる。

## テープ式カウンセリングの実践的な流れ

### 初回カウンセリングの詳細

**ステップ1：ご挨拶**
- 「はじめまして」から始まる
- 緊張をほぐす

**ステップ2：何が起きているのか聞く**
- 「どうされましたか」「何に苦しんでおられますか」
- 「何にお悩みですか」は使わない（命に関わるレベルの方には軽く感じる）
- 相手がうまく説明できなくても気にしない
- この段階では落書きしない
- 聞きながら推測する（恋愛相談か、ヘビー級か、ペルソナか、吐き出しか）

**ステップ3：推測と判断**
聞きながら判断する：
- ガムテープを剥がす必要があるか
- 吐き出しだけで解決するパターンか
- 意味づけを変えるだけで楽になるパターンか
- テープ式心理学が必要なヘビー級か

**ステップ4：テープ式心理学の説明**（ガムテープを触る必要がある場合）
- 私たちが苦しいのは精神病ではなく思い込みである
- ネガティブな感情の発生の仕組みを説明
- ガムテープとは何かを説明
- **ここで納得してもらわないと進まない**

**ステップ5：精神病ではないことの説明**（必要な場合）
- 「発達障害と言われた」など精神病だと思い込んでいる場合
- 1時間使ってでも説明する
- 納得してもらえないとガムテープに進めない

**ステップ6：なりたい私を聞く**
- **心理テストより先に聞く**（この順番が大事）
- アラジンの魔法のランプの質問を使う
- 全部出しきるまで続ける
- 「全部叶ったら今日生きてて良かったと思いますか」と確認
- 「思わない」なら「もっとあるんじゃないですか」と続ける

**ステップ7：心理テスト**
- 6つの文章をリラックスして書いてもらう
- 出てきたものはほぼ全部ガムテープ

**ステップ8：優先順位をつける**
- 「1番気になるのはどれですか」「2番目は」と聞く
- なりたい私と繋がっているか確認（ほぼ繋がっている）

**ステップ9：ガムテープ1枚1枚取り組む**
- **2つ同時にやらない**（スパゲティ状態になる）
- 今どれを触っているか必ず伝える
- 進まなかったら「一旦置いておきましょう」と次へ

### セッション回数の目安

| 状況 | 回数 |
|-----|------|
| 吐き出しだけで解決 | 1回 |
| 意味づけを変えるだけ | 1〜2回 |
| テープ式の説明で1時間使った場合 | 2回目からガムテープ |
| ヘビー級 | 複数回 |

### 話を聞くことが仕事ではない

- テープ式心理カウンセラーは「話を聞く」ことがメインではない
- 一般的なカウンセラー：話を聞くだけで終わる（楽だが効果が限定的）
- テープ式：話を聞きながら**ガムテープを見つける**ことが仕事
- 話を聞きながら**違和感を見つける**ことが仕事

## 落書きカウンセリング

### 落書きカウンセリングとは

相談者の話を聞きながら、A4の白紙にどんどんメモを取っていく手法。清書ではなく「落書き」レベルで良い。

### なぜ「落書き」か

- 相談者は待ってくれない
- 次々話してくる
- 綺麗に書いている暇がない
- ポイントだけをメモる

### 落書きの例

```
4歳 母親 暴力
家に帰ってこなかった
→ 寂しさ？無価値？
```

### 落書きの使い方

1. A4白紙を何枚もテーブルに置いておく
2. 話を聞きながらどんどん書いていく
3. 書いた紙を並べる
4. 俯瞰して見る
5. 繋がりを見つける

**例：**
- 1枚目：心理テストのガムテープ「私はひとりぼっちである」
- 2枚目：話の中で出てきた「1人が好き」という違和感
- →「これ繋がってますか？」と確認

### 違和感をメモる

話を聞いていると違和感が出てくる：
- 「私って価値がないですよ」（さらっと言う）
- 「人より後でいいんですよ」
- 「1番大事にされなくていいんです」
- 「友達って何かわからないんです」

→ その場で拾わなくていい。メモだけしておく。

### 後で繋げる

```
カウンセラー：「ところで、さっき『1人が好き』って
             言ってましたけど、それって何ですか」

相談者：「えっと、確かに1人が好きって思い込んでましたけど、
       実はそれ隠してるだけですかね」
```

### カウンセラーが迷子にならないために

- 常に「今どのガムテープを触っているか」を把握
- 相談者にも伝える：「今これを触ってますよ」
- 脱線しても戻れる
- 進まなかったら「一旦置いて次行きますね」

## まとめ

カウンセリングの流れは、ご挨拶→何が起きているか聞く→テープ式心理学の説明→なりたい私→心理テスト→優先順位→ピールダウン→五大ネガティブ特定→解放テクニック。話を聞くことがメインではなく、話を聞きながらガムテープと違和感を見つけることが仕事。落書きカウンセリングで俯瞰しながら繋がりを見つけ、常に今どのガムテープを触っているかを把握する。答えは100%クライアントが知っているという姿勢を忘れない。
