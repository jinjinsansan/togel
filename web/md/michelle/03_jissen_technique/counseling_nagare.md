# カウンセリングの流れ

## 定義

テープ式心理学に基づくカウンセリングの標準的な流れ。クライアントのガムテープを発見し、気づきを促し、解放へ導くためのプロセス。

## 全体の流れ

```
1. アイスブレイク・ラポール形成
    ↓
2. なりたい私を聞く
    ↓
3. 心理テスト（6つの文章）
    ↓
4. 優先順位をつける
    ↓
5. ピールダウン
    ↓
6. 五大ネガティブの特定
    ↓
7. 解放テクニック
    ↓
8. クロージング
```

## 各ステップの詳細

### 1. アイスブレイク・ラポール形成

**目的：** 安心できる場を作る

- 挨拶、自己紹介
- 緊張をほぐす会話
- 「今日はどんな気持ちで来ましたか？」
- カウンセリングの流れを説明

**ポイント：**
- 焦らない
- クライアントが話しやすい雰囲気を作る
- アクティブリスニングを心がける

### 2. なりたい私を聞く

**目的：** ゴールを明確にする

- アラジンの魔法のランプの質問を使う
- 「なりたい私」を落書きで書いてもらう
- 複数出てきてもOK

**ポイント：**
- 最初に聞く（ガムテープの話をする前に）
- 迷子になったら何度も見せる
- 逆接的に見ることでガムテープを推測

### 3. 心理テスト（6つの文章）

**目的：** ガムテープを一発で見つける

- 6つの穴埋め文章を書いてもらう
- リラックスして書いてもらう
- 思いついたまま、サラサラっと

**ポイント：**
- 出てきたものはほぼ全部ガムテープ
- なりたい私とリンクしていることが多い

### 4. 優先順位をつける

**目的：** 取り組む順番を決める

- 全ての答えを俯瞰する
- 「1番気になる答えはどれですか？」と聞く
- 丸をつけてもらう

**ポイント：**
- 1番気になるものから取り組む
- 命に関わる言葉が出たらそこから
- 最終的には全部取り組みたい

### 5. ピールダウン

**目的：** 感情を深掘りしてガムテープを発見

- 「それはなぜ？」「その奥には何がある？」
- 感情の層を剥いでいく
- 五大ネガティブにたどり着くまで

**ポイント：**
- 急がない
- クライアントのペースに合わせる
- 答えを誘導しない

### 6. 五大ネガティブの特定

**目的：** 適切な解放テクニックを選ぶ

五大ネガティブのどれかを特定：
- 恐怖
- 悲しみ
- 怒り
- 寂しさ
- 無価値観

**ポイント：**
- 怒りは「ガードの怒り」の可能性
- 怒りの下に他の感情が隠れていることが多い

### 7. 解放テクニック

**目的：** ガムテープを剥がす

| ネガティブ | 解放方法 |
|-----------|----------|
| 恐怖 | 勇気を持って行動する |
| 悲しみ | 時間をかけて涙を流す |
| 怒り | 許す（相手と自分を） |
| 寂しさ | 全体性の感覚 + 行動 |
| 無価値観 | ありのままを受け入れる |

**ポイント：**
- クライアントの決断が必要
- 向き合う準備ができているか確認
- 無理強いしない

### 8. クロージング

**目的：** セッションを締めくくる

- 今日の気づきを確認
- 次回に向けての課題（必要なら）
- 感謝を伝える

**ポイント：**
- 急に終わらせない
- クライアントの状態を確認
- フォローアップの予定を伝える

## セッションの時間配分の目安

| ステップ | 時間（60分の場合） |
|----------|-------------------|
| 1. アイスブレイク | 5分 |
| 2. なりたい私 | 5分 |
| 3. 心理テスト | 10分 |
| 4. 優先順位 | 5分 |
| 5. ピールダウン | 15分 |
| 6. 五大ネガティブ特定 | 5分 |
| 7. 解放テクニック | 10分 |
| 8. クロージング | 5分 |

※あくまで目安。クライアントのペースに合わせる。

## よくある誤解・注意点

- ❌ 流れ通りに進めなければならない
- ⭕ クライアントの状態に合わせて柔軟に対応

- ❌ 1回で全て解決しなければならない
- ⭕ 複数回のセッションで徐々に解放していく

- ❌ カウンセラーが答えを出す
- ⭕ 答えは100%クライアントが知っている

## テープ式カウンセリングの時間枠

### 基本的な考え方

テープ式は1時間1回というやり方をしない。**最低でも1ヶ月間**（月4回）。

### 1ヶ月目のタイムライン

| 週 | 内容 |
|----|------|
| 1週目 | なりたい私＋心理テスト→ガムテープ発見 |
| 2週目 | ガムテープに取り組み始める |
| 3週目 | ガムテープに取り組む |
| 4週目 | 剥がしていく |

**最初の1時間でなりたい私と心理テストを行い、ガムテープを見つけてしまう。2週間目からもうガムテープに取り組んでいく。**

### カウンセラー側の経済的視点

カウンセラーも生活がある：
- 前時間：情報分析、予習
- 本番：1時間〜1時間半
- 後時間：サポートのLINEを送る
- 結局1回で3〜4時間使う
- だから1時間1,000円や2,000円ではできない
- カウンセリングはどこでも高くなる

### 相談者目線の考え方

**テープ式は相談者目線**：
- わざと引き延ばして儲けるという考え方は1mmもない
- なるべく時間も短く、経済的負担も少なく
- 1発で気づきが起きたら一瞬

会話しながらガムテープを探すと2〜3ヶ月かかる。心理テストを使えば最初から一発で見つかる。これが相談者のためになる。

## テープ式カウンセリングの実践的な流れ

### 初回カウンセリングの詳細

**ステップ1：ご挨拶**
- 「はじめまして」から始まる
- 緊張をほぐす

**ステップ2：何が起きているのか聞く**
- 「どうされましたか」「何に苦しんでおられますか」
- 「何にお悩みですか」は使わない（命に関わるレベルの方には軽く感じる）
- 相手がうまく説明できなくても気にしない
- この段階では落書きしない
- 聞きながら推測する（恋愛相談か、ヘビー級か、ペルソナか、吐き出しか）

**ステップ3：推測と判断**
聞きながら判断する：
- ガムテープを剥がす必要があるか
- 吐き出しだけで解決するパターンか
- 意味づけを変えるだけで楽になるパターンか
- テープ式心理学が必要なヘビー級か

**ステップ4：テープ式心理学の説明**（ガムテープを触る必要がある場合）
- 私たちが苦しいのは精神病ではなく思い込みである
- ネガティブな感情の発生の仕組みを説明
- ガムテープとは何かを説明
- **ここで納得してもらわないと進まない**

**ステップ5：精神病ではないことの説明**（必要な場合）
- 「発達障害と言われた」など精神病だと思い込んでいる場合
- 1時間使ってでも説明する
- 納得してもらえないとガムテープに進めない

**ステップ6：なりたい私を聞く**
- **心理テストより先に聞く**（この順番が大事）
- アラジンの魔法のランプの質問を使う
- 全部出しきるまで続ける
- 「全部叶ったら今日生きてて良かったと思いますか」と確認
- 「思わない」なら「もっとあるんじゃないですか」と続ける

**ステップ7：心理テスト**
- 6つの文章をリラックスして書いてもらう
- 出てきたものはほぼ全部ガムテープ

**ステップ8：優先順位をつける**
- 「1番気になるのはどれですか」「2番目は」と聞く
- なりたい私と繋がっているか確認（ほぼ繋がっている）

**ステップ9：ガムテープ1枚1枚取り組む**
- **2つ同時にやらない**（スパゲティ状態になる）
- 今どれを触っているか必ず伝える
- 進まなかったら「一旦置いておきましょう」と次へ

### セッション回数の目安

| 状況 | 回数 |
|-----|------|
| 吐き出しだけで解決 | 1回 |
| 意味づけを変えるだけ | 1〜2回 |
| テープ式の説明で1時間使った場合 | 2回目からガムテープ |
| ヘビー級 | 複数回 |

### 話を聞くことが仕事ではない

- テープ式心理カウンセラーは「話を聞く」ことがメインではない
- 一般的なカウンセラー：話を聞くだけで終わる（楽だが効果が限定的）
- テープ式：話を聞きながら**ガムテープを見つける**ことが仕事
- 話を聞きながら**違和感を見つける**ことが仕事

## 落書きカウンセリング

### 落書きカウンセリングとは

相談者の話を聞きながら、A4の白紙にどんどんメモを取っていく手法。清書ではなく「落書き」レベルで良い。

### なぜ「落書き」か

- 相談者は待ってくれない
- 次々話してくる
- 綺麗に書いている暇がない
- ポイントだけをメモる

### 落書きの例

```
4歳 母親 暴力
家に帰ってこなかった
→ 寂しさ？無価値？
```

### 落書きの使い方

1. A4白紙を何枚もテーブルに置いておく
2. 話を聞きながらどんどん書いていく
3. 書いた紙を並べる
4. 俯瞰して見る
5. 繋がりを見つける

**例：**
- 1枚目：心理テストのガムテープ「私はひとりぼっちである」
- 2枚目：話の中で出てきた「1人が好き」という違和感
- →「これ繋がってますか？」と確認

### 違和感をメモる

話を聞いていると違和感が出てくる：
- 「私って価値がないですよ」（さらっと言う）
- 「人より後でいいんですよ」
- 「1番大事にされなくていいんです」
- 「友達って何かわからないんです」

→ その場で拾わなくていい。メモだけしておく。

### 後で繋げる

```
カウンセラー：「ところで、さっき『1人が好き』って
             言ってましたけど、それって何ですか」

相談者：「えっと、確かに1人が好きって思い込んでましたけど、
       実はそれ隠してるだけですかね」
```

### カウンセラーが迷子にならないために

- 常に「今どのガムテープを触っているか」を把握
- 相談者にも伝える：「今これを触ってますよ」
- 脱線しても戻れる
- 進まなかったら「一旦置いて次行きますね」

## まとめ

カウンセリングの流れは、ご挨拶→何が起きているか聞く→テープ式心理学の説明→なりたい私→心理テスト→優先順位→ピールダウン→五大ネガティブ特定→解放テクニック。話を聞くことがメインではなく、話を聞きながらガムテープと違和感を見つけることが仕事。落書きカウンセリングで俯瞰しながら繋がりを見つけ、常に今どのガムテープを触っているかを把握する。答えは100%クライアントが知っているという姿勢を忘れない。
