# 落書きカウンセリング

## 定義

落書きカウンセリングとは、クライアントに「落書き」として自由に書かせることで、ペルソナ（仮面）を外し、本音を引き出すテクニック。

## 背景・目的

### なぜ「落書き」と呼ぶか

- 「落書き」と言うと心理的ハードルが下がる
- 「ちゃんと書かなきゃ」というプレッシャーがなくなる
- 思考ちゃんを使わずに書ける
- 本音が出やすくなる

### ペルソナの問題

- クライアントはカウンセラーの前で「いい人」を演じがち
- 「こう言ったら嫌われるかも」という思いがある
- ペルソナがあると本音が出てこない
- 本音が出ないとガムテープが見つからない

## 中核ポイント

### 落書きカウンセリングの進め方

```
1. 紙とペンを用意してもらう（またはチャットに打ってもらう）
2. 「落書きなので、思ったことをそのまま書いてください」と伝える
3. 「誰も見ません」「正解はありません」と伝える
4. テーマに沿って自由に書いてもらう
5. 書いたものを共有してもらう（任意）
```

### 使う場面

| 場面 | 使い方 |
|------|--------|
| なりたい私を聞く時 | 「落書きで書いてみてください」 |
| 心理テストを行う時 | 「落書きなので思ったまま」 |
| 感情を言語化してもらう時 | 「落書きで表現してみて」 |
| 本音を引き出したい時 | 「落書きだから何でもOK」 |

### 効果的なフレーズ

- 「落書きなので、思いついたまま書いてください」
- 「誰も見ないので、本音で大丈夫です」
- 「日本語として変でも構いません」
- 「正解はありません」
- 「思考を使わずに、サラサラっと」

## 落書きの効果

### 1. ペルソナが外れる

- 「落書き」と言うだけで肩の力が抜ける
- 「ちゃんとしなきゃ」がなくなる
- 素の自分が出やすくなる

### 2. 思考ちゃんをバイパスする

- 考えすぎると本音が出てこない
- 「落書き」は思考を経由しない
- マインド様（本当の自分）の声が出やすい

### 3. 言語化が促進される

- 漠然とした感情が言葉になる
- 書くことで整理される
- 自分でも気づいていなかったことが見える

## オンラインカウンセリングでの活用

### チャットの場合

- 「チャットに落書きで打ってください」
- 「思いついたまま送ってください」
- 「途中で送信しても大丈夫です」

### ビデオ通話の場合

- 画面共有で書いてもらう
- 紙に書いて見せてもらう
- 口頭で「落書きのつもりで話して」

## 落書きカウンセリングの例

### 例1：なりたい私

```
カウンセラー：「なりたい私を落書きで書いてみてください。
             誰も見ないので、本音で大丈夫です。
             いくつ書いても構いません。」

クライアント：
- 人の目を気にしない私
- お金の心配をしない私
- 自由に生きている私
- 愛されている私
```

### 例2：今の気持ち

```
カウンセラー：「今の気持ちを落書きで表現してみてください。
             言葉じゃなくても、絵でも、記号でも何でもいいです。」

クライアント：「ぐちゃぐちゃ」「真っ黒」「重い」
```

→ そこから深掘りしていく

## よくある誤解・注意点

- ❌ 落書きだから適当でいい
- ⭕ 落書きだからこそ本音が出る。とても重要な情報

- ❌ クライアントが書いたものを評価する
- ⭕ 書いたものに正解・不正解はない。そのまま受け取る

- ❌ 落書きを強制する
- ⭕ 書きたくなければ書かなくてもいい。無理強いしない

## まとめ

落書きカウンセリングは、「落書き」というフレームを使うことでクライアントのペルソナを外し、本音を引き出すテクニック。「思いついたまま」「誰も見ない」「正解はない」と伝えることで、思考ちゃんをバイパスし、マインド様の声を引き出す。なりたい私、心理テスト、感情の言語化など様々な場面で活用できる。
