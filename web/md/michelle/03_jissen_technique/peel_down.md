# ピールダウン

## 定義

ピールダウンとは、感情を深掘りしていくことでガムテープ（思い込み）を発見するテクニック。「ピール（peel）」は「剥く」という意味で、玉ねぎの皮を剥いていくように感情の層を剥いでいく。

## 背景・目的

### なぜピールダウンが必要か

- 表面的な感情の下に本当の感情が隠れている
- 本人も気づいていない深層の感情がある
- ガムテープは表面からは見えにくい
- 深掘りすることで根本原因にたどり着く

### ピールダウンの目的

1. 五大ネガティブのどれかを特定する
2. その下にあるガムテープを発見する
3. 気づきを促す

## 中核ポイント

### ピールダウンの基本構造

```
表面の感情
    ↓ 「それはなぜ？」
より深い感情
    ↓ 「それはなぜ？」
さらに深い感情
    ↓ 「それはなぜ？」
五大ネガティブ（恐怖・悲しみ・怒り・寂しさ・無価値観）
    ↓
ガムテープ（思い込み）
```

### 使う質問

| 質問 | 用途 |
|------|------|
| それはなぜ？ | 基本の深掘り |
| その感情は何？ | 感情の特定 |
| その奥には何がある？ | さらに深く |
| それを感じた時、体のどこで感じる？ | 身体感覚への誘導 |
| その感情に名前をつけるなら？ | 言語化の促進 |

### ピールダウンの例

**相談：** 「上司に怒られるのが辛い」

```
カウンセラー：「上司に怒られると、どんな気持ちになりますか？」
クライアント：「嫌な気持ちになります」

カウンセラー：「その嫌な気持ちをもう少し教えてください」
クライアント：「自分がダメだと思います」

カウンセラー：「自分がダメだと思うと、どんな感情が出てきますか？」
クライアント：「悲しい気持ちです」

カウンセラー：「その悲しみの奥には何がありますか？」
クライアント：「認めてもらえない悲しさ」

カウンセラー：「認めてもらえないと、どう感じますか？」
クライアント：「自分には価値がないと思います」
```

→ **発見されたガムテープ：「私には価値がない」**

## 実践のコツ

### 1. 急がない

- クライアントのペースに合わせる
- 答えを待つ
- 沈黙を恐れない

### 2. 五大ネガティブを意識する

深掘りしていくと、必ず以下のどれかにたどり着く：
- 恐怖
- 悲しみ
- 怒り
- 寂しさ
- 無価値観

### 3. 身体感覚を活用する

- 「それを感じた時、体のどこで感じますか？」
- 胸、お腹、喉など身体の部位を聞く
- 身体感覚から感情を特定しやすくなる

### 4. ガードの怒りに注意

- 怒りが出てきたら「その怒りの下には何がありますか？」
- 怒りは他の感情を守っている場合が多い
- 怒りの下に悲しみや恐怖が隠れていることが多い

## ピールダウンの終着点

### 終着点の特徴

- 五大ネガティブのいずれかにたどり着く
- ガムテープ（思い込み）が言語化される
- クライアント自身が「あ、そうか」と気づく

### 気づきの瞬間

- 「あ、私これをずっと信じてたんだ」
- 「そうか、これが苦しみの原因だったんだ」
- この瞬間がピールダウンの成功

## よくある誤解・注意点

- ❌ 答えを誘導する
- ⭕ クライアント自身の言葉を待つ。答えは100%クライアントが知っている

- ❌ 早く答えにたどり着こうとする
- ⭕ 急がない。クライアントのペースが大事

- ❌ カウンセラーが「それは無価値観ですね」と断定する
- ⭕ クライアント自身に気づかせる。「それは何という感情ですか？」と聞く

## まとめ

ピールダウンは感情を深掘りしてガムテープを発見するテクニック。「それはなぜ？」「その奥には何がある？」という質問を繰り返し、五大ネガティブのいずれかにたどり着き、その下にあるガムテープ（思い込み）を発見する。クライアント自身が気づくことが重要で、カウンセラーは答えを誘導しない。
