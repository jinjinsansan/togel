# Why（なぜ）を使わない

## 定義

ピールダウンやカウンセリングにおいて、「なぜ」「どうして」「なんで」という質問を使わず、「どう」（How）を使う質問技法。

## 背景・目的

### なぜWhyを使わないのか

Whyを使うと：
- 理論が出てくる
- 言い訳が出てくる
- 本当に聞きたいネガティブの感情の種類が現れてこない

### 目的

感情を引き出し、ピールダウンを効果的に進めること。

## 中核ポイント

### Whyを使った場合の問題

```
❌ 悪い聞き方：
「どうしてうまくできないと思うんですか？」

→ 相談者の回答：
「私はこうでこうで、昔からうつ病と診断されていて、
経済的にもこうで...」

結果：理論・言い訳が出てくる（感情が出てこない）
```

### Howを使った場合

```
✅ 良い聞き方：
「うまくできなかったらどう感じますか？」

→ 相談者の回答：
「愛されないと感じます」

結果：感情が出てくる（ピールダウンが進む）
```

## Why vs How の比較

| Why（なぜ） | How（どう） |
|------------|-----------|
| 理論が出る | 感情が出る |
| 言い訳が出る | 本音が出る |
| 夫婦喧嘩のような議論になる | ピールダウンが進む |
| 一般社会でよく使われる | わざと使う（意図的に） |
| 防御反応を引き起こす | 内省を促す |

## 使うべき質問フレーズ

### 推奨フレーズ

- 「どう感じますか？」
- 「どうなりますか？」
- 「どういうことですか？」
- 「それはどういう意味ですか？」
- 「〜だったらどう感じますか？」
- 「〜したらどうなりますか？」

### 避けるべきフレーズ

- 「なぜそう思うんですか？」
- 「どうしてそうなったんですか？」
- 「なんでそうしたんですか？」
- 「何で〜なんですか？」

## 具体例

### 例1：仕事の悩み

```
相談者：「上司に怒られて落ち込みました」

❌「なぜ落ち込んだんですか？」
→「だって理不尽だし、私は悪くないのに...」（言い訳）

✅「怒られるとどう感じますか？」
→「自分がダメな人間だと感じます」（感情）
```

### 例2：人間関係の悩み

```
相談者：「友達に裏切られました」

❌「なぜ裏切られたと思うんですか？」
→「あの子は昔からこうで、私がこうしたから...」（分析・言い訳）

✅「裏切られたらどう感じますか？」
→「一人ぼっちだと感じます」（感情 → 寂しさ）
```

### 例3：自己否定

```
相談者：「私は何をやってもダメなんです」

❌「どうしてダメだと思うんですか？」
→「だって今まで全部失敗してきたし...」（過去の羅列）

✅「ダメだったらどう感じますか？」
→「愛されないと感じます」（感情 → 無価値観）
```

## なぜ一般社会ではWhyが使われるのか

### 日常会話での「なぜ」

- 「なぜ遅刻したの？」
- 「どうして連絡しなかったの？」
- 「なんでそんなことしたの？」

これらは**責める**ニュアンスが含まれている。

### カウンセリングでの問題

- 責められている感じがする
- 防御反応が起きる
- 正当化しようとする
- 本当の感情が出てこない

## よくある誤解・注意点

- ❌ 「なぜ」は絶対に使ってはいけない
- ⭕ 文脈によっては使っても問題ない場合もある。ただし、感情を引き出したい時は避ける

- ❌ 「How」なら何でもいい
- ⭕ 「どう感じますか？」のように感情に焦点を当てた質問が重要

## まとめ

ピールダウンやカウンセリングでは「なぜ」「どうして」を避け、「どう感じますか？」「どうなりますか？」などのHow系の質問を使う。Whyは理論や言い訳を引き出し、Howは感情を引き出す。感情を引き出すことで、核となるネガティブの種類（五大ネガティブ）に到達できる。
