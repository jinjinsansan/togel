# マインド様

## 定義

マインド様とは、「**本当の自分**」「**本音**」「**心の声**」を指すテープ式心理学の用語。思考ちゃん（頭で考えること）の奥にある、真の願いや感情を持つ存在。

## 背景・目的

### なぜ「様」をつけるか

- 尊いものとして扱う
- 大切にすべき存在
- 敬意を込めて

### マインド様の位置づけ

```
表面
  ↓
ペルソナ（仮面）
  ↓
思考ちゃん（思考、分析、判断）
  ↓
マインド様（本当の自分、本音）← ここにアクセスしたい
```

### マインドエリアの構造（詳細）

私を構成する3つの要素（ボディ・マインド・Y）のうち、マインド部分の詳細構造：

```
【マインドエリア全体】
┌────────────────────────────────────┐
│                                    │
│    マインド様エリア（超巨大）         │
│    - 感覚のみ（言葉がない）          │
│    - 「今」だけ存在する              │
│    - アーキタイプ（原型）がある       │
│                                    │
│         ・← 思考ちゃんエリア        │
│           （鉛筆の芯ぐらい小さい）   │
│           - 言葉を使う              │
│           - 過去と未来を担当        │
│           - ネガティブを担当        │
│                                    │
└────────────────────────────────────┘
```

**サイズの比較：**
- マインド様エリア：超超超超が付くくらい巨大
- 思考ちゃんエリア：鉛筆の芯くらい小さい
- 比率は50対50ではない。何十万対1くらいの差

### スクリーンの例え（トリシアの教え）

映画のスクリーンで例える：

| 状態 | 説明 |
|------|------|
| スクリーンの中の俳優 | 思考ちゃんエリアに入り込んでいる状態。「私は人生がうまくいかない」「寂しい」と一生懸命演じている |
| スクリーンそのもの | マインド様（本当の私）。どのような映画でもチャンネルを変えれば映せる |

**トリシアの言葉：**
「あなたは一生懸命スクリーンの中の俳優さん本人だと思い込んで苦しんでいるけれど、あなたはスクリーンそのものなんだよ。スクリーンの中に映し出す映画はあなたが選べるんだよ。」

### 心の目（フォーカス）

心に目があるとしたら、どこを見ているかで全く違う：

| フォーカス先 | 結果 |
|-------------|------|
| 思考ちゃんエリア | ネガティブばかり感じる。「もうダメだ」「怖い」「私は価値がない」 |
| マインド様エリア | 思考ちゃんが黙り、本来の私になれる |

**俯瞰できるようになる：**
心を見る訓練を積むと、マインドエリア側から自分の心の中を俯瞰できる。
- 「今、思考ちゃんを選ぼうとしているな」
- 「今、ネガティブに意味付けしようとしているな」
これが丸見えになる。

### バシャールの「2本の角」

バシャールは「あなたの2本の角を両方同じ方向に向けてください」と言っている。

**ダブルバインドの状態：**
```
思考ちゃん：「嫌だ嫌だ、私は不幸になる」 → 不幸の方向
マインド様：「幸せになりたい」 → 幸せの方向
```

2本の角が全く真逆を向いている状態。

**統合した状態：**
バシャールが言うのは「本当の願いは1つでしょ」ということ。2本の角を同じ方向（幸せの方向）に向けること。

**フォーカスの違いで見える世界が変わる：**
- 思考ちゃんにべったりくっついているとき → マインドエリアがあることすら見えない、受け入れられない
- 気づきが起きると → マインドエリア側から俯瞰して思考ちゃんが見れる
- 立ち位置が一気に変わる

## 中核ポイント

### マインド様の特徴

| 特徴 | 説明 |
|------|------|
| 本音を知っている | 本当に何を望んでいるか知っている |
| 傷を持っている | 過去のトラウマを抱えている |
| 幸せを望んでいる | 全ての人のマインド様は幸せを望む |
| 答えを持っている | 解決の糸口を知っている |

### 思考ちゃんとの違い

| 思考ちゃん | マインド様 |
|-----------|-----------|
| 頭で考える | 心で感じる |
| 分析的 | 直感的 |
| 防衛的 | 正直 |
| うるさい | 静か |
| 嘘をつける | 嘘をつかない |

### マインド様が隠れる理由

- 思考ちゃんが守っている
- ペルソナで覆われている
- 傷つくのが怖いから
- ガムテープで押さえつけられている

## 顕在意識・潜在意識・集合意識

### 意識の層（イメージとして）

```
【意識の層】
┌─────────────────────────┐
│ 顕在意識                 │ ← 思考ちゃんの状態
│ （チカチカ忙しい状態）    │
├─────────────────────────┤
│ 潜在意識                 │ ← マインド様の状態
│ （リラックス・静か）      │    ★ここからチャイルドにアクセス可能
├─────────────────────────┤
│ 集合意識                 │ ← 完全な瞑想・睡眠状態
│ （深い瞑想・眠り）        │
└─────────────────────────┘
```

**注意：インターネットで調べないこと。間違った情報と混ざると仁さんの伝えたいことが変わってしまう。**

### インナーチャイルドへのアクセス

- **顕在意識（思考ちゃん状態）からはアクセスできない**
- 潜在意識（マインド様状態）に入って初めてチャイルドに触れられる
- だからインナーチャイルドセラピーは瞑想・深呼吸から始まる

### なぜ瞑想が必要か

思考ちゃんが「チカチカチカチカ」忙しい状態：
- LINEの返事、スマホの返事など
- わーっとなっている状態ではチャイルドは出てこない

深呼吸してリラックス → 思考ちゃんが黙る → 潜在意識に入る → チャイルドにアクセス可能

### 訓練を積むと

- いちいち目を閉じなくても出せるようになる
- マインドエリアから見ると、瞑想しなくても出せるようになる
- 仁さんは目を開けたまま、心でちょっとすればすぐにチャイルドが出てくる

## マインド様へのアクセス方法

### 1. 思考ちゃんを静かにする

- 深呼吸
- 瞑想
- 「考えずに感じてください」
- 身体感覚に意識を向ける

### 2. 安全な場を作る

- 「ここでは本音を言っても大丈夫」
- 「評価しません」
- 「秘密は守ります」

### 3. 落書きカウンセリング

- 「思いついたまま」
- 「誰も見ません」
- 思考ちゃんをバイパス

### 4. なりたい私を聞く

- アラジンの魔法のランプ
- 条件なしで本当の願いを
- マインド様の声が出やすい

### 5. 身体感覚を聞く

- 「体のどこで感じますか？」
- 「その感覚は何と言っていますか？」
- 体はマインド様に近い

- 「私はこれを信じていた」

## カウンセリングでのマインド様

### マインド様を引き出すフレーズ

- 「本当はどう感じていますか？」
- 「心の奥では何と言っていますか？」
- 「もし何の制限もなかったら？」
- 「体は何と言っていますか？」

### マインド様が出てきた時

- 「それがマインド様の声ですね」
- 「本当の願いが出てきましたね」
- そのまま受け止める
- 深掘りを続ける

### 答えは100%クライアントが知っている

- カウンセラーは答えを出さない
- マインド様が答えを知っている
- カウンセラーは引き出す手伝いをするだけ

## マインド様と幸せ

### 根底にある願い

**全ての人間は例外なく幸せになりたいと思っている**

- マインド様は幸せを望んでいる
- だからカウンセリングに来る
- だからインターネットで検索する
- だから解決したいと思う

### 迷子になったらマインド様に戻る

- 「なりたい私を見てください」
- 「本当は何を望んでいますか？」
- マインド様の願いに立ち返る

## よくある誤解・注意点

- ❌ マインド様はスピリチュアルな概念
- ⭕ マインド様は「本音」「本当の願い」という意味。オカルトではない

- ❌ マインド様は常に正しい
- ⭕ マインド様もガムテープで歪んでいることがある。ガムテープを剥がすことで真の声が聞こえる

- ❌ 思考ちゃんは敵
- ⭕ 思考ちゃんも守ってくれている。バランスが大事

## まとめ

マインド様は本当の自分、本音、心の声を指す。思考ちゃんやペルソナの奥に隠れているが、安全な場を作り、落書きカウンセリングやなりたい私を通じてアクセスできる。答えは100%クライアントのマインド様が知っており、カウンセラーはそれを引き出す手伝いをする。全てのマインド様は幸せを望んでいる。

**全員がマインド様本人である：**
- 全員アーキタイプを持っている
- 全員マインド様を持っている
- いつでも今この瞬間からマインド様になれる
- マインド様本人なのに、一生懸命思考ちゃんを選んで遊んでいる

**仁さんがどんな人でもカウンセリングできる理由：**
全員をマインド様と見ているから。今は一生懸命苦しんでいるかもしれないが、必ずマインド様を選ぶ日が来ると分かっている。

## マインド様が出てきた体験談（受講生の例）

### 殴り書き中に自然に出てきた声

**状況：**
- ガムテープに取り組んでいる時
- ノートに殴り書きで書いていた
- 瞑想状態ではない

**出てきた声：**
- 「私は価値があるよ」
- 「支配されなくてもいいよ」
- 「自由だよ」
- 勝手に、自然にポジティブな考えが出てきた

**特徴：**
- 何の疑いもない
- 根底から100%「これだ」という声
- 気分がいい
- 感じたことのない感覚
- 自然に「私は存在していい」と思った（初めての経験）
- その文章がお守りのようになっている

### マインド様が出てきた＝意味付けを変えられている

**重要な気づき：**

マインド様が出てきたということは：
1. その瞬間、ネガティブな意味付けをしていないということ
2. 人生も住んでいる場所も人間関係も何も変わっていないのに
3. 心の中だけは気分がいい
4. つまり意味付けを変えられているということ

**テープ式での意味：**
- 意味付けを変えることができている
- ＝ガムテープは剥がれている
- まだ30個40個あると思っても、剥がれていると信じてください

### 思考ちゃんのテスト

マインド様が出てきた後、思考ちゃんがテストしてくる：

**テストの形：**
- 「本当に大丈夫かい」
- 「ガムテープ剥がしていいのかい」
- まるで現実として試されるようなことが起きる
- 習慣・癖のように「やっぱり私ダメだ」となりがち

**対応方法（勝負ではなく受け入れる）：**

```
思考ちゃんが来た時の対応：

❌ 勝負する、邪魔する
⭕ 「思考ちゃん来たのね」「また来たのね」「あなたまだいたのね」
  ぐらいの余裕で
  横目で見ながら「よしよしよし」「もうわかってるよ」
  「けどね、私もマインド様出せちゃうのよ」
  という余裕

一緒に抱きしめながら：
「大丈夫だから」
「もうあんたのおかげで私今日まで生きてこれたし」
「で、私マインド様に繋がっちゃったから」
「もうテストしてくるのもいいけれど」
「そばで大人しくしといてね」
```

**ネガティブな声が聞こえてきたら：**
- 嫌がらずに
- 「来た来た来たーっ」と喜ぶ

### かすかな声でもゆるぎない

**マインド様の声の特徴：**
- 最初はかすかな声かもしれない
- しかし、どんなに重いガムテープに取り組んでも消えない
- 絶対負けない
- ゆるぎない
- どんなネガティブが来たってなんてことない
