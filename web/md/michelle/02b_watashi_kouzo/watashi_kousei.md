# 私を構成する3つの要素

## 定義

私（自分）を構成する要素は3つある。トリシアから教わった概念。

```
【私を構成する3つの要素】

1. ボディ（Body） - 体
2. マインド（Mind） - 心・意識エリア
3. Y（U） - 言葉では表せないもの
```

## 背景・目的

### なぜこの理解が重要か

- インナーチャイルドの概念を理解するために必要
- 「本当の私」を理解するため
- 脳科学者などが言う「全部脳が感じている」という説への反論

### 脳科学者への反論

テレビや様々な人たち（脳科学者、メンタリストDaiGoなど）が「全部脳が感じている」「心も脳の中にある」と断言しているが、**仁さんはそうでないと断言する**。

トリシアから教わり、自分でも間違いないと感じている。

## 1. ボディ（Body）

### 定義

物理的な体のこと。

### 重要なポイント

**この体は「私」ではなく「私の一部」**

- 仁さんはこの体ではない
- 体も仁さんの一部ではある
- しかし仁さんという全部を構成するには体だけではない
- つまり脳みそだけでもない

### 体がなくなっても

体が病気や寿命でなくなっても：
- 私がなくなるわけではない
- 私は永遠に存在し続ける

## 2. マインド（Mind）

### 定義

心、顕在意識、潜在意識などを全てひっくるめたエリア。

### マインドエリアの構造

マインドエリアの中に2つのエリアがある：

```
【マインドエリア全体】
┌────────────────────────────────────┐
│                                    │
│    マインド様エリア（巨大）          │
│    - 感覚のみ                      │
│    - 「今」だけ                    │
│    - アーキタイプ                  │
│                                    │
│         ・← 思考ちゃんエリア        │
│           （鉛筆の芯ぐらい小さい）   │
│           - 言葉を使う              │
│           - 過去と未来              │
│           - ネガティブ担当          │
│                                    │
└────────────────────────────────────┘
```

### サイズの比較

- マインド様エリア：超超超超巨大
- 思考ちゃんエリア：鉛筆の芯くらい小さい
- 比率は50対50ではない。何十万対1くらい

### 思考ちゃんエリアの役割

| 担当 | 説明 |
|------|------|
| 言葉を使う | 言語による思考 |
| 意味付け | 出来事に意味をつける |
| ネガティブ | ネガティブな感情を担当 |
| 考える | 「電話しようかな」「右に曲がろうかな」など |
| 過去と未来 | 過去と未来を作り出している |

※ネガティブ担当だからといって嫌いになることではない。私のために頑張ってくれている大事なエリア。

### マインド様エリアの役割

| 担当 | 説明 |
|------|------|
| 今だけ | 今しかない。過去も未来もない |
| 感覚 | 言葉がない。「ほわーん」「ボーン」「うーん」という感覚のみ |
| アーキタイプ | 私たちの原型。強さ、愛、感謝など。本当の私の源 |

### なぜ「様」と「ちゃん」

- **マインド様**：敬意を表すくらいとてもとても素晴らしい
- **思考ちゃん**：敬意を表すが、どちらかというと可愛らしい存在

### 心の目（フォーカス）

心に目があるとしたら：

- 思考ちゃんエリアにフォーカス → ネガティブばかり感じる
- マインド様エリアにフォーカス → 思考ちゃんが黙り、本来の私になれる

**あなたは今日どちらを見ていますか？**

### スクリーンの例え（トリシアの教え）

映画のスクリーンで例える：

- スクリーンの中の俳優 → 思考ちゃんに入り込んでいる状態
- スクリーンそのもの → マインド様（本当の私）

「あなたは一生懸命スクリーンの中の俳優さん本人だと思い込んで苦しんでいるけれど、**あなたはスクリーンそのもの**なんだよ。スクリーンの中に映し出す映画はあなたが選べるんだよ。」

## 3. YOU

### 定義

言葉では言い表せないもの。言葉で表現できない何か。トリシアは「言葉では説明できない」と言っていた。

### 呼び方

様々な呼び方がある（全て同じ意味）：

| 呼び方 | 出典 |
|--------|------|
| YOU | トリシア |
| 源（げん） | 日本語訳 |
| ソース（Source） | ザ・シークレット |
| ボルテックス | エイブラハム |
| 未知なる存在 | 「それはある」（ヘルメス・J著） |
| ワンネス | スピリチュアル一般 |

### YOUの正体＝アーキタイプ

YOUとは何かというと、**アーキタイプ（原型）そのもの**。

私たちは「愛を持っている人」ではなく「**愛そのもの**」。「豊かさを持っている人」ではなく「**豊かさそのもの**」。

**アーキタイプの例：**
愛、豊かさ、美しさ、感謝、全体性、永遠、強さ、母性、父性、バランス、夕価値、光、誠意、調和、1つ、自由、平等、無、信頼、勇気、許し　など

### バシャールの4つの法則

1. **あなたは存在している** - 私は絶対に消えない。命が尽きても消えない
2. **1つは全てであり、全ては1つである** - 無限に分かれた個別は、全て1つである
3. **あなたは自分が出したものを受け取る** - 引き寄せの法則。鏡の世界
4. **全ては常に変化し続ける** - 時間があるように体験させていただける

### 永遠の存在

体がなくなっても、YOUは永遠に存在し続ける。

- 第1の法則：あなたは存在している
- 消えたいと思っても消せない（法則だから）
- 自ら命を絶っても消えない
- 私は永遠に私

### この世界の目的

YOUは「自分とは何者なのか」を知るために、この世界を作った。

- 1から無限の個別の世界を作った
- それぞれがそれぞれの世界を同時に味わっている
- 真逆を体験することで、本来の自分（アーキタイプ）に気づく

### アーキタイプと真逆

アーキタイプには全て真逆が存在する。真逆を体験しないと、本来の自分に気づけない。

| アーキタイプ | 真逆 |
|-------------|------|
| 愛 | 愛がない |
| 豊かさ | 貧しさ |
| 勇気 | 恐怖 |
| 価値 | 無価値 |
| 全体性 | 孤独 |
| 永遠 | 時間・期限 |
| 許し | 怒り |

### スピリチュアルについて

- スピリチュアルと言われればそれまで
- しかしトリシアから教わっており、間違いないと感じている
- テープ式心理学はスピリチュアル要素を排除しても心理学として論理ができている
- しかし、カウンセラー側はこの部分を理解していないと、命に関わるレベルの人に対応できない
- 心理学を語るとき、死を避けて通れない。そのとき、スピリチュアルも避けて通れない

## 全員がマインド様

### 本当の私

- 全員アーキタイプを持っている
- 全員マインド様を持っている
- **いつでも今この瞬間からマインド様になれる**

### 選んでいるだけ

思考ちゃんにフォーカスしている時は：
- 「もうダメだ」「怖い」「私は価値がない」「寂しい」
- ずっとネガティブで遊んでいる

しかし本来の私は：
- 永遠にマインド様
- 巨大なマインドエリアがある
- ボディがなくなっても私はずっと存在する
- 最強

**マインド様本人なのに、一生懸命思考ちゃんを選んで遊んでいる。**

### 仁さんがどんな人でもカウンセリングできる理由

どのようなネガティブを感じている人でも、多重人格の方でも対応できる理由：

**全員をマインド様と見ているから**

- 今は一生懸命苦しんでいるかもしれない
- でも必ずマインド様を持っている
- 絶対にマインド様を選ぶ日が来ると分かっている

### 俯瞰できるようになる

心を見る訓練を積むと：
- マインドエリア側から自分の心の中を見れるようになる
- 「今、思考ちゃんを選ぼうとしているな」
- 「今、ネガティブに意味付けしようとしているな」
- これが丸見えになる

思考ちゃんエリアに入っていたら：
- それどころではない
- 「今苦しいんです、助けてください」となる

## ネガティブを選ばなくなるには

### 飽きること

ネガティブを選ぶのをやめるには：
- ネガティブに**飽きる**こと
- やり足りないからずっと選び続ける
- 十分にやって認めてもらったらやめられる

子供がわがままで抱っこをせがるのと同じ。十分に愛情を受け取って認めてもらったら「もうやめた」となる。

### 究極の選択

もう1つの道：
- 「もう限界だ」「もう飽きた」「このままだったら死んでしまう・壊れてしまう」
- この究極の選択の時に、思考ちゃんを選ぶのをやめざるを得なくなる
- マインドエリアをグッと選ぶ

### 本人が選ぶ

心理カウンセラーは心の中を操作できない。本人が選ぶ：
- 思考ちゃんを選ぶのか
- マインド様を選ぶのか

ネガティブが好きな方はずっとネガティブを選び続ける。これは仁さんにはどうすることもできない。

## よくある誤解・注意点

- ❌ 全ては脳が感じている
- ⭕ 私はボディ・マインド・Yの3つで構成されている。脳だけではない

- ❌ 体がなくなったら私もなくなる
- ⭕ 体は私の一部。Yは永遠に存在し続ける

- ❌ ネガティブを感じるのは仕方ない
- ⭕ 思考ちゃんを選んでいるだけ。いつでもマインド様になれる

- ❌ 苦しみから逃れられない
- ⭕ スクリーンの中の俳優を演じているだけ。私はスクリーンそのものだから、いつでも映画を変えられる

## まとめ

私を構成する要素は3つ：ボディ（体）、マインド（心・意識エリア）、Y（言葉では表せない源）。マインドエリアの中には巨大なマインド様エリアと、鉛筆の芯ほど小さい思考ちゃんエリアがある。全員がマインド様本人であり、いつでもマインド様になれるのに、一生懸命思考ちゃんを選んで苦しんで遊んでいる。ネガティブに飽きたらマインド様を選べる。
