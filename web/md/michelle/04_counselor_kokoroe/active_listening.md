# アクティブリスニング

## 定義

アクティブリスニングとは、相手の話を積極的に聴き、理解し、受け止めるコミュニケーション技法。単に「聞く（hear）」のではなく、「聴く（listen）」こと。

## 背景・目的

### なぜアクティブリスニングが重要か

- クライアントが「聴いてもらえた」と感じる
- 安心感・信頼感が生まれる
- 本音が出やすくなる
- ガムテープが見つかりやすくなる

### 聞く vs 聴く

| 聞く（hear） | 聴く（listen） |
|-------------|---------------|
| 音が耳に入る | 意味を理解しようとする |
| 受動的 | 能動的 |
| ながら聞き可能 | 集中が必要 |
| 反応しない | 反応を返す |

## 中核ポイント

### アクティブリスニングの要素

| 要素 | 説明 |
|------|------|
| 注意を向ける | 相手に集中する |
| 遮らない | 最後まで話を聞く |
| 映し返す | 言葉を繰り返す |
| 確認する | 理解を確認する |
| 共感を示す | 感情を受け止める |

### 具体的なテクニック

**1. 相槌（あいづち）**

- 「はい」「ええ」「うんうん」
- 「なるほど」「そうなんですね」
- 適度な頻度で打つ

**2. 繰り返し（オウム返し）**

```
クライアント：「上司に怒られて落ち込んでいます」
カウンセラー：「上司に怒られて落ち込んでいるんですね」
```

**3. 言い換え（パラフレーズ）**

```
クライアント：「何をやってもうまくいかないんです」
カウンセラー：「何をしても思うような結果が出ないと感じているんですね」
```

**4. 感情の反映**

```
クライアント：「もう全部嫌になりました」
カウンセラー：「とても辛くて、投げ出したい気持ちなんですね」
```

**5. 要約**

```
カウンセラー：「ここまでのお話をまとめると、
             仕事のプレッシャーがあって、
             家族にも相談できず、
             一人で抱え込んでいる状態なんですね」
```

**6. 開かれた質問**

```
❌ 閉じた質問：「辛いですか？」（はい/いいえで終わる）
⭕ 開かれた質問：「どんな気持ちですか？」（自由に答えられる）
```

## 非言語コミュニケーション

### オンラインの場合

- カメラを見る（目を合わせる）
- うなずく
- 表情で反応する
- 姿勢を前のめりに

### 対面の場合

- 適度なアイコンタクト
- 体を相手に向ける
- 腕を組まない
- リラックスした姿勢

## やってはいけないこと

### 1. 遮る

```
クライアント：「あのですね、私は—」
❌ カウンセラー：「それはこういうことですよね」
```

### 2. 先回りする

```
クライアント：「親との関係が...」
❌ カウンセラー：「虐待されたんですか？」
```

### 3. 評価・判断する

```
クライアント：「会社を辞めようと思っています」
❌ カウンセラー：「それは良くないですね」
```

### 4. アドバイスを押し付ける

```
クライアント：「どうしたらいいかわからなくて」
❌ カウンセラー：「こうすべきですよ」
```

### 5. 話を奪う

```
クライアント：「最近眠れなくて」
❌ カウンセラー：「私もそうなんですよ。私の場合は...」
```

## 沈黙の扱い方

### 沈黙は悪いものではない

- クライアントが考えている時間
- 感情を処理している時間
- 言葉を探している時間

### 沈黙への対応

- 焦らない
- 待つ
- 沈黙を埋めようとしない
- 必要なら「ゆっくりで大丈夫ですよ」と伝える

## よくある誤解・注意点

- ❌ アクティブリスニングは相槌を打つこと
- ⭕ 相槌は要素の一つ。理解し、受け止め、返すことが本質

- ❌ たくさん質問すればいい
- ⭕ 質問しすぎると尋問になる。バランスが大事

- ❌ 沈黙は気まずいから埋めるべき
- ⭕ 沈黙はクライアントの大切な時間。待つ

## まとめ

アクティブリスニングは、相手の話を積極的に聴き、理解し、受け止める技法。相槌、繰り返し、言い換え、感情の反映、要約、開かれた質問などのテクニックを使う。遮らない、評価しない、アドバイスを押し付けないことが重要。沈黙も大切な時間として扱う。
