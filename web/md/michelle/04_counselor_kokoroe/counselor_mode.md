# カウンセラーモードへの切り替え

## 定義

カウンセリング中は「カウンセラーモード」に切り替える必要がある。プライベートモード（友達モード、恋人モード、家族モード）とは異なる心の状態でカウンセリングに臨むこと。

## 背景・目的

### モードの違い

人は状況によってモードが異なる：
- 友達には何でも甘えられるが親には難しい
- 恋人には出せるが友達には気を使う
- 家族とは遠慮なく話せる

→ これらはそれぞれ「モード」が違う（人格の話ではない）

### カウンセラーモードとは

心理カウンセラーとして対価を受け取り、お仕事として活動する時のモード。

**プライベートモード（友達モード）との違い：**

| 友達モード | カウンセラーモード |
|-----------|------------------|
| 「わかる〜そういう感じだよね〜」 | 一定の距離感・境界線 |
| 仲良くなってOK | 仲良くなりすぎない |
| 距離感が近い | 距離感を保つ |

## 中核ポイント

### カウンセラーモードの3つの要素

**1. 心がセンター**

- 自分のネガティブをコントロールできる状態
- 不安定ではない
- センターにいる

**2. ノージャッジ**

- 良い悪い、正しい間違いをジャッジしない
- 常にノージャッジの状態
- 個人的な意見を押し付けない

**3. ネガティブが映らない**

- 相談者のネガティブな話を聞いても自分が苦しくならない
- その日1日ドヨーンとしない
- 相談者の問題は相談者のもの

### ネガティブが映らないとは

**映る状態：**
- 残酷なニュースを見てその日ずっとしんどい
- 相談者の話を聞いて自分も苦しくなる
- 頭の中でその人の話がループする

**映らない状態：**
- ニュースを見ても「安らかに眠ってください」とサッと切り替え
- 相談者の話を聞いても消耗しない
- 17秒以内にコントロールして元に戻る

### 自分もネガティブな状態だとカウンセリングにならない

**NGパターン：**

```
相談者：「夫にDVされて辛いんです」

カウンセラー：「そうなんですよ！私も夫に殺されかけて…
             本当に許せないですよね！」
```

→ これはカウンセリングではなく**お友達の会話**

## カウンセラーモードへの切り替え方

### 切り替えのタイミング

- カウンセリング開始時に切り替える
- 対価をいただく場合は必ず
- お仕事として活動する時は必ず

### 具体的な切り替え

1. 深呼吸する
2. 心をセンターに戻す
3. 「今からカウンセリング」と意識する
4. ノージャッジを意識する
5. 鏡になることを意識する

### 切り替えられないとどうなるか

**カウンセラーが不安定な場合：**

| カウンセラーの状態 | 相談者への影響 |
|-----------------|--------------|
| 今日は優しい | 混乱する |
| 明日はめっちゃ怒る | 「どっちが正しいの？」 |
| 親密になったと思ったら3日返事なし | 「嫌われた？」 |

→ 相談者は自分がどうしたらいいかわからなくなる

### 鏡がぶれるとどうなるか

- 相談者は「私ってこうだったんですね」と気づけない
- 「私が間違っている」「私が正しい」と混乱する
- 結局カウンセリングにならない
- 最悪の場合、傷つけて終わる

## かわいそう・助けなければはNG

### やってはいけないこと

以下の感情を持った瞬間、カウンセラーモードではなくなる：

- かわいそう
- 助けなければ
- 救わなければ
- 力になってあげなければ

### なぜNGか

- 依存関係の始まり
- 相談者が気になって気になって仕方なくなる
- 「〇〇さん大丈夫ですか」と親密になっていく
- これはカウンセリングではない

### 投影に注意

- 相談者を「かわいそう」と感じる
- 母親と投影してしまう
- 自分と投影してしまう

→ ノージャッジ、センター、鏡になるだけのはずが、個人的な感情が入ってしまう

## LINEグループでネガティブが映らない仁さんの例

### 1期生のLINEグループ

- 連日様々なネガティブの吐き出しがある
- 人生に対する怒りの文章
- 苦しみの表現

### 仁さんの状態

- **映らない**
- しんどくなっていない
- 心をコントロールできている状態だから

## よくある誤解・注意点

- ❌ カウンセラーモードは冷たい
- ⭕ 適切な距離感を保つこと。冷たいのではなくプロフェッショナル

- ❌ プライベートでも常にカウンセラーモード
- ⭕ モードは切り替えるもの。プライベートは友達モード、恋人モードでOK

- ❌ 共感するなら一緒に苦しむべき
- ⭕ 共感はするが、同化はしない。自分は自分、相談者は相談者

## まとめ

カウンセリング中は必ずカウンセラーモードに切り替える。カウンセラーモードとは「心がセンター」「ノージャッジ」「ネガティブが映らない」状態。プライベートモードとは異なり、一定の距離感・境界線を保つ。「かわいそう」「助けなければ」という感情はNG。相談者のネガティブに巻き込まれず、常に鏡として同じ場所に立ち続けることが大切。
