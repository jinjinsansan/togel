# 支える側について

## 定義

カウンセリングの依頼が、苦しんでいる本人からではなく、その人を支えている側（家族、恋人、友人など）から来るケース。このような状況での対応方法と、隠れた本当のクライアントについて。

## 通常のケースと稀なケース

### 9割9分は本人から

- カウンセラーを訪ねてくる、連絡してくる方はほとんどが本人
- 本人が苦しんでいるから連絡してくる

### 稀に支える側から来る

「私の付き合っている彼女がとても苦しんでいて、この人をカウンセリングしてもらえませんか」

「私の娘がすごく今精神病院とかに行っていて苦しんでいるので、この娘を何とかしてもらえませんか、カウンセリングしてもらえませんか」とお母さんが連絡してくる

## 本人が向き合う気がなければ何もできない

### カウンセリングの大前提

心の仕組みというのは：
- **本人が向き合うぞと決断しない限り**
- 何を話しても
- どれだけ心の仕組みを話しても
- 何をしても
- **何ともなりません**

### テープ式心理学のカウンセリング

- ほとんどが質問
- 質問に答えてもらいながら進める
- 本人が向き合う気がなければ質問に答えない
- ペルソナで終わるか、怒りを出してくるか

### 本人は頼んでいない

```
状況：
彼氏が勝手に頼んでいる
母親が勝手に頼んでいる
→ 本人は頼んでいない

カウンセラー：
「あ、すみません。お電話変わりました。涙サポート協会の仁と
 言いますけれど、カウンセリングさせていただきたいんですけど」

本人：
「は？」で終わる
```

### 無理やり持っていった場合

- 彼氏が「いいからカウンセリングやってみろよ」と押す
- 本人は「とりあえず」「今最近寝れないです」とか言う
- しかし勉強するつもりはさらさらない
- ただこの場を過ごすため
- **カウンセリングにならない**

## 支える側にカウンセリングが必要なケース

### 隠れた真のクライアント

支える側から連絡が来た時に気づくべきこと：
- 支える側本人が**実は苦しんでいる**ことが多い
- 連絡してきた人こそカウンセリングすべき人

### カウンセラー側の心構え

```
相談を聞きながらの心の中：
「なるほどなるほど」と話を合わせながらちゃんと聞いてあげながら
「この本人をカウンセリングするっていう方向に促していこう」
と感じ取る
```

### 直接的に言わない

❌ 直接的：
「じゃああなたをカウンセリングしましょうか」とダイレクトに言う

⭕ 促していく：
「なるほどなるほど、まあその彼女さんのことはちょっと、まあわかりました。
 で、その彼女さんと今お付き合いされている中で、今〇〇さんも苦しんでるんですよね。
 よかったらそっち側のお話も僕でよければお聞きしますけれど」

### 結果

- 「じゃあちょっとお願いします」となれば
- 実は本人が苦しんでいたということ
- そちらのカウンセリングに移行

## 具体的なケース（仁さんの経験）

### ケース1：「娘がおかしい」と言ってくる母親

**状況：**
- 本人は「娘がおかしいからカウンセリングしてください」
- 「私は間違っていません」というスタンス
- 実際は本人（母親）が向き合いたくない
- 娘が悪いと思い込んでいる状態

**アプローチ：**
```
「なるほどなるほど、娘さんが大変なんですね」と言いながら
まずはその「どういう状態なのか聞かせてください」と言って
そのお母さん側をカウンセリングしていった
```

**結果：**
- 1回で終わってしまった
- 本人は向き合いたくなかった
- カウンセリング成立とまではいかなかった

### ケース2：LINEだけのつながり

**状況：**
- 支える側のふりをしていた
- 誰かを悪者にしておきたい状態

**結果：**
- 時間はかかったが、結局なかなか向き合わなかった
- 支える側の顔をしているが、本当は自分が苦しい

### このパターンの特徴

支える側から連絡が来た時に多い状態：
- 支える側の顔をしているが、本当は自分が苦しい
- 「おかしいのはこの人であって、僕ではない・私ではない・母ではない」という思い込み
- 誰かを悪者にしておきたい

**重要な気づき：**
いくら娘さん側・彼女さん側が不安定であっても、支える側がネガティブに意味付けをしない方であったら、何も苦しまないはず。わざわざカウンセラーを探して「うちの娘をカウンセリングしてください」と言わないはず。

## カウンセラーの対応まとめ

### やってはいけないこと

1. **向き合うつもりがない方を無理やりカウンセリングしない**
   - 無理やり持っていっても必ずペルソナで終わる

2. **料金だけ取って中身のないカウンセリングをしない**
   - テープ式心理学では本人が向き合いたいと決断している方のみ契約
   - 向き合っている方はきっちり理論的に解放できる

### やるべきこと

1. **支える側の話をよく聞く**
   - 相談内容を聞きながら本当のクライアントを見極める

2. **支える側本人の苦しみに気づく**
   - 「あなたも苦しんでいるのでは？」と促す

3. **本人が向き合うまで待つ**
   - 決断は本人しかできない
   - カウンセラーには何もできない（鏡になるだけ）

## よくある誤解・注意点

- ❌ 家族が頼んだらカウンセリングできる
- ⭕ 本人が向き合う気がなければカウンセリングにならない

- ❌ 支える側の相談は断る
- ⭕ 支える側こそがカウンセリング対象かもしれない

- ❌ 無理やりでも話せば何か変わる
- ⭕ 本人が決断しない限り何も変わらない

## まとめ

カウンセラーに連絡してくるのはほとんど本人だが、稀に支える側（家族、恋人など）から来ることがある。本人が向き合う気がなければカウンセリングは成立しない。しかし、支える側から連絡が来た時は、その人自身が苦しんでいるケースが多い。直接的に言わず、話を聞きながら「あなたも苦しんでいるのでは」と促していく。支える側こそが真のクライアントであることを見極める目を持つこと。
