# 依存関係への対応

## 定義

相談者がカウンセラーに依存しようとする際の適切な対応方法。依存関係を成立させないための心構えと技術。

## 背景・目的

### なぜ相談者は依存したがるのか

- 自分から離れないって確認ができる
- コントロールができる
- 見やすくなる（得体の知れない人じゃなくなる）
- 依存関係であることが楽になれるから

## 中核ポイント

### 相談者がやってくること

依存関係を作ろうとして、以下のような行動をとる：

- 攻撃的な言葉を使ってくる
- 命に関わるアクションの話をしてくる
- 感謝の言葉
- 傷つけてくる
- 魅力を出してくる

→ 何回ドアをノックしても依存になってくれなかったら「得体の知れない怪物」のように感じる

### 仁さんの対応

**「ずっと同じ距離」**

何年経ったとしても立ち位置は変わらない。なぜなら**鏡になるだけ**だから。

## 依存が成立する条件

### 依存関係が成立する時

両方がネガティブを持っていて、両方ともがネガティブを埋め合うから初めて依存が成立する（親子でも恋愛でも友達でも兄弟でも全て同じ）

```
相談者：ネガティブあり
カウンセラー：ネガティブあり
→ 依存関係が成立する
```

### 依存にならない条件

どちらか片方がネガティブで、どちらか片方が**ノージャッジ**だった場合、依存にならない

```
相談者：ネガティブあり
カウンセラー：ノージャッジ（センター）
→ 依存関係は成立しない
```

## テープ式心理カウンセラーの場合

### 何をされても依存にならない

相談者が：
- 怒りを出してくる
- 涙を流してくる
- 命に関わるアクションを見せてくる

何をしようとも、カウンセラー側の心がセンターで「今日も元気だね、大丈夫かね」とニコニコしている状態だった場合、**依存にならない**

### 具体的な対応

- 17秒でネガティブをコントロールしてしまう
- 相談者のネガティブが映らない
- だから相談者側がどんどん依存しようとしてきても大丈夫

### 相談者への言葉

「いいですよ、全然大丈夫ですよ、依存して大丈夫ですよ。こちらは依存になりませんので」

## 相談者が「依存しそう」と言ってきた場合

### 相談者の言葉

- 「カウンセラーや人に依存してしまいがちです」
- 「依存してしまいそうで、すいません」
- 「依存する癖があるんです」
- 「迷惑かけちゃうかも」

### 対応

**「大丈夫ですよ」「依存してて大丈夫ですよ」**と伝える

カウンセラー側が依存にならないので、相談者がどれだけ依存しようとしても問題ない。

### 具体的な言葉

```
相談者：「私って依存してしまいがちなんです。
       迷惑かけちゃうかもしれません」

カウンセラー：「大丈夫ですよ。依存は何も悪くないですよ。
             それはバランスを保つ行為ですから。
             ただ、私は対応できる時間が限られていますし、
             返信も対応できる時しかできませんが、
             依存というもの自体は何も悪くないですよ」
```

### 言わなくていいこと

- 「依存されても結構ですよ」（誤解を招く）
- 「こちらは依存しませんので」（言わなくていい）

### 伝えるべきこと

1. 依存は悪くない
2. バランスを保つ行為である
3. ただし対応には限界がある（境界線）
4. 返信できる時しか返信できない

### なぜ「大丈夫」と言えるか

依存関係が成立するには**両方がネガティブ**を持っている必要がある。

```
相談者：依存したい（ネガティブあり）
カウンセラー：センター（ネガティブなし）
→ 依存関係は成立しない
```

相談者が一生懸命もたれようとしても、カウンセラー側がセンターなら**もたれきることができない**。

## 鏡の役割

### 鏡はぶれない

- しっかり鏡になることで相談者が自分の状態を認識できる
- 「私って今こんな状態だったんだ」
- 「私って今こんなに小さかったんだ」
- 「私って今ガムテープが反応してるんですね」

### 距離を近づけるとどうなるか

距離を近づけたり依存し合ったりしたら、もう分からない（鏡になっていないから）

## 心がセンターである状態

### カウンセラーに必要な状態

1. 自分のネガティブをコントロールできる状態
2. 相談者の話をどれを聞いても**ノージャッジ**（良い悪い、正しい正しくないがない）

### ネガティブが映るとは

自分の暮らしは別に困っていないのに：
- 相談者の話を聞いたことでその日1日がドヨーンとしてしまう
- その人の話が頭の中で何回もループする
- 怖い話を聞いたら自分の生活でも怖くなってきた

→ 元々自分の人生に起きているネガティブじゃないのに、他人のネガティブの話を聞くことでその通りになっていく状態

### テープ式心理カウンセラーの場合

- ネガティブが映ったとしてもすぐ心の中を見る
- 「ん、これは映ったな。違うな、これはあの人の話か」と意味付けを変えられる
- **一切ネガティブが映らない**

## 「もたれ中」という呼び方

### 依存の新しい呼び方

テープ式心理学では、依存のことを**「もたれ中」**と呼ぶことがある。

### 理由

- 「依存」という言葉がメディアや精神科医によって悪いイメージで使われすぎている
- ギャンブル依存症など、「依存＝悪」というイメージが植え付けられている
- しかし**依存は素敵なこと、最高なこと**
- 「持たれてるだけだよ」という軽いニュアンス

### 使い方

「依存してしまいそう」ではなく「もたれてるだけ」と捉える。

## 身体的依存と精神的依存の見分け方

### 依存の2種類

| 種類 | 説明 | 担当 |
|------|------|------|
| 身体的依存（肉体的依存） | 体がそう感じるから依存している | 専門外（病院など） |
| 精神的依存 | ガムテープがあるがゆえに依存している | テープ式心理カウンセラー |

### 身体的依存の例

- タバコ：肺がぐーっと苦しくなる、吸いたくなる
- 化粧水：塗らないと顔がヒリヒリする

→ 体がそう感じるから依存している。ガムテープ（恐怖・悲しみ・怒り・寂しさ・無価値）は発生しない。

### 精神的依存の例

ガムテープがあるがゆえに依存アクションが発生している。

**人でないもの：**
ギャンブル、薬物、お酒、リストカット、ゲーム、過食・拒食、YouTube、スマホ、アイドル

**人：**
恋愛依存、共依存、反依存

### 見分ける魔法の呪文

**「あなたはそれがなくなったらどう感じますか？」**

- ソワソワしますか？
- ガムテープが反応しますか？

| 依存の種類 | なくなった時の反応 |
|-----------|-------------------|
| 身体的依存 | 体が吸いたい・使いたいと感じるだけ。五大ネガティブは発生しない |
| 精神的依存 | すぐガムテープが反応する。恐怖・悲しみ・怒り・寂しさ・無価値が発生する |

### テープ式心理カウンセラーの担当

**精神的依存のみ担当する。**

身体的依存は専門外であり、病院や皮膚科などに行く必要がある。

## ネガティブの症状の段階との関係

### 4ステップ理論（仁さんオリジナル）

依存は「ネガティブの症状の段階」における**ステップ2**に該当する。

| ステップ | 状態 |
|---------|------|
| 1 | 吐き出しレベル（愚痴で解放） |
| **2** | **依存対象に依存している状態（もたれ中）** |
| 3 | 無気力・うつ状態 |
| 4 | 命に関わるアクション |

### ステップ2の特徴

- ステップ1の吐き出しだけではネガティブが解放されない
- 依存対象に依存することでバランスを保っている
- **依存対象がいる限り、学校や仕事には行ける**

### 仁さんの例

**「仁さんは一生2ステップ」**

- 依存対象がなくならないようにずっと確保している
- 2ステップでも幸せに生きている
- 依存対象に感謝している

### ステップ間の移動

- 行ったり来たりする
- 2ステップから1ステップに移動することもできる
- 2ステップから3ステップに移動することもある（依存対象がなくなった場合など）

## よくある誤解・注意点

- ❌ 依存されないように冷たくする
- ⭕ 冷たくするのではなく、常に同じ距離を保つ

- ❌ 依存は絶対に避けるべき
- ⭕ カウンセラー側がセンターであれば、相談者が依存しようとしても依存関係は成立しない

## まとめ

相談者は依存関係を作ろうとするが、カウンセラーが心のセンターを保ち、ノージャッジでいれば依存関係は成立しない。「ずっと同じ距離」を保ち、鏡としての役割を果たすことが重要。相談者が「依存しそう」と言ってきても「大丈夫ですよ」と伝えられる。
