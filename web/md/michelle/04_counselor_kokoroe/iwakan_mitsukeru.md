# 違和感を見つける

## 定義

相談者の話を聞きながら「何か違うな」「本当はそうじゃないのでは」という違和感を見つけ、メモしておくこと。後からガムテープと繋げていく重要なテクニック。

## 背景・目的

### 話を聞くことがメインではない

テープ式心理カウンセラーは「話を聞く」ことがメインではない。

| 一般的なカウンセラー | テープ式心理カウンセラー |
|-------------------|----------------------|
| 話を聞くことがメイン | 話を聞きながらガムテープを見つける |
| 聞くだけで終わることも | 違和感を見つけることが仕事 |
| 楽だが効果が限定的 | ガムテープを剥がす手伝いまで |

### 話を聞きながらやること

1. ガムテープを見つける
2. **違和感を見つける**

## 中核ポイント

### 違和感とは

相談者が何気なく言う言葉の中に、本来の状態（マインド様の状態）とは違う部分が現れる。

**違和感の例：**

- 「私って、やっぱり価値がないですよ」
- 「人より後でいいんですよ」
- 「1番大事にされなくていいんです」
- 「1人の方が好きなんです」
- 「友達って何かわからないんです」
- 「自分のことを話す必要ってないと思うんですよ」

→ さらっと言うけど、**本来リラックスして満ち足りている状態なら出てこない言葉**

### 違和感が出るタイミング

- 話の流れの中で自然に出てくる
- 意識して言っているわけではない
- 本人は気づいていないことが多い

### 違和感の見分け方

**本来の状態（マインド様）から考える：**
- 本来は価値がある
- 本来は大事にされていい
- 本来は友達との繋がりがわかる

**違和感：**
- 上記と矛盾することを言っている
- ネガティブな自己認識が隠れている
- ペルソナを出している可能性

## 違和感への対応

### その場で拾わなくていい

違和感に気づいても、その場で深掘りしなくていい。

**やること：**
1. 違和感を感じる
2. メモする
3. **後で使う**

### 落書きカウンセリングと連動

```
話を聞きながらメモ：
「1人が好き」← 違和感
「友達って何かわからない」← 違和感
「価値がない」← 違和感
```

→ A4の白紙にどんどん書いていく

### 後から繋げる

**心理テストで出たガムテープと違和感を繋げる：**

```
心理テスト：「私はひとりぼっちである」（ガムテープ）
違和感メモ：「1人が好き」

カウンセラー：「ところで、さっき『1人が好き』って
             言ってたけど、それって何ですか」

相談者：「確かに1人が好きって思い込んでましたけど、
       実はそれ隠してるだけですかね」
```

→ **繋がった！**

## 整合性を見る

### 整合性のチェック

以前言っていたこと・表情と、今の言葉が合っているかを見る。

**整合性がない例：**
- 以前「人と繋がりたい」と言っていた
- 今「1人が好き」と言っている
- → **矛盾＝違和感**

### ペルソナの発見

**整合性がない ＝ ペルソナを出している可能性：**

- こう見せたい
- ごまかしている
- 隠している

→ ガムテープが隠れている

## 違和感を繋げる質問

### 直接聞く

```
カウンセラー：「ところで、さっき〇〇って言ってたけど、
             それって何ですか」
```

### 心理テストと繋げる

```
カウンセラー：「心理テストで『私はひとりぼっちである』って
             出てましたけど、さっき『1人が好き』って言ってた
             あれ、ひょっとして繋がってますか？」

相談者：「あ…確かに。1人が好きって言ってたけど、
       本当は寂しいのかもしれません」
```

### なりたい私と繋げる

```
カウンセラー：「なりたい私で『友達がほしい』って書いてましたけど、
             さっき『友達って何かわからない』って言ってましたよね。
             これって繋がってますか？」
```

## 違和感メモの俯瞰

### テーブルに並べる

- A4の白紙に違和感をどんどん書いていく
- 書いた紙をテーブルに並べる
- 心理テストの結果も並べる
- 俯瞰して見る

### 繋がりを見つける

```
紙1：心理テスト「私はひとりぼっちである」
紙2：違和感「1人が好き」「友達って何かわからない」
紙3：なりたい私「友達がほしい」

→ 全部繋がっている！
```

## よくある誤解・注意点

- ❌ 違和感を感じたらすぐ指摘する
- ⭕ メモしておいて後から繋げる。焦らない

- ❌ 違和感は問い詰めるネタ
- ⭕ 相談者自身が気づくための材料。問い詰めない

- ❌ 違和感は全部拾わないといけない
- ⭕ 重要なものだけ。心理テストのガムテープに繋がりそうなものを優先

## まとめ

話を聞きながら「違和感」を見つけることがテープ式心理カウンセラーの重要な仕事。相談者がさらっと言う言葉の中に、本来の状態とは違う部分が隠れている。違和感はその場で拾わず、メモしておいて後から心理テストのガムテープやなりたい私と繋げていく。整合性がない部分はペルソナやガムテープが隠れている可能性が高い。俯瞰して繋がりを見つけることで、相談者自身の気づきを促す。
