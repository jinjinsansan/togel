# 境界線を持つ

## 定義

境界線（バウンダリー）とは、カウンセラーとクライアントの間に引く心理的・物理的な線。自分を守り、健全な関係を維持するために必要。

## 背景・目的

### なぜ境界線が必要か

- カウンセラーが消耗しないため
- クライアントの自立を促すため
- 共依存を防ぐため
- プロフェッショナルな関係を維持するため

### 境界線がないとどうなるか

- カウンセラーが燃え尽きる
- クライアントが依存する
- 共依存関係になる
- どちらも不幸になる

## 中核ポイント

### 境界線の種類

| 種類 | 説明 | 例 |
|------|------|-----|
| 時間の境界線 | セッション時間を守る | 「今日は60分です」 |
| 感情の境界線 | 感情を持ち帰らない | 「クライアントの問題はクライアントのもの」 |
| 物理的境界線 | 適切な距離を保つ | オンラインでも距離感を意識 |
| 経済的境界線 | 料金を明確にする | 値引き交渉に応じない |

### 感情の境界線

**クライアントの感情 ≠ カウンセラーの感情**

- クライアントが悲しくても、カウンセラーは悲しみに飲まれない
- クライアントが怒っていても、カウンセラーは巻き込まれない
- 共感はするが、同化はしない

### 時間の境界線

- セッション時間は守る
- 延長は基本的にしない（有料なら可）
- 「時間なので今日はここまで」と言える

### セッション外の境界線

- セッション外の連絡は最小限に
- 24時間対応しない
- 緊急時の対応ルールを決めておく

## 境界線を越えてくるパターン

### 1. 時間を延長しようとする

```
クライアント：「もう少しだけ話を聞いてください」

❌ 境界線を越える：
「いいですよ、特別に」

⭕ 境界線を守る：
「今日は時間なのでここまでにしましょう。
 続きは次回お話しましょう」
```

### 2. セッション外で連絡してくる

```
クライアント：（深夜にLINE）「辛いです、話を聞いてください」

❌ 境界線を越える：
すぐに返信して話を聞く

⭕ 境界線を守る：
翌日の営業時間内に「辛かったですね。次回のセッションで
詳しくお話しましょう」と返信
```

### 3. 値引きを求めてくる

```
クライアント：「お金がないので安くしてもらえませんか」

❌ 境界線を越える：
「特別に半額にしますね」

⭕ 境界線を守る：
「お気持ちはわかりますが、料金は一律です。
 分割払いなどは相談できます」
```

### 4. 依存してくる

```
クライアント：「先生がいないと生きていけません」

❌ 境界線を越える：
「私がずっとそばにいますから」

⭕ 境界線を守る：
「そう思うくらい辛いんですね。
 でも答えはあなたの中にあります」
```

## 境界線と冷たさの違い

### 境界線 ≠ 冷たい

- 境界線を引くことは愛情
- 境界線がないと共倒れ
- 境界線があるから長く続けられる
- 境界線があるからクライアントも自立できる

### 伝え方の工夫

```
冷たく聞こえる：
「それは私の仕事ではありません」

温かく境界線を引く：
「そのお気持ちはとてもよくわかります。
 私にできることには限りがありますが、
 次回のセッションで一緒に考えましょう」
```

## カウンセラー自身のケア

### セッション後のリセット

- 深呼吸する
- 手を洗う（儀式として）
- 散歩する
- クライアントの感情を手放す

### 定期的なスーパービジョン

- 先輩カウンセラーに相談
- 自分のガムテープを剥がす
- 客観的な視点をもらう

### 休息を取る

- 連続してセッションを入れすぎない
- 休日を確保する
- 自分の時間を大切にする

## よくある誤解・注意点

- ❌ 境界線を引くと嫌われる
- ⭕ 境界線を引くことで信頼される。プロフェッショナルとして

- ❌ クライアントのためなら何でもすべき
- ⭕ カウンセラーが倒れたら誰も救えない

- ❌ 境界線は厳しくするもの
- ⭕ 境界線は自分とクライアント両方を守るもの

## カウンセラーとしての境界線の実践

### 仁さんの例

- どのようなネガティブを受けても、どのような苦しい話を聞いても、ネガティブが映ることがない
- 私は私、じさんはじさん、あなたはあなた
- どんなに残酷な話を聞いても、涙することはあるが、必ず境界線は保ったまま
- 境界線を保っていなければ、私すら保てなくなり、相手を支えることができない私になってしまう

### 境界線がある状態でカウンセリング

- 他者のネガティブが映らない
- クライアントの話を何時間聞いても消耗しない
- 相手を支えることができる自分でいられる
- 10年以上、誰かのネガティブが映ることがない

### 境界線のワーク（トリシアの教え）

クライアントに境界線を教える際に使えるワーク：

**やり方：**
1. クライアントが壁のところに立つ
2. 反対側の壁に何人か立ってもらう（または人形や10円玉などで代用）
3. プラカードを持ってもらう（お母さん役、恋人役、友達役、上司役、部下役など）
4. 順番に近寄ってきてもらう
5. 止まって欲しいところで「ストップ」と言う
6. 全員について繰り返す

**トリシアの言葉：**
「これが境界線なんですよ」

**ワークの意味：**
- 人と人との境界線・距離感は「私が決める」
- 私以外の全ての人間関係の距離感を、全部私が「ストップ」と言って決めることができる
- これは毎日変わってもいい
- 全部私のため、私を守るため、私らしくあるために、私がストップをして決めていい

### 簡易版ワーク

友達を集められない場合：
- 机の上で10円玉や50円玉を置く
- それを「誰々役」と設定
- グーっと近づけて「ストップ」

**重要：話を聞くだけでなく、必ずこのワークをやらせること**

## 自分をワークしなさい（トリシアの教え）

### トリシアが繰り返し強調したこと

1ヶ月間の中でトリシアが何度も強く言っていたこと：
- **自分をワークしなさい**
- **境界線を保ちなさい（バウンダリー）**

### 自分をワークするとは

自分の心の中で発生するネガティブを：
- **見ることができる状態**
- **コントロールできる状態**
にしておくこと

### なぜ自分のワークが必要か

カウンセリングでは十人十色の相談が来る：
- 様々な人生体験
- 様々なガムテープ
- 登場人物も出来事も様々

**もし自分のワークが終わっていなかったら：**

```
状況：
カウンセラー自身がDVを受けた経験があり、
そのトラウマがまだコントロールできない状態で
DVを受けている相談者がやってきた

ワークが終わっていない場合：
→ 自分のガムテープが反応してしまう
→ 「どうしてあなたは離婚しないんですか！今すぐ！」
→ 個人的なジャッジをしてしまう

本来のカウンセラーの仕事：
→ ただただ鏡になるだけ
→ 「離婚すべき」「すべきでない」はカウンセラーが言うことではない
→ 「どうしたいですか？」と本人の決断を尊重する
```

### 個人的なジャッジは不要

カウンセラーの仕事は：
- テープ式心理学を話す
- 個人的なジャッジは出さない
- どのクラスでも同じことを話す

個人的なこと（「会社の社長を務めた時はこうだった」「生きるってこうじゃないか」）は出てこない
→ 教わっていないから出さない
→ カウンセラーとしての役割をよく分かっているから

### 自分のワークが終わっていないとどうなるか

```
例：DVの経験がある自分、「もっと早く離婚しておけばよかった」という後悔が残っている

相談を受けた時：
「取り返しのつかないことになりますよ！」
「男ってね、こうなんですよ、こうなんですよ」

→ その時点でカウンセリングは終わっている
→ 後悔になっている
→ カウンセリングになっていない
→ ただの「私の愚痴を聞いてもらうネガティブ座談会」になっている
```

**これがわかっていないカウンセラーさんが多い**

### カウンセラーがしんどくなる問題

Yahoo知恵袋などで見かける質問：
「カウンセラーさんが相談を受けてしんどくならないんですか」

スーパーバイザーカウンセラーの存在：
- 心理カウンセラーがしんどくなった時にカウンセリングする人

**しかしテープ式心理学の視点では：**

カウンセラーがしんどくなる ＝ **ネガティブが映っている**

```
状況：
1人目の相談でしんどい
→ まだ残っている状態で2人目
→ 「こんにちは」「先生聞いてください。私実は親に虐待されていて…」
→ うわぁ…

これを続けると：
月曜日から始まって金曜日までしんどい
→ 土日休んでサザエさん見て「明日から始まる…」症候群
→ カウンセラーなのにカウンセリングすることがしんどい
```

**これは境界線が引けていないということ**

### 他のカウンセラー団体の利用規約の例

仁さんが見た某カウンセラー団体の利用規約：
- 「カウンセラーを批評・侮辱する発言をする場合はカウンセリング終了」
- 「カウンセラーに対するバッシング・攻撃はその場で終了」

**これを見た仁さんの感想：**
- この状態で苦しんでいる方は思っていることを全部話すだろうか
- カウンセリング終了されることが怖くて言葉を選ぶのでは
- ビクビクしながらカウンセリングするのでは

**なぜこうなるか：**
- きっと団体のトップが境界線を引けていない
- バッシングされると無価値を感じてしまう
- 意味づけを変えられない

→ テープ式心理学とは考え方が違う

### 仁さんの期生への配慮

1期生の期間をわざと半年と長くしている理由：
- 今は自分と向き合ってもらう時間
- 自分のワークを終わらせる時間
- これがとても大事

## 恋愛感情への対応（トリシアの教え）

### トリシアの明確な方針

女性カウンセラーは特に注意が必要。トリシアはこう教えた：

**「相談者が恋愛対象としてカウンセラーを見てきた場合は、その時点でカウンセリングをお断りする」**

### 具体的な対応

```
相談者：（恋愛感情を示す言葉やリクエストがあった場合）

カウンセラー：「今のような恋愛のような言葉、希望、リクエストになりますと、
             これ以上カウンセリングは続けられません。
             いかがされますか」
```

### 境界線を明確に提示する

- 1回目：明確に警告する
- それでも続く場合：お断りする
- トリシアも実際にお断りした

### なぜ断るか

- カウンセラーは鏡である
- 恋愛関係になったら鏡ではなくなる
- カウンセリングが成立しなくなる
- 両者にとって不健全

### 性の多様性への配慮

- 女性カウンセラーだけでなく男性カウンセラーも同様
- 同性間でも同様
- 性の多様性がある現代では全てのケースに当てはまる

## 距離の一貫性

### 一度でも近寄ったら戻せない

**重要な注意点：**

一度でも近寄ったかのようなカウンセリングをしてしまったら、2回目以降は冷たく感じられてしまう。

### 具体例

| 1回目 | 2回目以降 | 相談者の感覚 |
|------|----------|-------------|
| 4時間話を聞いた | 60分で終了 | 「冷たい」「嫌われた」 |
| 深夜でも電話対応 | 営業時間内のみ | 「見捨てられた」 |
| すぐ返信 | 翌日返信 | 「距離を置かれた」 |

### 元に戻せない理由

- 最初の対応が基準になる
- 基準より下がると「冷たい」と感じる
- 信頼関係が崩れる
- カウンセリングにならなくなる

### 正しい対応

- **最初から最後まで同じ距離感を保つ**
- 1回目の対応 ＝ 100回目の対応
- ぶれない
- 鏡はずっと同じ場所にある

### 仁さんの例

- 何年経っても立ち位置は変わらない
- 相談者が近寄ってきたら離れる
- 距離感が常に同じ
- 「境界線の鬼」と呼ばれた

## まとめ

境界線はカウンセラーとクライアント両方を守るもの。時間、感情、物理的距離、経済面で境界線を意識する。境界線を引くことは冷たいのではなく、愛情であり、プロフェッショナリズム。境界線があるからこそ長くカウンセリングを続けられ、クライアントの自立も促せる。境界線のワークを通じて「人との距離感は私が決めていい」ということをクライアントに体感させること。

恋愛感情が出てきた場合は、明確に警告し、続くならお断りする（トリシアの教え）。一度でも近寄ったカウンセリングをしたら元に戻せないので、最初から最後まで同じ距離感を保つこと。

トリシアは「自分をワークしなさい」「境界線を保ちなさい」と繰り返し強調した。自分のネガティブをコントロールできる状態にしておかないと、相談者に対して個人的なジャッジをしてしまい、カウンセリングではなくなってしまう。カウンセラーがしんどくなるのは境界線が引けていない証拠。必ず自分のワークを終わらせてからカウンセリング活動を始めること。
