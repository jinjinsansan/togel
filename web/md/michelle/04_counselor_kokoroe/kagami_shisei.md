# 鏡としての姿勢

## 定義

カウンセラーは「**鏡**」である。クライアントの言葉や感情をそのまま映し返す存在であり、答えを与える存在ではない。

## 背景・目的

### テープ式心理学の根本原則

**答えは100%クライアント本人が知っている**

- カウンセラーは何もできない
- カウンセラーは何もしていない
- カウンセラーは無力である
- 本人がそのように選んだだけ

### なぜ鏡なのか

- 鏡は映すだけ
- 鏡は判断しない
- 鏡は正しい/間違いを言わない
- 鏡はそのまま返すだけ

## 中核ポイント

### 鏡になるとは具体的に何か

**物理的な鏡の役割：**
- 髪型を直したり、今日の顔を見たり
- 自分の顔が今こんな顔なんだと気づく
- 「年取ったな」「今日は可愛いな」と気づく

**カウンセラーも同じ：**
- 何もアドバイスしない
- 鏡は「あなた今日髪型こっちの方がいいに決まってるやん」とか言わない
- カウンセリングは鏡になる

**本質：ネガティブを選んでいることを見せてあげる**
- 相談者が苦しんでいる＝本当は幸せになりたいという前提
- しかしネガティブを選ぼうとしていることがカウンセラー側には丸分かり
- 「ネガティブを選んでいるんだよ」ということを見せてあげる
- あなたは今ネガティブをこういうの感じていますよね
- 本当はこうしたいんでしょ、本当はこうしたかったんだよね
- ただただ見せてあげてヒントにしてあげる

### 鏡になる会話の具体例

```
相談者：「右に行くか左に行くか決められないです」

❌ アドバイス：
「右に行った方がいいですよ」
「絶対右にしてください」

⭕ 鏡になる：
「右に行きたいんでしょ？本当はどう？」
「わざと左行く？」
「本当はどう？知ってるでしょ？」

相談者：「私、元々右に行きたいって知ってました」
```

**ポイント：**
- 「やってください」「必ず右向いてください」とは言わない
- 「右向きたいんでしょ？本当はどう？」と鏡になる
- 本人が「私、元々右に行きたいって知ってました」と気づく

### 鏡の役割表

| 鏡がすること | 鏡がしないこと |
|-------------|---------------|
| そのまま映す | 判断する |
| 言葉を返す | アドバイスする |
| 感情を反映する | 評価する |
| 質問する | 答えを出す |
| 待つ | 急かす |
| ネガティブを選んでいることを見せる | 正解を押し付ける |

### 具体的な姿勢

**映し返しの例：**

```
クライアント：「上司に怒られて落ち込んでいます」

❌ 悪い例（アドバイス）：
「上司の言うことは気にしなくていいですよ」

⭕ 良い例（鏡）：
「上司に怒られて落ち込んでいるんですね」
```

**深掘りへの誘導：**

```
クライアント：「自分がダメだと思います」

❌ 悪い例（否定）：
「そんなことないですよ、あなたは素晴らしいです」

⭕ 良い例（鏡＋質問）：
「自分がダメだと思うんですね。その感覚をもう少し教えてください」
```

## カウンセラーは無力

### 無力であることの意味

- クライアントを変える力はない
- クライアントを救う力はない
- できるのは「映す」ことだけ
- 変わるのはクライアント自身

### 無力を受け入れる

- 「救わなければ」という思いは手放す
- 「何とかしてあげたい」は境界線を越えている
- 無力であることを認めると、むしろ力が発揮できる

## 鏡のメリット

### 1. クライアント自身が気づく

- 映し返されることで「あ、私こう言ってたんだ」
- 自分の言葉を聞き直すことで気づきが生まれる
- 他人に言われるより、自分で気づく方が深い

### 2. カウンセラーが消耗しない

- アドバイスを考えなくていい
- 正解を出そうとしなくていい
- エネルギーの消耗が少ない

### 3. 信頼関係が築ける

- 判断されないと感じると安心する
- 安心すると本音が出やすくなる
- 本音が出るとガムテープが見つかる

## よくある落とし穴

### 1. アドバイスしたくなる

- 「こうすればいいのに」と思っても言わない
- 言いたくなる気持ちは自分のエゴ
- 黙って映すだけ

### 2. 共感しすぎる

- 一緒に泣いてしまう
- 一緒に怒ってしまう
- 感情に巻き込まれると鏡が曇る

### 3. 助けたくなる

- 「かわいそう」と思う
- 「何とかしてあげたい」と思う
- これは境界線を越えている

## 鏡であり続けるために

### 自分のガムテープを剥がす

- カウンセラー自身もガムテープを持っている
- 自分のガムテープがあると鏡が歪む
- 定期的にスーパービジョンを受ける

### 境界線を意識する

- クライアントの問題はクライアントのもの
- 自分の問題と分ける
- 持ち帰らない

### 休息を取る

- 疲れると鏡が曇る
- セッション後は休む
- 自分を大切にする

## よくある誤解・注意点

- ❌ 鏡は冷たい対応
- ⭕ 鏡であっても温かさは伝わる。無表情という意味ではない

- ❌ 何も言わないのが鏡
- ⭕ 映し返しの言葉、質問は積極的に使う

- ❌ カウンセラーの意見を言ってはいけない
- ⭕ 聞かれたら答えてもいい。ただし押し付けない

## まとめ

カウンセラーは鏡である。クライアントの言葉や感情をそのまま映し返し、判断やアドバイスをしない。答えは100%クライアントが知っており、カウンセラーは無力。無力を受け入れることで、むしろクライアント自身の気づきを促すことができる。
