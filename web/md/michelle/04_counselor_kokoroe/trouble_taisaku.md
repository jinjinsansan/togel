# トラブル対策

## 定義

カウンセリング中に起こりうるトラブルとその対処法。境界線を守りながら、クライアントの安全と健全な関係を維持するための指針。

## よくあるトラブルと対応

### 1. 依存してくるクライアント

**症状：**
- 「先生がいないと生きていけない」
- 頻繁に連絡してくる
- セッション以外でも相談したがる
- カウンセラーに執着する

**対応：**

```
クライアント：「先生だけが頼りなんです」

カウンセラー：「そう思ってくださるのは嬉しいですが、
             答えはあなたの中にあります。
             私はそれを映す鏡でしかありません。
             一緒にあなたの力を見つけていきましょう」
```

**ポイント：**
- 境界線を明確にする
- 依存を強化しない言葉を選ぶ
- クライアントの自立を促す
- 必要なら他の専門家を紹介

### 2. 攻撃的なクライアント

**症状：**
- カウンセラーを責める
- 「あなたのせいで悪くなった」
- 怒りをぶつけてくる
- 批判的な態度

**対応：**

```
クライアント：「あなたのカウンセリングは効果がない！」

カウンセラー：「そう感じさせてしまったんですね。
             その怒りについてもう少し教えていただけますか。
             その奥にある感情は何でしょう」
```

**ポイント：**
- 防御的にならない
- 怒りの下にある感情を探る（ガードの怒り）
- 個人攻撃には巻き込まれない
- 危険を感じたらセッションを終了する権利がある

### 3. 自殺・自傷をほのめかすクライアント

**症状：**
- 「死にたい」「消えたい」
- リストカットなどの自傷行為
- 具体的な方法を語る
- 遺書を書いた

**対応：**

```
クライアント：「もう死んでしまいたいです」

カウンセラー：「そう思うくらい辛いんですね。
             その気持ちを話してくれてありがとうございます。
             具体的に何か計画はありますか？」
```

**ポイント：**
- 慌てない
- 具体性を確認する（計画があるか）
- 「死にたい」を否定しない
- 必要に応じて専門機関につなぐ
- 命の危険がある場合は緊急対応

**緊急連絡先を伝える：**
- いのちの電話
- よりそいホットライン
- 精神科救急

### 命に関わるアクションへの対応（回数別）

**第1回目：全力で止める**

初めて命に関わるアクション（未遂など）があった場合：
- **なりふり構わず止める**
- 境界線なしでOK
- 深夜でも、電話でも、何でも
- 本気で「ダメです」「生きててください」と伝える
- **ここだけは境界線をぶっ放していい**

**第2回目：様子を見ながら**

2回目の場合：
- 様子を見ながら対応
- ひょっとしたら全力で止める
- 状況による判断

**第3回目以降：様子を見る**

3回目、4回目、5回目と続く場合：
- **様子を見る**
- 依存の表現（リストカットに似た状態）の可能性
- ギリギリの状態に行くことでバランスを保っている可能性

**バランスを保っている状態の例：**
- 命に関わるサイト（いわゆる闘サイト）を見ることで安心する
- 見ることでバランスを保っている
- 本人の本当の望みではないが、保つ手段になっている

**判断基準：**
- 何とか保ってくれている
- ガムテープに取り組んでいく方向へ
- 繰り返しの場合は依存パターンの可能性を考慮

### 4. セッション外での過剰な連絡

**症状：**
- 深夜の連絡
- 毎日のメッセージ
- 返信を急かす
- 電話に出ないと怒る

**対応：**

```
最初のセッションで伝えておく：
「セッション外のご連絡は、緊急時以外は
 翌営業日にお返事します。
 深夜や休日は対応できませんのでご了承ください」
```

**ポイント：**
- 最初にルールを伝えておく
- ルールを破っても例外を作らない
- 必要なら連絡手段を制限する

### 5. 料金を払わない・値引きを求める

**症状：**
- 「お金がないから安くして」
- 支払いを先延ばしにする
- 無料でやってほしい

**対応：**

```
クライアント：「今月厳しくて、来月払ってもいいですか」

カウンセラー：「お気持ちはわかりますが、
             料金は当日払いとさせていただいています。
             分割払いをご希望でしたら事前にご相談ください」
```

**ポイント：**
- 料金体系は最初に明確に
- 例外を作ると際限がなくなる
- 分割払いなど代替案を提示
- 経済的に厳しい場合は他のリソースを紹介

### 6. 恋愛感情を持たれる

**症状：**
- 好意を伝えてくる
- デートに誘われる
- プレゼントを渡される
- 性的な話題を出す

**対応：**

```
クライアント：「先生のことが好きになりました」

カウンセラー：「そのお気持ちを伝えてくださりありがとうございます。
             カウンセラーとクライアントの関係では、
             そうした感情が生まれることがあります。
             これは『転移』と呼ばれる現象で、
             あなたの心の中にある何かを投影しているかもしれません」
```

**ポイント：**
- 否定も肯定もしない
- 転移として扱う
- 境界線を明確にする
- 必要なら他のカウンセラーを紹介

### 7. 効果がないと言われる

**症状：**
- 「変わらない」
- 「意味がない」
- 「お金の無駄だった」

**対応：**

```
クライアント：「何回来ても変わらないです」

カウンセラー：「そう感じているんですね。
             変わりたいという気持ちがあるからこそ、
             もどかしいですよね。
             もう一度、なりたい私を見てみませんか」
```

**ポイント：**
- 防御的にならない
- 変わりたい気持ちを肯定
- 決断の重要性を再確認
- 向き合う準備ができているか確認

## トラブルを防ぐために

### 1. 最初にルールを伝える

- セッション時間
- 料金
- 連絡方法と返信タイミング
- キャンセルポリシー
- 緊急時の対応

### 2. 契約書・同意書を交わす

- 口頭だけでなく書面で
- 双方の署名
- 控えを渡す

### 3. 記録を残す

- セッション内容のメモ
- 連絡のやり取り
- 問題が起きた場合の記録

### 4. スーパービジョンを受ける

- 先輩カウンセラーに相談
- 客観的な視点をもらう
- 自分のガムテープに気づく

## よくある誤解・注意点

- ❌ トラブルは自分のせい
- ⭕ クライアントの状態によるものも多い。自分を責めすぎない

- ❌ 全てのクライアントを助けなければならない
- ⭕ 相性もある。他のカウンセラーを紹介するのも選択肢

- ❌ トラブルが起きたら終わり
- ⭕ トラブルから学ぶことも多い。成長の機会

## まとめ

トラブルは起こりうるもの。依存、攻撃、自殺・自傷、過剰連絡、料金問題、恋愛感情、効果不満などに対して、境界線を守りながら対応する。最初にルールを明確にし、記録を残し、スーパービジョンを受けることで、トラブルを防ぎ、起きた場合も適切に対処できる。

## 先天性障害の判断基準

### 「カウンセリング無理だな」という人はいない

- 誰であってもカウンセリングは可能
- カウンセリングとして進まないことはある（本人が決断しない場合）
- しかし「この方はカウンセリング無理だな」という人はいない

### 先天性の脳の障害の場合

**仁さんの専門外：**
- 先天性の脳の障害は完全に専門外
- テープ式心理学は「思い込みを変える・外す」ことが専門
- 先天性の場合はお力になれないかもしれない

**確率的に低い理由：**
- 発信しているコンテンツ（YouTube、ブログ）のキーワードが違う
- 精神的に苦しんでいて、ガムテープがあるという内容
- 先天性の方がアクセスしてくる確率は非常に低い
- 一度もそのようなケースはない

### 支える側から来る場合

可能性として考えられるケース：
- 支える側が「どうにかなりませんか」と連絡してくる
- 先天性の方の家族など

**対応：**
- おそらく正直に話してくれる
- 「小さい頃から病院に行っている」など
- その場合は「お力になれないかもしれません」と伝える

### 現代の「発達障害キャンペーン」への対応

**仁さんの考え：**
- 今の精神科医たちが勝手に「発達障害キャンペーン」をやっている
- 昭和52年頃は本当の先天性の方は少数派だった
- 今アクセスしてくる方の大多数は、実際は先天性ではない可能性

**判断基準：**
- 会話していたらわかる
- ある程度のジャッジは持っている
- 「発達障害なんです」と言われても疑ってかかる
- 多分カウンセリングする（対応する）

**圧倒的に少ないケースへの対応：**
1. 昭和52年頃の本当の少数派の先天性の方は確かにいる
2. しかし圧倒的に少ない
3. さらにその方々が私たちにアクセスする確率はさらに低い
4. もしアクセスしてきて明らかに先天性なら、正直にお伝えする
