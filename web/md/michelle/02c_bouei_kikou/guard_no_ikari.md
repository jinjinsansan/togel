# ガードの怒り

## 定義

怒りの下に別の感情（恐怖、悲しみ、寂しさ、無価値観）が隠れている状態。怒りが本当の感情を守る「ガード（防御）」として機能している。

## 怒りの2種類

怒りには**2種類**ある。

### 1. 純粋な怒り

その場その場で発生する、素直な怒り。

**例：**
- 通りすがりにいきなりひったくりに遭った
- 順番を抜かされた
- 理不尽な扱いを受けた

**特徴：**
- その瞬間に発生してその瞬間に消える
- 心理カウンセラーが対応するものではない
- 正当な怒り

### 2. ガードの怒り

本当の感情（恐怖、悲しみ、寂しさ、無価値観）を守るための怒り。

**特徴：**
- 下に別の感情が隠れている
- コントロールが難しい
- **心理カウンセラーが対応するのは主にこちら**

## 背景・目的

### なぜガードの怒りが発生するか

- 本当の感情（恐怖、悲しみ、寂しさ、無価値観）を感じたくない
- 怒りは「強さ」を感じさせる感情
- 怒っている間は、弱さを感じなくて済む

### ガードの怒りの3ステップ

```
【ステップ1】怒り以外のネガティブな感情を感じる
    ↓
【ステップ2】そのネガティブをこれ以上感じたくない
    ↓
【ステップ3】ガードの怒り発動
```

**例：**
1. 彼氏に無視されて**寂しさ**を感じた
2. これ以上寂しさを感じたくない
3. 「なんで無視するの！」と**怒り**が発動

## 中核ポイント

### ガードの怒りの構造

```
【表面】怒り ← ガード（防御壁）
    ↓
【深層】恐怖、悲しみ、寂しさ、無価値観 ← 本当の感情
```

### 怒りの特殊性

五大ネガティブの中で、怒りだけが特殊な位置づけを持つ。

| 感情 | 特徴 |
|------|------|
| 恐怖 | 直接感じることが多い |
| 悲しみ | 直接感じることが多い |
| 寂しさ | 直接感じることが多い |
| 無価値観 | 直接感じることが多い |
| **怒り** | **他の感情を隠すガードとして現れることが多い** |

### 怒りの下にある感情の例

| 表面の怒り | 隠れている本当の感情 |
|-----------|---------------------|
| 「裏切られて許せない！」 | 悲しみ、寂しさ |
| 「なんで分かってくれないの！」 | 寂しさ、無価値観 |
| 「あいつが悪い！」 | 恐怖（自分が悪いと認めるのが怖い） |
| 「理不尽だ！」 | 無価値観 |
| 「ふざけるな！」 | 悲しみ |

## カウンセリングでの対応

### 怒りが出てきた時

単純に「怒りの解放」に進まない。まず、その下に何があるかを確認する。

### 質問の例

「その怒りの下には、何がありそうですか？」
「怒りを感じる前に、何を感じましたか？」
「もし怒りがなかったとしたら、何を感じますか？」

### ピールダウンの流れ

```
相談者：「許せない！怒ってます！」

カウンセラー：「その怒りの下には何がありそうですか？」

相談者：「...悲しい...」

カウンセラー：「悲しいんですね。その悲しみについて教えてください」
```

## なぜガードするのか

### 弱さを見せたくない

- 悲しい、寂しい、怖いと感じることは「弱さ」と認識される
- 特に男性は「怒り」以外の感情を表現することに抵抗がある
- 怒りは「強さ」を感じさせる

### 感じたくない

- 本当の感情を感じると痛い
- 怒りでガードしている方が楽
- 無意識の防衛機制

## 怒りを出せない人の問題

### 怒りを全く出せない人

「あまり怒ることがない」「怒りを感じない」という人がいる。

**考えられる原因：**
- 怒りを許可できていない
- 怒りを封印している
- 怒りを出すことが「悪いこと」だと思っている

### ノーガード状態の危険性

怒りが出せない＝**ガードができない**

```
ネガティブ（恐怖・悲しみ・寂しさ・無価値観）
    ↓
ガードなし
    ↓
全部自分にボコボコ食らう
    ↓
深刻な状態に陥りやすい
```

### 怒りを出せない人の傾向

- 寝込みやすい
- 動けなくなる（無気力）
- 深刻な状態になりやすい
- 怒りを出せる人に比べて重症化しやすい

### 「仏様みたい」は危険信号かも

「全然怒らないよね」「仏様みたいだね」と言われる人がいる。

**本当にバランスが取れている場合：**
- ネガティブな意味づけをしない
- 健やかに意味づけできている
- 本当の意味で穏やか

**実は危険な場合：**
- 怒りを封印している
- 感じないようにしている
- いずれ爆発するか、深刻な状態になる

### 怒りを出せない人へのアドバイス

**「怒ってください」**

怒りは素敵な感情。自分を守るために必要。怒りを許可することが大切。

## 怒りとコミュニケーション

## 怒りの解放方法

### ガードの怒りの場合

1. まず、下にある本当の感情を特定する
2. その感情に対する解放テクニックを適用する
3. 本当の感情が解放されれば、ガードの怒りも消える

### 例：悲しみが隠れている場合

```
怒り（ガード）
    ↓ ピールダウン
悲しみ（本当の感情）
    ↓ 解放
時間をかけて涙を流す、吐き出しを受け止める
    ↓
悲しみが軽くなる
    ↓
ガードの怒りも消える
```

### 純粋な怒りの場合

本当に怒りそのものが核の場合は「許し」のワークを行う。

## トラブル対策としてのガードの怒り

### カウンセリング中に相談者が怒り出した時

- 相談者がガードの怒りを出してくる
- カウンセラーに攻撃的な言葉を使う
- 命に関わるアクションを見せてくる

### 対応

- **沈黙で持って答える**
- 何もしない
- 怒りでワッとしない
- 感情がコントロールできている状態を保つ

### 理解すべきこと

表現している、頑張ってガードの怒りを出してるってことは**命に関わることにはならない**。そういう見方でいる。

## よくある誤解・注意点

- ❌ 怒りを感じたら怒りの解放をする
- ⭕ まず怒りの下に何があるか確認する

- ❌ 怒りは悪い感情
- ⭕ 怒りはガードとして機能している場合がある。本当の感情に気づくきっかけになる

- ❌ 相談者の怒りに対して怒りで返す
- ⭕ 沈黙で答え、センターを保つ

## まとめ

ガードの怒りとは、本当の感情（恐怖、悲しみ、寂しさ、無価値観）を守るために表面に現れる怒り。カウンセリングでは、怒りの下に何があるかを確認し、本当の感情に対する解放テクニックを適用する。相談者がガードの怒りを出してきた時は、沈黙で答え、センターを保つことが重要。
