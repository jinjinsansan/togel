# 回避行動の仕組み

## 定義

ネガティブな感情（特に無価値観）を感じたくないために、無意識に行う回避的な行動パターン。

## 背景・目的

### なぜ回避行動が起きるか

根本にあるネガティブ感情（例：無価値観）を感じたくないため、自動的に回避行動をとってしまう。

## 中核ポイント

### 回避行動の構造

```
① 幼少期から無価値観がある
    ↓
② それを感じたくない
    ↓
③ 否定・責められる場面に遭遇
    ↓
④ 過剰に傷つく
    ↓
⑤ これ以上感じたくない
    ↓
⑥ 回避行動をとる
```

### よくある回避行動パターン

| パターン | 具体例 |
|---------|-------|
| 人のせいにする | 「それはAさんが言ったから」「Bさんの指示だった」 |
| 過剰に謝る | 「すみません、すみません」を繰り返す |
| 言い訳をする | 「だって〜だから」「でも〜だったから」 |
| 逃げる | その場から立ち去る、話題を変える |
| 攻撃する | 相手を先に責める、怒りで防御する |

## 具体例：人のせいにするパターン

### 実際の相談例

後輩から「息をするように人のせいにしている」と言われてショックを受けた方の例：

```
本人の自覚：
- 意識せずに人の名前を出してしまう
- なぜそうなるのかわからない

根本の原因：
- 幼少期からの無価値観
- 責められると過剰に傷つく
- それを回避するために人のせいにする
```

### 悪循環

```
無価値観がある
    ↓
責められると傷つく
    ↓
傷つきたくないから人のせいにする
    ↓
「人のせいにする人」と評価される
    ↓
また無価値観を感じる（さらに傷つく）
    ↓
さらに回避行動が強まる
```

## 謝りまくるパターンも同じ

### 構造

```
無価値観がある
    ↓
責められると傷つく
    ↓
傷つきたくないからすぐ謝る
    ↓
2度3度続くと...
    ↓
「全然反省しないな」「謝ればいいと思ってるのか」
    ↓
また無価値観を感じる
```

### 共通点

どちらのパターンも根本は**無価値観を感じたくない**という回避から来ている。

## 解放のアプローチ

### 気づきの段階

まず「なぜ回避行動をとっているか」を理解する：

- 根本に無価値観があること
- それを感じたくないから回避していること
- 回避行動がさらなる無価値感を生んでいること

### 意味づけの転換

責められた時に：

```
旧パターン：
「責められた」→「私はダメだ」→「回避しなきゃ」

新パターン：
「責められた」→「仕事で頭を下げることはある」→「私の価値は変わらない」
```

### イメージトレーニング

実際の場面を想像して、新しいパターンで対応できるかを確認する：

- 後輩が否定してきた場面を想像
- 人の名前を出さずに対処できる自分をイメージ
- 「ごめんなさい」と頭を下げる自分をイメージ
- その時の気持ちを確認（気持ちいい感じがしたらOK）

### ふとした瞬間をキャッチする

```
出来事 → 意味づけ（ふとした瞬間）→ 感情発生
```

この「ふとした瞬間」を意識的に捕まえる訓練が必要。

## よくある誤解・注意点

- ❌ 回避行動は性格の問題
- ⭕ 回避行動は根本にあるネガティブ感情を感じたくないための無意識の反応

- ❌ 頭で分かればすぐ変えられる
- ⭕ 「ふとした瞬間」を捕まえる訓練が必要。イメージトレーニングを繰り返す

## まとめ

回避行動は、根本にあるネガティブ感情（特に無価値観）を感じたくないために無意識に行う行動パターン。人のせいにする、過剰に謝るなど様々なパターンがあるが、根本は同じ。解放には、根本のガムテープへの気づきと、「ふとした瞬間」を捕まえて意味づけを変える訓練が必要。
