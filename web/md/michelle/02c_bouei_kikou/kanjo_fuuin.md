# 感情封印（失感情症）

## 定義

### 一般的な定義（心療内科）

失感情症（アレキシサイミア）は、感情を言語化できなくて心身症を伴う、さらにはうつ病が発症するなどと説明される。また「病気ではなく性格に由来するものだからそこまで深く考えなくてもいい」という記事も多い。

**問題点**: 現在の精神医学では失感情症について正確に説明できる人がおらず、治療できる医師も存在しないと推測される。

### テープ式心理学での定義

失感情症とは呼ばず**「感情封印」**という言葉を使用する。

感情封印とは、**「見て見ぬふり」機能の最強級バージョン**であり、文字通り感情を封印している状態。

## 背景・目的

### 感情を感じるプログラム

私たちの心は赤ちゃんの時から、ネガティブな感情もポジティブな感情も全てを感情豊かに感じることができるプログラムを持っている。感情は誰でも豊かに感じることができるはず。

### 「見て見ぬふり」機能

心の仕組みの機能の一つで、誰にも教わっていなくても全員が無意識にできる。

**例：**
- 本当は怖がっているのに、怖がっていないそぶりをする
- 本当は寂しいのに、強がっている
- 嫌なことが起きてネガティブを感じているのに見て見ぬふりをする

これは**回避行動**であり、苦しい感情を1秒でも感じたくないからこそ心の中を見て見ぬふりする機能を使って回避している。

## 中核ポイント

### 感情封印の発生メカニズム

感情封印を行うことになる原因は、ほとんどが**幼少期の虐待**（身体的虐待・感情的虐待）。

```
毎日毎日ひどい扱いを受け続ける
    ↓
泣いても泣いてもさらにひどい虐待が続く
    ↓
「これ以上感情を感じていたら自分が壊れてしまう」
「もう生きていくのが困難になる」
    ↓
「今日から感情を一切封印する」と決断
    ↓
感情を感じないようにしてしまうプログラムが起動
```

### 具体例：5歳の少女

- 父親はアルコール依存で仕事もせず昼間から酒を飲む
- 酒を飲むと暴力的になり、母親や少女に暴力を振るう
- シャワーでの水攻め、お風呂に顔をつけられるなど命に関わるレベルの身体的虐待
- 本来守ってもらうべき家が危険地帯
- 本来守ってもらうべき父親が命の危険性がある対象
- 唯一の救いだった母親がある日何も告げずに家を出ていく
- 少女の心は完全に崩壊
- 「これ以上感情を感じていたら崩壊してしまう」→「感情を封印する」と決断

**参考**: アニメ「るろうに剣心」の「天剣の瀬田宗次郎」の幼少期のシーン

### 感情封印の特徴

感情封印を決断した日から、その機能は開始される：

| 項目 | 状態 |
|-----|------|
| 苦しみのネガティブな感情 | 一切感じない |
| 喜びや幸せ | 感じない |
| 笑ったり喜んだり | するが、ほんの少ししか感じていない |
| 状態 | まるでロボットのよう |

感情を感じてしまえばガラスが割れてしまうように、**感情を感じないよう揺らさないようブレーキをかけている**。

### 本当は感じている

感情封印は見て見ぬふりの最強級で感じないようにしているが、**実際には深く深く奥底では思いきり感じている**。

```
表面: 最強級の見て見ぬふり
  ↓
奥底: 思いきり感情を感じている
```

## 感情封印の解除方法

### 感情封印の本質

感情封印は心がバランスを崩して崩壊してしまう前に、**自らを助けるために自動的に対処したプログラム**。精神的には殺されているレベルのひどい環境にいたということ。

### 感情封印を守っている対価

感情封印を行うことで守っている対価は**心の崩壊**。

逆説的に言えば、**感情を感じてしまえば心が再度崩壊するという恐怖**を感じている。それくらいのトラウマが形成されている。

### そのままでも問題ない

本来自分の身を守るためのプログラムだから、**感情封印のままでも全く問題ない**。

ただし感情封印を続けていることで幸せや喜びも感じない。

### 解除のステップ

幼少期の時は自らの命を守るための適切なプログラムだったが、そこから何十年も経過し大人になった今は自分でも命を守れると認識している。

**ステップ1: 再確認**

封印せざるを得なかった幼少期の出来事を真正面から向き合い、「私はあの時に感情封印をしたんだ」と再確認する。

**ステップ2: 決断**

- 感情封印をやめると決断する
- 感情を豊かに感じる自分になると決断し続ける

### 最も難しい壁

感情を感じることで自分が崩壊してしまうのではないかという**恐怖**。

### 壁を超える方法

この恐怖を超えるには**勇気しかない**。勇気を出すことで感情封印を解くことが可能。

ただし感情を感じないため、勇気の感覚が分からない可能性もある。

### 大切なメッセージ

```
本当は感情を感じている
    ↓
あなたは壊れていない
    ↓
崩壊していない
    ↓
あの幼少期の時は自らを守るために適切なアクションだった
    ↓
そのおかげで今日も生きている
    ↓
しかしもう大丈夫
    ↓
心は壊れない
    ↓
全ての感情を受け取る力がある
    ↓
感情を感じることでこれからの人生はさらに豊かになる
```

## よくある誤解・注意点

### 性格ではない

- ❌ 失感情症は性格に由来するもの
- ⭕ 幼少期の虐待環境から自らを守るために選んだプログラム

### 治療できないわけではない

- ❌ 失感情症は治療できない
- ⭕ 心の仕組みを理解し、勇気を出して決断することで解除可能

### 感情封印した人へのメッセージ

本当に今日までよく頑張って生きてくださった。生きているだけでもう十分に頑張ってこられた方。

## まとめ

感情封印（失感情症）は「見て見ぬふり」機能の最強級バージョンで、幼少期の過酷な虐待環境から自分を守るために発動したプログラム。表面では感情を感じていないように見えるが、心の奥底では思いきり感じている。解除には、幼少期の出来事と向き合い、「感情封印をやめる」と決断し、崩壊への恐怖を勇気で超えることが必要。心は壊れない。
