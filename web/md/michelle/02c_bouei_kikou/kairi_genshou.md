# 乖離現象（離人症）

## 定義

### 一般的な定義（心療内科）

離人症とは、自分が自分から離れてしまう感じがするという症状。自分の心が肉体から離れて自分を遠くから観察しているような感覚になったり、自分が自分でないような感覚になること。

### テープ式心理学での定義

離人症とは呼ばず**「乖離」または「乖離現象」**と呼ぶ。病気ではなく、心の仕組みによって起きる**身体的特徴の一つ**であり、**瞬間的な回避機能**として働く。

## 背景・目的

### 心と体のリンク

心にネガティブな感情を感じると、体にも特徴が現れる：

| 身体的特徴の例 |
|--------------|
| 緊張する、ソワソワする |
| パニックになる |
| 汗をかく |
| 頭が痛い、お腹が痛い、肩が痛い |
| 眠れない |
| 食欲がない |
| 太陽の光を拒否する |

これらは心にネガティブな感情を感じることで現れる身体的特徴のため、医学的な検査をしても**原因不明**と診断される。そして心の中のネガティブな感情が消えると、身体的特徴も同時に消えていく。

### 乖離現象が起きる仕組み

```
機能不全家庭での身体的虐待・感情的虐待が普段から続く
    ↓
毎日毎日ストレスがピークに達するレベルが続く
    ↓
ある日、心は「これ以上苦しみを感じていればこのまま壊れてしまう」と感じる
    ↓
瞬間的に体から幽体離脱したかのような状態になる
    ↓
視点だけが肉体から出ていく（＝乖離現象）
```

## 中核ポイント

### 乖離現象の特徴

- 自分の体からまるで魂が抜け出して天井から体を見つめる
- 家の中を見つめるなどの状態になる
- あるのは**視点だけ**
- 物事を考えたり次はどこに移動しようかなどはできない
- 完全に切り離されたものではなく、**一時的に視点だけが肉体から乖離する現象**

### 乖離現象が意味すること

乖離現象が起きているということは：

- **相当な地獄の環境が続いている**
- または**一気にとてつもなく強いストレスがかかった**

ということを表す。「このままでは自分が壊れてしまう」と感じるから回避しているイメージ。

### 症状の出現時期

幼少期や小学生から現れることもある。

## 修正の方法

### 基本的な考え方

乖離現象は身体的特徴の一つ。

```
パニックの場合：
① 何か心にネガティブな感情を感じて → ② パニックになる

乖離現象の場合：
① 何か心にネガティブな感情を感じて → ② 乖離現象が起きる
```

**②にフォーカスするのではなく、①にフォーカスを行う。**

### 確認すべきこと

1. 心の中に乖離現象が起きるほどの**どのようなネガティブな感情を感じているのか**
2. 今暮らしている環境は**安全な環境であるか**

乖離現象が起きているということは強いストレスがかかり続けているということ。いずれにしても心の中と向き合う必要がある。

### 自然な解消

乖離現象はそのまま放置していても、**心の中が解放されれば自然に消えていく**。

## よくある誤解・注意点

### 恐怖を感じる必要はない

- ❌ 乖離現象が起きているからこの先に怖いことが待っている
- ⭕ 恐怖を感じる必要性は一切ない。心の中が解放されれば自然に消える

### 解離性同一性障害との違い

心療内科では「解離性同一性障害としても患者を見ていく」という記事も見られるが：

| 乖離現象 | 解離性同一性障害（多重人格） |
|---------|-------------------------|
| 一時的に視点だけ幽体離脱のようになる | 完全に主人格から人格が分裂する |
| ネガティブな感情を感じた時の身体的特徴 | もっと高度な心の機能（身体的特徴とは違う） |
| 回避機能としては同じ | 次元が異なる |

**乖離現象が起きているから解離性同一性障害になるという安易な推測は不要。**

### 病気ではない

- ❌ 離人症は精神疾患の症状
- ⭕ 心の仕組みによって起きる身体的特徴の一つ

## まとめ

乖離現象（離人症）は、強いストレス環境で心が壊れないように働く瞬間的な回避機能であり、身体的特徴の一つ。幽体離脱のように視点だけが一時的に肉体から離れる現象だが、病気ではなく、解離性同一性障害とも次元が異なる。修正には症状そのものではなく、根本にあるネガティブな感情と向き合い、安全な環境を確保することが必要。心の中が解放されれば自然に消えていく。
