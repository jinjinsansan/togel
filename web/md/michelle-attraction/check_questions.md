# ミシェル引き寄せ講座 理解度チェック質問集

## 使い方

各セクション終了時に、該当する質問を実施してください。

### 判定基準
- **合格（PASS）**: 正答 or 自分の言葉で正しく説明できている
- **要復習（REVIEW）**: 部分的に理解 or 曖昧な回答
- **不合格（FAIL）**: 誤答 or 全く理解していない

### 対応
- PASS → 次のセクションへ進む
- REVIEW → 補足説明後、簡易質問で再確認
- FAIL → 該当セクションを再説明、必要なら前のセクションに戻る

## レベル1：基礎編

### セクション1：私とソースの関係

#### メイン質問
「私とソースの関係を、自分の言葉で説明してみてください」

#### 確認ポイント
- [ ] 「私＝ソースの一部」または「私はソースである」という概念が含まれているか
- [ ] 分離ではなく、つながり・一体感の理解があるか

#### 模範回答例
「私はソースの一部であり、ソースそのものでもあります。波が海の一部であるように、私たちは皆ソースから生まれ、ソースとつながっています」

#### フォローアップ質問（REVIEW判定時）
「では、海と波の関係で例えると、私たちはどちらに当たりますか？」
→ 「波」と答えられればOK

---

### セクション2：ソースの本質

#### メイン質問
「ソースは私たちの願いに対して、どのような態度を取りますか？」

#### 選択式（補助）
```
a) 良い願いだけ叶えてくれる
b) 悪い願いは罰を与える
c) ジャッジメントせず、すべてを受け入れる
d) 願いの内容によって対応が変わる
```
→ 正解：c

#### 確認ポイント
- [ ] 「ジャッジメントしない」というキーワードが出るか
- [ ] 良い悪いの区別なく受け入れることを理解しているか

#### フォローアップ質問（REVIEW判定時）
「ソースが『良い悪いをジャッジしない』とすると、ネガティブな願いも叶う可能性がありますか？」
→ 「はい、だからこそ注意が必要」と理解していればOK

---

### セクション3：思考ちゃんとマインド様

#### メイン質問
「思考とマインドの違いを説明してください」

#### 確認ポイント
- [ ] 思考＝エゴ、過去や未来の心配、ジャッジメントという理解
- [ ] マインド＝本質、静けさ、ソースとつながる状態という理解
- [ ] 両者が異なるものであることを区別できている

#### 選択式（補助）
「瞑想中に雑念が浮かんできました。この雑念は…」
```
a) マインドからのメッセージ
b) 思考ちゃんのおしゃべり
c) ソースからの指示
d) 気にしなくていいノイズ
```
→ 正解：b

#### フォローアップ質問（REVIEW判定時）
「思考ちゃんを黙らせるには、どうすればいいですか？」
→ 「瞑想」「今に集中」「マインド状態に入る」などが出ればOK

#### 確認ポイント

#### 選択式（補助）


## レベル2：理論編

### セクション5：100%信じる力

#### メイン質問
「引き寄せにおいて『100%信じる』とは、具体的にどういう状態ですか？」

#### 確認ポイント
- [ ] 疑いが全くない状態
- [ ] 「もしかしたら叶わないかも」という考えがない
- [ ] 結果に執着せず、叶うことを当然と思っている

#### 選択式（補助）
「『99%信じている』と『100%信じている』の違いは？」
```
a) ほとんど同じ
b) 1%の疑いがすべてを台無しにする可能性がある
c) 99%で十分
d) 数字は関係ない
```
→ 正解：b

#### フォローアップ質問（REVIEW判定時）
「なぜ100%でないといけないのですか？」
→ 「1%の疑いが周波数を乱す」「ソースに混乱したメッセージを送る」などが出ればOK

### セクション6：量子物理学と引き寄せ

#### メイン質問
「二重スリット実験と引き寄せの法則は、どのように関係していますか？」

#### 確認ポイント
- [ ] 観測者の意識が現実に影響を与えるという理解
- [ ] 「見る」ことで結果が変わるという概念
- [ ] 意識と現実のつながりを理解している

#### 選択式（補助）
「二重スリット実験が示していることは？」
```
a) 光は波である
b) 光は粒子である
c) 観測するかどうかで結果が変わる
d) 実験結果はランダムである
```
→ 正解：c

#### 簡易説明（難しいと感じた場合）
「細かい物理学の理解は必要ありません。大切なのは『私たちの意識が現実に影響を与える』という科学的な示唆があるということです」

## レベル4：実践編

### セクション15：引き寄せの6ステップ ★最重要

#### メイン質問
「引き寄せの6ステップを順番に説明してください」

#### 確認ポイント（必須暗記）
1. 選ぶ（感覚体験として願いを明確に）
2. 瞑想する（毎日マインド状態に入る）
3. 動く（両手両足を使って実際に行動）
4. 試練を受け止める（反対勢力を乗り越える）
5. 小さな変化に感謝する（一歩一歩を喜ぶ）
6. 繰り返し100%信じて待つ（1-5を継続）

#### 判定基準
- 6つすべて言える → PASS
- 4-5つ言える → REVIEW（残りを確認）
- 3つ以下 → FAIL（再説明）

#### フォローアップ質問
「ステップ3の『動く』とは、具体的にどういうことですか？」
→ 「魔法ではない、実際に行動する、両手両足を使う」が出ればOK
